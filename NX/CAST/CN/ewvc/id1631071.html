创建中间产品包络体

为了创建忽略内部具体属性的关联部件文件，您必须创建两个产品包络体部件文件，第一个是中间产品包络体，您在其中创建了链接到外部体的 WAVE 链接，您必须移除所有您不想让消费者或供应商看到的内部特征。

  1. 点击标准工具条上的打开 。

  2. 在打开对话框中，点击选项。

  3. 在装配加载选项对话框中，设置如下参数：

    部件版本组：

    • 加载 = 从文件夹

    范围组：

    • 加载 = 所有组件

    • 使用部分加载 =

    • 使用轻量级表示 =

    • 加载部件间数据 =

    • 加载父项 = 所有级别

  4. 点击确定。

  5. 打开 wav3_pkg_assembly_drill。

  6. 在装配工具条上的组件下拉菜单中，选择新建父对象 。

  7. 在新建父对象对话框的模板组中，确保选择的是装配。

  8. 在单位列表中，确保选择的是毫米。

  9. 在名称输入框中，键入 i_pkg 并回车。

    在装配导航器中，您可以看到已创建了一个父对象，并且手钻装配是其中的唯一组件。

  10. 在装配工具条上，点击 WAVE 几何体链接器 。

  11. 在 WAVE 几何体链接器对话框中，设置如下参数：

    • 类型 = 体

      设置组：

    • 关联 =

    • 隐藏原先的 =

    • 固定于当前时间戳记 =

    • 使用父部件的显示属性 =

    • 设为与位置无关 =

    • 复制螺纹 =

  12. 选择如图所示的电池箱(battery pack case)。

  13. 选择其余组成手钻外部的体。

  14. 点击确定。

  15. 在装配导航器中，右击 wav3_pkg_assembly_drill 节点并选择替换引用集→空。

    在创建了 WAVE 链接之后，就不需要再显示父项部件。
