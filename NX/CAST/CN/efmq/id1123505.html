Set up the cross section edit plane

  1. To display the WCS, press W.

  2. Choose Insert→Synchronous Modeling→Edit Section .

  3. In the Cross Section Edit dialog box, in the Sketch Plane group, from the Plane Method list, select Create Datum CSYS.

  4. Click Create Datum CSYS .

  5. In the Datum CSYS dialog box, in the Reference CSYS group, from the Reference list, select WCS and click OK.

  6. In the Cross Section Edit dialog box, click OK.
