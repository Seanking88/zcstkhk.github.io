Sketch and extrude the pivot boss

  1. Create a sketch on the Y-Z plane of the Datum Coordinate System.

  2. Sketch a rectangle with the dimensions shown below.

  3. On the Feature toolbar, click Extrude .

  4. Select the rectangular sketch.

  5. In the Extrude dialog box, in the Limits group, from the Start list, select Symmetric Value.

  6. In the Distance box, type 100 and press Enter.

  7. From the Boolean list, make sure Inferred is selected.

  8. Click OK.
