Create a shell

  1. On the Feature toolbar, click Shell .

  2. In the Type list, make sure Remove Faces, Then Shell is selected.

  3. In the Preview group, clear the Preview check box.

  4. Select the top planar face of the slot profile and the bottom planar face of the part.

  5. In the Thickness box, type 2.5.

  6. Click Show Result .

  7. Rotate the part and make sure the correct faces were removed.

  8. Click OK.

  9. Close the part.
