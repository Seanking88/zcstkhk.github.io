Blend edges

  1. On the Feature toolbar, select Edge Blend .

  2. From the Shape list, select Circular.

  3. In the Radius 1 box, type 6.

  4. Select the eight vertical edges of the top plate.

  5. Click OK.

  6. Blend the tangent edges around the bottom edges of the plate. Use the same blend radius.

  7. Blend all of the edges around the pivot boss under the plate using the same radius.
