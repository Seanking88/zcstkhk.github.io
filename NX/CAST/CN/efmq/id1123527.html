Make faces parallel

Make the four faces you just created parallel to the angled faces at the other end of the part.

  1. For Motion Face, select the face shown.

  2. For Stationary Face, select the face shown.

  3. Make sure the Symmetric check box is selected.

  4. For Through Point, select the point shown.

  5. Use the same method to make the bottom faces parallel to the corresponding angled faces at the other end of the part.

  6. Close all parts.
