Create a circular hole pattern

  1. Click Pattern Feature .

  2. Select the hole.

  3. In the Pattern Definition group, click Circular for the Layout.

  4. For the Rotation Axis select the Datum CSYS origin point and +Z vector.

  5. In the Angular Direction group, from the Spacing list, select Count and Pitch.

  6. Type 4 in the Count box, 90 in the Pitch Angle box, and click OK.
