Sketch and extrude the base

  1. Sketch the profile below on the X-Y plane of the Datum Coordinate System.

    The midpoint of the upper horizontal line is constrained on the Datum Coordinate System origin. The slot is centered relative to the Y-axis.

  2. On the Feature toolbar, select Extrude .

  3. If not already selected, select the sketch.

  4. In the Extrude dialog box, type 0 in the Start Distance box and 50 in the End Distance box.

  5. Click OK.
