Edit face angles

Change the angle between the faces at the rear of the part.

  1. For Origin Object, select the face shown.

  2. For Measurement Object, select the face shown.

  3. Locate the dimension.

  4. In the Angular Dimension dialog box, in the Face Finder group, make sure the Select Symmetric check box is selected. This guarantees that the angled faces on both sides will change.

  5. In the Angle input box, type 120 and press Enter.
