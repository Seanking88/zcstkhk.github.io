Create the basic shape

  1. Click Extrude .

  2. In the Extrude dialog box, in the Section group, click Sketch Section .

  3. Sketch a circle on the XY plane of the Datum CSYS with a diameter of 600 millimeters.

  4. Click Finish Sketch .

    You are returned to the Extrude dialog box, where the circle should already be selected for the section.

  5. In the Limits group, select Value for both the Start and End limits.

  6. Type 0 in the Start Distance box, 300 in the End Distance box, and click OK
