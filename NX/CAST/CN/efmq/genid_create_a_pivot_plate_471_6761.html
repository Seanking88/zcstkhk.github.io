Create a pivot plate

Estimated time to complete this lesson: 27–53 minutes

In this activity, you will create a pivot plate.

This is a plate with a pivot boss on its underside and several mounting holes.

To complete this project quickly, you should be familiar with the following features. Review these lessons if you are unfamiliar with these features.

  • Extrude

  • Datum Plane

  • Hole

  • Edge Blend

  • Chamfer
