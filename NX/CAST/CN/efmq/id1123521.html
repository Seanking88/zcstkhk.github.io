Edit the length of the part

Create a linear dimension between the datum plane and the end of the part.

  1. For Origin Object, select the datum plane shown.

  2. For Measurement Object, select the edge shown.

  3. For Orientation, select the axis shown.

  4. Locate the dimension.

  5. Make sure all three faces shown are selected in Face to Move group.

  6. For the dimension value, type –500.
