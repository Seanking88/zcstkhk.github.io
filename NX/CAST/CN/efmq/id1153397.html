Extrude the base

  1. Open des17_miter_housing_1.

    The part contains three separate sketches that were created on the planes of a Datum Coordinate System.

  2. On the Feature toolbar click Extrude .

  3. On the Selection bar, from the Curve Rule list, make sure Infer Curves is selected.

  4. Select the base sketch.

  5. Make sure the Start Distance is 0.

  6. In the End Distance box, type 15.

  7. Click Apply.
