Extrude the slot

  1. On the Feature toolbar click Extrude .

  2. On the Selection bar, from the Curve Rule list, make sure Infer Curves is selected.

  3. Select the slot sketch.

    You can change the rendering style, rotate the part, or use the Part Navigator to help select the sketch.

  4. Make sure the vector points up in the +Y direction.

  5. In the End Distance box, type 20.

  6. In the Boolean list, make sure Unite is selected.

  7. Click OK.

  8. In the Part Navigator, right-click the Extrude slot feature and choose Make Sketch Internal.
