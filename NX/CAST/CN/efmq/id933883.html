Create datum planes to center the hole

  1. On the Feature toolbar, from the Datum/Point Drop-down list, select Datum Plane .

  2. From the Type list, make sure Inferred is selected.

  3. Select the bottom face of the plate.

  4. Select the bottom face of the pivot boss.

  5. Click OK.

  6. Click Datum Plane .

  7. Select the front face of the pivot boss.

  8. Select the back face of the pivot boss.

  9. Click OK.
