Nastran solution monitor

The NX Nastran Solution Monitor displays real-time information about the progress of the current solve. Some Solution Monitor plots include:

  • Solution Information — displays the .f04 file.

  • Sparse Matrix Solver — plots the number of equations completed.

  • Iterative Solver Convergence — plots convergence against iterations.

  • Eigenvalues Extraction — plots the number of eigenvalues extracted.

It requires Nastran version 6.0 or later.

The Sparse Matrix Solver monitor

As your solve progresses, one or more graphs showing progress or convergence is generated. You can access each graph by clicking the tabs that appear across the top of the monitor window. You can view graphs as they are generated in real time, or you can review all graphs at the completion of the solve.

The information displayed depends on the version of NX Nastran you are running, your model setup, and the solution type.

Use the Solution Monitor to monitor the convergence of the solution and estimate the remaining time required for the solution. Based on the monitor's output, you may want to cancel the solve, refine your model setup, or adjust solution parameters.
