Save As an existing item

To perform a save as:

Note

The item must exist in the Teamcenter database.

  1. Select and display the item to make it the current working item.

  2. Choose File→Save As.

  3. In the Save Part File As dialog box, select the action from the Action list.

    Note

    The options in the Action list are different for master and non-master items.

  4. Use the automatically assigned values, click Assign to display the next value in the sequence, or enter specific values for the boxes that are not grayed out. The boxes that are grayed out cannot be modified and depend on the item (master or non-master) and the action selected.

  5. Click Projects to assign the item to a project. The list of projects are those that are available in Teamcenter and have you as a member. You can select one or more projects.

    Note

    You can change the assignment of projects later by choosing File→Utilities→Projects.

  6. Click Attributes to display the attributes and their values for the item. You can change the value of an attribute by selecting it and entering the value in the Enter Attribute Value box.

    Note

    You may not be able to modify some attributes. In the list in the upper right of the dialog box, select Writable to see the attributes that you can modify, or Read-Only to see the attributes that you are not able to modify.

  7. Click Alternate ID to use an alternate ID. If alternate IDs are not defined for the existing item, this button is not displayed in the dialog box.

  8. Click OK to perform the save as. The new item is saved in the Teamcenter database and becomes the current working item.
