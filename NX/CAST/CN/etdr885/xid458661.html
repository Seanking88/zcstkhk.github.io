Replace the master symbol definition of a locked custom symbol

  1. Right-click the custom symbol in the title block and choose Unlock Update.

  2. Right-click the custom symbol in the title block again and choose Replace.

  3. In the Symbol Instance group, clear the Replace All Instances of Selected Symbols check box.

  4. In the Folder View group, expand the Custom Symbol Library node and select the Part Symbols folder.

  5. In the Symbol View group, select the new_logo_label symbol and click OK.

  6. (Optional) Replace the last locked custom symbol with the new label logo.

  7. Close all parts.
