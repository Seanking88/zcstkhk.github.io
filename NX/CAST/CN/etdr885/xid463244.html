Create a custom view orientation

  1. From the Model Views list, make sure Front is selected.

  2. Click Customized View .


    Front@3 WORK (EXPLODED)

  3. On the View Triad, select the X axis.


    Front@3 WORK (EXPLODED)

  4. From the Angle list, select –45.00.


    Front@3 WORK (EXPLODED)

  5. Select the Y axis.


    Front@3 WORK (EXPLODED)

  6. From the Angle list, select 45.00.


    Front@3 WORK (EXPLODED)

  7. In the Orient View Tool dialog box, click OK.

  8. Click Next to advance to the Layout step.
