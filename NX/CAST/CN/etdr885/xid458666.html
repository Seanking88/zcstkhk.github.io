Update a hole table

You will update the through holes in the base plate and update the hole table for the base plate.

  1. Examine the hole table for the base plate and notice the dimension of the through holes.

  2. Start the Modeling application.

  3. Double-click drf7_85_hole_tables to make it the work part.

  4. In the Part Navigator, double-click Simple Hole (28) to edit the hole’s parameters.

  5. In the Hole dialog box, in the Form and Dimensions group, change the Diameter dimension to 5 mm, and then click OK to accept the change and close the dialog box.

  6. In the Assembly Navigator , double-click drf7_85_hole_tables_dwg to make the drawing the active part.

  7. Start the Drafting application.

    Examine the base plate and note that the holes have updated in the drafting view, but the hole table still displays the old value for the hole diameter.

  8. On the Table toolbar, click Hole Table .

  9. In the Hole Table dialog box, in the Type group, select Update Table.

  10. With Select Table highlighted, select the table containing the hole information.

  11. With Select Hole highlighted, pick the two changed holes in the base plate.

  12. Click OK to update the table.

    Notice that the hole labels change and the hole information is repositioned in the hole table.

    Tip

    The display of the hole table and hole labels can be controlled from the settings on the Drafting→Hole Table tabs in the Customer Default dialog box.

  13. Close all parts.
