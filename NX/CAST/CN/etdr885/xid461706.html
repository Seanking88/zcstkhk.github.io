Create a set of smart lightweight views

  1. Right-click the border of Sheet : Sheet 2, and then choose Add Base View.

  2. In the Model View to Use list, make sure Top is still selected.

  3. Right-click and choose View Style.

  4. On the General tab, from the View Configuration group, select:

    Representation

    =

    Smart Lightweight

    Resolution

    =

    Coarse

  5. Click OK.

  6. Place the base view above the text SMART LIGHTWEIGHT-
    COARSE    .

    SMART LIGHTWEIGHT-
    COARSE

  7. Project another smart lightweight view directly above the first one.

    SMART LIGHTWEIGHT-
    COARSE

  8. Click Close.
