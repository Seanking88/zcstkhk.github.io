Display an exploded view

  1. In the Options group, make sure:

    View Boundary

    =

    Manual

    Auto-Scale to Fit

    =

    Scale

    =

    1:1

    Process Hidden Lines

    =

    Show Centerlines

    =

    Show Silhouettes

    =

    Show View Label

    =

    Preview Style

    =

    Shaded

  2. Click Next to advance to the Arrangement step.

  3. From the Arrangement list, select exploded.

  4. Click Next to advance to the Orientation step.
