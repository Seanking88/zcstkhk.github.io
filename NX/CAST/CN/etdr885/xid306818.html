Smash View

What is it?

Use the Smash View command to move view dependent objects from a view onto the sheet. You can move objects from the following views:

  • Drawing views

  • Detail views created from Drawing views

If you smash a Drawing view that has associated Detail views, the Detail views become independent.

Drawing views to smash

View dependent objects on the sheet

Note

The original view is removed from the drawing.

Objects maintain their apparent size and position. As a result, dimension values change whenever view scale is not 1:1.

Hidden objects are also transferred onto the sheet and view clipping is removed.

The following view dependent objects are deleted:

  • Retained annotations

  • Tabular notes

Exact annotation orientation cannot be maintained when the original view is rotated, due to constraints imposed on sheet annotations.

Why should I use it?

This command helps you reorganize how data is represented on a 2D layout by moving content contained in a Drawing view to a sheet. This supports a more flexible 2D design workflow.
