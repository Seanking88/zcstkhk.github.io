Create a new set of exact views

  1. In the graphics window, right-click the border of Sheet : Sheet 2, and then choose Add Base View.

  2. In the Model View group, from the Model View to Use list, make sure Top is selected.

  3. In the graphics window, right-click and choose View Style.

  4. In the View Style dialog box, click the General tab, and in the View Configuration group, select:

    • Representation = Exact

  5. Click OK.

  6. In the Scale group, make sure that:

    • Scale = 1:2

  7. Place the new base view above the word EXACT.

    EXACT

  8. Project another exact view directly above the first one.

    EXACT

  9. Click Close.
