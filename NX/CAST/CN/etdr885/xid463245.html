Complete the view

  1. In the Layout group, make sure the only view button selected is Parent View.

  2. Under Placement, make sure Option is set to Automatic.

  3. Expand the Margins group, and then select the Ignore Title Blocks check box.

    The view boundary now expands over the area of the title block.

  4. Under Margins, type the following values:

    Between Views

    =

    5

    To Border

    =

    50

  5. Press Enter.

    The view boundary adjusts to the new values.

  6. Click Finish.
