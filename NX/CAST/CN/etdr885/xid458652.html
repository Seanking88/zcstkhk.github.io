Add a permanently aligned projected view

  1. Open drf7_85_broken_view_axle_dwg.

  2. Confirm you are in the Drafting application.

  3. Right-click on the border of the Front@1 view, and then choose Add Projected View.

  4. In the Projected View dialog box, in the View Origin group, make sure the Placement method is set to Hinge and the Associative Alignment check box is selected.

  5. In the graphics window, move the cursor above the front view and click to place the projected view.

  6. Close the Projected View dialog box.

  7. Drag the border of the front view.

    Note that the projected view you added moves with the front view, but the existing orthogonal view remains stationary.

  8. Right-click and choose Undo, or type Ctrl+Z to return the views to their original locations.
