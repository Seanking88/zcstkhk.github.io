Hole table enhancements

Estimated time to complete this lesson: 9–13 minutes

What is it?

You can now create a hole table for multiple bodies in one part. Holes from all the solids in the part are added to the same hole table. All holes in the same solid are indexed first, before holes in another solid are indexed. The first solid indexed is based on the selected ordinate origin point.

Where do I find it?

Application

Drafting

Prerequisite

You must set the environment variable UGII_DRAFTING_HOLE_TABLE = 1

Toolbar

Table→Hole Table

Menu

Insert→Table→Hole Table
