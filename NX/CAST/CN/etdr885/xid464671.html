Create a break-out section view

  1. In the Part Navigator, select the View Break “1” check box to unsuppress the view break.

  2. Choose Insert→View→Section→Break-out.

  3. Click OK to close the Break-Out Section message.

  4. Select the ORTHO@15 view from the Break-Out Section dialog box or from the graphics window.

  5. Select the arc center shown below as the base point.

  6. In the Break-Out Section dialog box, click Select Curves .

  7. Select the studio spline, and then click Apply to create the break-out section view.

  8. In the Break-Out Section dialog box, click Cancel.
