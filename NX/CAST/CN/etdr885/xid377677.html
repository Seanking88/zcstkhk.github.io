View Creation Wizard enhancements

Estimated time to complete this lesson: 11–16 minutes

What is it?

Several improvements have been added to the View Creation Wizard to enhance its overall performance and usability, especially in the realm of large assembly drawings.

Part step

You must explicitly select a part from either the Loaded Parts or Recent Parts list box. If your active part contains no assembly components, a message window advises you that no model part is loaded into your session, and then gives you the choice of either choosing one from the Part Name dialog, or returning to the Part step in the View Creation Wizard.

Large Assembly step

The Large Assembly step is a new step dynamically added to the Task Navigator whenever the number of components in the master assembly part exceeds the value given for the Show when Component Count Exceeds preference. The display settings on the Large Assembly page take precedence over those settings on the Options page and are designed to maximize system performance while placing large assembly views.

You can:

  • Use the Automatically Optimize Settings option to automatically select the optimum View Configuration and Custom View Settings options.

  • Control the display quality of exact or smart lightweight views.

  • Control preview style using the Set View Preview to Border option.

  • Use the Custom View Settings option to manually override traceline, smooth edge, hidden line, thread, and virtual intersection displays.

Options step

When the Large Assembly step is displayed, its options supersede those in the Options step, and the following options are removed:

  • Process Hidden Lines.

  • Hidden line color, font, and width lists.

  • Show Centerlines.

  • Preview Style list.

When Automatically Optimize Settings is selected on the Large Assembly page, the View Style settings on the Options page are disabled.

Orientation step

  • To improve performance, view boundaries are only calculated one time as the parent view is selected.

Layout step

  • With the Manual placement option, a view scale appears on the Status Line as you position your views on the drawing sheet. The scale remains on the Status Line and continually updates until you complete your view placement.

  • When Ignore Title Blocks is selected, the title block area is ignored when you place a view. The view and any inherited PMI can intrude into the area of the title block. By default this option is not selected.

  • The view button arrangement in the Layout group is more intuitive and more accurately reflects the corresponding view layout on the drawing sheet.

  • View boundaries are only recalculated one time when changing Placement from Manual to Automatic, or when changing margin values.

Why should I use it?

The View Creation Wizard provides a simple and effective way of creating one or more drafting views. The steps in the wizard dynamically display options relevant to the content found in the source model. These changes and performance improvements enhance the wizard’s overall usability and offer a greater degree of flexibility and control in managing large assembly views, including the smart lightweight view capabilities introduced in NX 8.5.
