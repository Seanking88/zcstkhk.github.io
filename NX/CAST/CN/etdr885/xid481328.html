Insert a custom symbol into a note

  1. On the Annotation toolbar, click Note .

  2. Click Reset to reset all of the options to their default condition.

  3. In the Text Input group, clear the text box.

  4. Under Symbols, from the Category list, select Custom Symbol.

  5. Under Folder View, expand the Custom Symbol Library node and select the Part Symbols folder.

  6. Under Symbol View, select logo_label.

  7. Under Settings, from the Preference list, select Use Definition.

  8. Click Insert Symbol to add the symbol to the text window.

  9. In the Text Input group, place your cursor in the text box after the text string for the custom symbol, add a space, and then type IS A TRADE MARK OF SYSTEMS INC.

  10. Press Enter and type ALL RIGHTS RESERVED.

  11. Position the note near the top of the drawing border, click to place it, and then close the Note dialog box.
