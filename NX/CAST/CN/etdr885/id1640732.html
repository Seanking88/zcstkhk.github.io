Feature Parameters support for pattern features

When you use the Feature Parameters command, you can now automatically create callouts that display the number of instances in a pattern feature. To do this, you can select either:

  • The parent feature

  • An instance of the pattern feature

You can make your selection from the Feature Parameters dialog box or in the graphics window.

Note

This enhancement supports all hole types, with the exception of tapered holes.
