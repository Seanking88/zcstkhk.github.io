Smart Lightweight views

Estimated time to complete this lesson: 12–18 minutes

What is it?

The NX 8.5 Smart Lightweight view offers a greater degree of flexibility than its predecessor faceted representation view (now simply referred to as a ‘Lightweight’ view), while utilizing many of the same capabilities found in the more resource intensive Exact (precise) views. A new set of view configuration options on the View Preferences/View Style dialog box allows you to choose between a new smart lightweight view or a lightweight view, (as well as a new exact or pre-NX 8.5 exact view).

With the smart lightweight view, you are now able to:

  • Set its resolution quality to coarse, medium, or fine. The resolution quality plays a role in how quickly the view retrieves and updates.

  • Apply true dimensioning to it without fully loading the referenced geometry.

  • Apply shading, hidden line renderings, and centerline symbols to it.

  • Section it.

  • Apply view breaks to it.

  • Employ regional updates to the view so that only those areas affected by a model edit are updated in the view.

  • Make the extracted edges in the view non-associative by applying the Snapshot option.

You are not able to:

  • Build smart lightweight break-out section or legacy broken views.

  • Shade a smart lightweight section view.

  • Apply view perspective, UV grids, or thread renderings to the smart lightweight view.

You can continue creating lightweight and Pre-NX8.5 exact as you always have, but you cannot convert any Pre-NX8.5 view to an exact or smart lightweight view.

Why should I use it?

Smart lightweight views offer a wide array of view options similar to exact views, including support of hidden line displays, sectioning, dimensioning with minimal part loading, and regional updating, while achieving faster retrieval and update times. These capabilities inherent in smart lightweight views improve system performance and memory usage when constructing large assembly drawings.
