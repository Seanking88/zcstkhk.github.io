为黄铜添加属性

  1. 在装配导航器中，右击会话组件组节点并选择添加到组件组 。

  2. 在添加到组件组对话框中，点击根据属性选项卡。

    默认的系统属性包括：mass, area, volume, name, reference set name 以及 part name。

  3. 在值输入框中，键入黄铜。

  4. 点击确定。

    一个包含黄铜属性的新组被添加到了会话组件组的树中，并被高亮显示以便更改名称。

    会话组件组

    新建组件组

    黄铜

    部件中的组件组

  5. 键入材料检查并回车。

    这个属性搜索将检查任何含有黄铜字段的属性—可假定是一个材料属性。

  6. 右击材料检查节点并选择应用。

    带黄铜属性的组件在视图和装配导航器中被选中。

    装配导航器中数量一列显示存在6个部件符合条件。
