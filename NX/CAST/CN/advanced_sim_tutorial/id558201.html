Mesh the walls

You are now ready to mesh the satellite walls.

  Simulation Navigator

  •     Polygon Geometry (expand)

  •   Select the node for the lower walls to redisplay the polygon body.

  •   Deselect the node for the payload polygon body.

  2D Mesh (Advanced Simulation toolbar, Mesh Drop-down list)

  •   Starting with the footprints of the two equipment boxes that are mounted to the wall, select the 18 wall surfaces (do not mesh the floor surfaces).

  • Type

    CQUAD4

  • Element Size

    2.75

  • Mesh Quality Options

     

  •   Split Quads

    Note

    To minimize the solve time for this model, a coarse mesh is used. With Split Quads selected, the software can create a coarse mesh that does not contain highly distorted quadrilateral elements.

  • Destination Collector

     

  •   Automatic Creation

    This option creates a physical property table and mesh collector for the mesh automatically. Automatic Creation is appropriate in this situation because your three 2D meshes require different thicknesses and therefore unique physical property tables.

  • Preview

     

  •   Show Result

  • Rotate your model and look closely to be sure all faces on the walls (including the two equipment box footprints) are included in the mesh.

  • Apply

     

  • Leave the 2D Mesh dialog box open for the next step.
