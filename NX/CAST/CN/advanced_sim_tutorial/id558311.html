Assign the material and mesh thickness

Assign material and physical properties to the PSHELL physical property tables that were created automatically when you created the 2D meshes.

  Physical Properties (Advanced Simulation toolbar)

In the Selection group at the bottom of the dialog box, you should see four physical property tables: PSHELL1, PSHELL2, PSHELL3, and PSHELL4.

  • Selection

    PSHELL1

  • Verify that the mesh selected in the graphics window is the satellite wall mesh (if it is not the satellite wall, select another physical property table until the wall mesh is selected).

  •   Edit

  • Name

    Sat_Walls

  • Material 1

    ALUMINUM_6061

  • Default Thickness

    0.394

  • OK

     

Repeat the above steps for the PSHELL2, PSHELL3, and PSHELL4 physical property tables, using this information:

PSHELL2 (top and bottom mesh)

PSHELL3 (payload footprint mesh)

PSHELL4 (platform mesh)

  • Name: Top_Bottom

  • Material 1: ALUMINUM_6061

  • Default Thickness: 0.093

  • Name: Payload

  • Material 1: ALUMINUM_6061

  • Default Thickness: 1.0

  • Name: Platform

  • Material 1: ALUMINUM_6061

  • Default Thickness: 1.0
