Add distributed mass to the payload footprint

Model the payload on the upper level by modifying the density of its footprint mesh. The payload weighs a total of 87 lb. You need to calculate an equivalent distributed weight to be included as non-structural mass in the physical property table for the elements in the payload footprint.

  Information→Object

  •   the face that represents the payload footprint (use QuickPick to select the correct face)

    Note

    Select the polygon face, not the mesh mating condition. If you are unable to select the face, you may need to click Reset Filter in the Class Selection dialog box.

  • OK

     

  • In the Information window that appears, note the value of the Face Area:
    Face Area = 636.39617919922

  • For shell elements, non-structural mass is entered as mass per unit area. Thus, you will enter 87 lbm / 636.4 in^2 = 0.137 lbm/in^2 as the non-structural mass in the Payload physical properties record.
