Define the footprint for the payload

Use a mesh mating condition to define the footprint for the payload that is mounted to the face above the satellite equipment boxes.

  Simulation Navigator

  •     Polygon Geometry (expand)

  •   Deselect all Polygon Body nodes except the upper walls polygon body and the payload polygon body.

  Mesh Mating Condition (Model Preparation toolbar)

  •     the bottom face of the payload

  •     the face of the satellite floor to which the payload is mounted

  • The settings in the Mesh Mating Conditions dialog box should remain as you set them in the previous step.
