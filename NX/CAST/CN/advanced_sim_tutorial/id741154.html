Activity: Import an I-deas universal file

You can use one of three methods to transfer FEM data from I-deas to NX Advanced Simulation.

  • Content Migration Manager (CMM)

  • I-deas Universal file (UNV)

  • NX Nastran bulk data file

The CMM is the only method that maintains the part associativity with the FEM. If associativity is required, contact your Siemens PLM Software account representative for more information about the CMM. The CMM requires training to use and is not covered in this tutorial. For more information about the CMM, see Content migration process in I-deas to NX Transition: A Roadmap.

In this tutorial, you will import an I-deas FE model using an I-deas Universal file and also using an NX Nastran bulk data file.

Launch the Import I-deas models into NX Advanced Simulation activity.
