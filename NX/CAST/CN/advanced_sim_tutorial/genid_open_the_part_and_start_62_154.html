Open the part and start Advanced Simulation

Before starting the tutorial, you must download the part.

  1. On your desktop or the appropriate network drive, create a folder named pcb-w-chip.

  2. Click the Download link below:

    Download the part

  3. Extract the part files to your pcb-w-chip folder.

  4. Start NX.

Open (Standard toolbar)

  • Files of Type

    Part Files (*.prt)

  • Name

    pcb-w-chip.prt

  • OK

     

Start→ Advanced Simulation if necessary
