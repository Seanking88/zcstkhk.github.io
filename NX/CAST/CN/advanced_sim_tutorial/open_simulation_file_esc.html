Open the Simulation file

Before starting the tutorial, you must download the part.

  1. On your desktop or the appropriate network drive, create a folder named power_supply.

  2. Click the Download link below:

    Download the part

  3. Extract the part files to your power_supply folder.

  4. Start NX.

Open

  • Look in

    power_supply

  • Files of type

    Simulation Files (*.sim)

  • File name

    power_supply.sim

  • OK

     

    Note

    The graphic above shows the model with the Translucency option enabled. To enable Translucency, choose Preferences→Visualization. On the Visual tab, select Translucency.
