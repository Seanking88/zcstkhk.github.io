Create a physical property table

You need to create a physical property table as a container for the properties of the elements in the mesh (you will create the mesh in a later step).

Alternatively, you can let the software create the physical property table and mesh collector automatically when you mesh the part. However, with more complex models, you can better organize your meshes and physical and material properties by fully defining these containers and then selecting them when you create the mesh.
