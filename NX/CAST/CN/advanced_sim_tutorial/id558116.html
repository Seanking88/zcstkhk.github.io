Import the Nastran bulk data file

File→Import→Simulation

  • Select solver

    NX NASTRAN

  • OK

     Import dialog box

  • Browse (Input File)

  • Navigate to the folder where you saved the files that you downloaded for this tutorial.

  • File name

    fork.dat

  • OK

      Select file to import dialog box

  • OK

      Import Simulation dialog box

  • Simulation Name

    your part file folder name\fork_s.sim

  • FEM Name

    your part file folder name\fork_f.fem

  • OK

     New FEM and Simulation dialog box

Leave the Information window open for the next step.
