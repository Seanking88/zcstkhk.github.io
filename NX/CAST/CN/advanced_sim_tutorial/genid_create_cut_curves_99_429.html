Create cut curves

You will create two cut curves on Ply 1 and one on Ply 2 to remove tearing during the draping process.

  • Simulation Navigator

  • Ply 1 (under Mud_guard)

  • Edit

  • Select Cuts (0)

  • OK

     

  • Ply 2

  • Edit

  • Select Cuts (0)

  • on the previous image

  • OK

     

  • Mud_guard

  • Update
