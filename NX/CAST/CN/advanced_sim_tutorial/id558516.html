Plot the pulse function

Use the NX Functions and Graphing tools to plot the pulse function.

  XY Function Navigator

  • Under NX AFU Files – XY Tabular Functions→Associated AFU, expand the node that represents the AFU file where you saved the pulse function in the previous step.

    Note

    If you did not create the pulse function in the previous step, right-click the Associated AFU node and choose Open. Browse to and open the response_functions.afu file that was included with the zip file for this tutorial.
