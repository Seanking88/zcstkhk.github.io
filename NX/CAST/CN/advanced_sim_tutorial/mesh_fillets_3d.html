Create a 3D mesh

Now that you have a 2D seed mesh on the fillets, create a 3D mesh on the part. The software uses the location of the nodes in the 2D mesh to generate the 3D elements through the volume.

  3D Tetrahedral Mesh (Advanced Simulation toolbar, Mesh Drop-down list)

  •   the part to activate the dialog box and select the body to mesh

  • Type

    CTETRA(10)

  •   Automatic Element Size

  • OK

     

  File→ Close→ All Parts
