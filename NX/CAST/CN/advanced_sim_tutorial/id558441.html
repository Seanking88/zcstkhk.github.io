Generate stress contour results

Next, you will evaluate Stress Response Results for all elements on the model, and select the time point in the response function where stress is highest. Use the following steps, referring back to the previous tutorial page for details if necessary:

  1. In the Evaluate Response Results dialog box, select Stress check box under Requested Results, and clear the Displacement check box. Select all elements in the model.

  2. Select From XY Graph as the Method.

  3. Under Point Value, click the button.

  4. In the Equation Selection list, select 2_(16E_VONMTOP) and click OK.

  5. In the plotted elemental stress function, select the time point where stress is highest, and then click to generate the response results.

  6. Click OK to close the Evaluate Response Results dialog box.

In the next step, you will view the contour plot of these stress results as well as the displacement results you generated earlier.
