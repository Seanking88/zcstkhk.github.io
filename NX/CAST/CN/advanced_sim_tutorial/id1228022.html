Examine the results of the structural solution

In the Simulation Navigator, observe the following nodes:

  1. Static Solution — This is the node for the NX Nastran SOL 101 Linear Statics – Global Constraints solution that will be referenced by the static durability event.

  2. Transient Solution — This is the node for the NX Nastran SOL 109 Direct Transient Response solution that will be referenced by the transient durability event.

  •   Static Solution (expand)

  •   Results

  • Examine the stress results of the Static Solution.

    Note

    Notice that the maximum stress of Static Solution (522.8 MPa) is higher than the material yield strength. Therefore, the Smith–Watson–Topper fatigue life criterion will be used for the static durability event that references results from this solution. This criterion needs both stress and strain results and is typically used for low cycle fatigue analysis.

  •   Post Processing Navigator

  •   Transient Solution to load the Transient Solution results.

  • Animate the stress results of the Transient Solution over all iterations.

    Note

    Notice that when the stress results of Transient Solution are animated over all iterations the maximum value (261.06 MPa) is lower than the material yield strength. Therefore, the stress life criterion will be used for the transient durability event that references results from this response simulation. This criterion is typically used for high cycle fatigue analysis.
