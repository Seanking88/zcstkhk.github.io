Create PSD function

Use the Function Tools for Response Simulation utility to create a PSD function for use as a dynamic excitation.

Note

To create the PSD function, your system must have Java Runtime Environment (JRE) version 1.7 installed. You can install the JRE from www.oracle.com. You can optionally use the PSD function provided in the response_functions.afu file, which is located in the parts zip file you downloaded for this tutorial. If you use the provided file, skip to the next step.
