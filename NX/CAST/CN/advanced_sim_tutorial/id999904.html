Setup

  1. On your desktop or the appropriate network drive, create a folder named strut3.

  2. Click the Download link below:

    Download the part

  3. Extract the file to your strut3 folder.

  4. Start NX.

  5. Open strut_fem1.fem.

  6. In the Simulation Navigator, right-click the strut_fem1_i.prt and select Load.
