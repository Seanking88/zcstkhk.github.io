Evaluate transmissibility

Note

Before starting this activity, you must have completed Response Simulation – FE model setup and the Simulation and FEM files you created in that activity must be open in NX.

Transmissibility is a frequency response function (FRF) that lets you evaluate the response of one or several output nodes to an enforced motion input such as displacement, velocity, or acceleration at a selected node.
