Examine the imported model

When the import process is complete, the Information window displays this information:

  • The Analysis of Import section lists information about each Nastran bulk data entry recognized and the number of entities imported for each bulk data entry.

  • The Summary of Import section list the number of nodes, coordinate systems, restraints, elements, physical properties, materials, and loads imported.

Notice that a Simulation file has been created and the loads and boundary conditions have been imported. Also, all of the rigid elements and beam elements have been imported.

  the Information window

The new Simulation file should be displayed in NX.

As you can see, importing the NX Nastran bulk data file offers a more complete import of your I-deas model.
