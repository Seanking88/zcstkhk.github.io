Open and explore the thermal Simulation file

Before starting the tutorial, you must download the part.

  1. On your desktop or the appropriate network drive, create a folder named pcb_mapping.

  2. Click the Download link below:

    Download the part

  3. Extract the part files to your pcb_mapping folder.

  4. Start NX.

Open the Simulation file to include and explore the assembly.

Open

  • Files of Type

    Simulation Files (*.sim)

  • Look in

    pcb_mapping

  • File name

    pcb_thermal_sim.sim

  • OK
