High Quality Image

Use the High Quality Image command to create static rendered images.

You can:

  • Set specific parameters for rendering a static image.

  • Plot and save the generated images to an external file.

  • Use a series of the generated images to create an animated movie or QuickTime VR file.

  • Use the Showroom Environment display in combination with high quality image rendering for a more realistic display of your rendered images.

Where do I find it?

Prerequisite

A camera icon in any Visualization dialog box indicates that the setting or input is visible only in a high quality image.

Toolbar

Visualize Shape→ High Quality

Visualization→ High Quality

Menu

View→Visualization→High Quality Image

Shortcut menu

Ctrl+Shift+H
