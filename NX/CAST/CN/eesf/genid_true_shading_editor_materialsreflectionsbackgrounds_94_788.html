True Shading Editor – Global and object specific materials

Global Materials group

The global materials palette is displayed with available materials

These are the same options that are available using the Global Materials Drop-down on the True Shading toolbar.

Some examples:

brass

brushed metal wash

brushed chrome

clear plastic

Object Specific Materials group

The object specific materials palette is displayed with available materials

These are the same options as available using the Override Materials Drop-down on the True Shading toolbar.

Example:

  1. Global Materials = clear plastic

  2. Object Specific Materials = black glossy paint

Select Object lets you redesignate specific material types on objects.

Remove Material From Object removes any designated material from selected objects.

Global Reflections group

Presents a list of Images that can be used as the reflection maps.

  • Shiny Metal (two versions)

  • Brushed Metal (two versions)

  • Glossy (two versions)

  • Surface Analysis (lines and horizon)

  • Automotive Lighting

Background group

Presents a list of Background Types for selection.

  • Graduated Dark

  • Graduated Light

  • Dark

  • Light

  • Custom Plain

  • Custom Graduated

  • Inherit

  • Image Background
