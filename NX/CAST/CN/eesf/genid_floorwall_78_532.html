Floor/Wall

You can further control shadows using the Floor option which has four conditions:

None

  • Displays True Shading views with no floor or back wall environments.

  • No modal options are available.

Bottom

  • Displays shadows and object reflections on a floating bottom plane.

  • Show Shadows

  • Show Floor Reflection

  • Grid modal options are available.

Back

  • Displays shadows on a fixed back plane.

  • Show Shadows modal option is available.

  • Show Floor Reflection and Grid modal options are not available.

Bottom Fixed

  • Displays a fixed bottom plane.

  • When you rotate the view, the fixed bottom plane does not change.
