The High Quality Image dialog box

The High Quality Image dialog box is divided into three sections.

  1. Rendering methods

  2. High Quality Image (HQI) options

  3. Action options

Rendering Methods

Use Method to designate the rendering method. The methods are arranged in the order of increasing realism and increasing processing time.

Each rendering method represent a different mathematical imaging algorithm, and each has specific characteristics and applications.

High Quality Image (HQI) options

Image Preferences control such shading aspects as:

  • Display color

  • Image size

  • Shadows

  • Resolution

Use the Information option if you need information:

  • Regarding the shading process.

  • On how the model is setup for shading.

The HQI information option is very useful when working on model renderings with which you are not familiar.

Import from LWK File (imports Lightworks files from a directory) provides a means of customizing material, texture, and light configurations by retrieving them from LWK (Lightworks) files.

Action options

  • The Save button becomes active once an image has been shaded. This brings up the Save Image dialog box. You can also compress the image file if it is very large.

  • The Plot button becomes active once an image has been shaded. This where you control the resolution and quality of the plot.

  • Start Shade starts the process of image generation.

  • Erase Shade erases the shaded images.
