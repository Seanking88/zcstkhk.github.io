Create the first section

  1. Open des07_dynamic_sections.

  2. On the View toolbar, click Edit Work Section .

    If a section does not already exist, a new section is created.

  3. In the View Section dialog box header, click Reset .

    This resets the dialog box options to their default condition.

  4. Drag the Rotate about XC-axis handle 90 degrees around the X-axis.

  5. Drag the Rotate about YC-axis handle 90 degrees around the Y-axis.

  6. In the Section Curve Settings group, select the Show Section Curves Preview check box.

  7. Select the Color swatch and change it to Yellow.

  8. Click OK.
