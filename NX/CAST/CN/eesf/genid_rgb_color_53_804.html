RGB Tolerance

The RGB Tolerance sets a range of colors within the tolerance as transparent pixels. The tolerance is equal to the transparency color value + RGB tolerance value.

The RGB Tolerance can range from 0 to 255.

  • 0 = very strict adherence to the Transparent Color.

  • 255 = very loose adherence to the Transparent Color.

Sometimes, the area of an image that is to be made transparent is not a pure color, resulting in a messy decal appearance. To avoid this, increase the RGB Tolerance.
