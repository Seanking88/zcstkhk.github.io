Set up for cameras

  1. Open des07_teapot_2 and start the Modeling application.

  2. In the Part Navigator, expand the Cameras node.

    A camera takes the name of the view from which it was created; if multiple cameras are started from the same view, a number is appended to the name.

  3. On the View toolbar, click Fit .

  4. In the Part Navigator, notice that the Trimetric camera now has this icon beside it , which indicates that it is out-of-date.

    This is because you changed the original orientation of the trimetric camera.

  5. Right-click Trimetric, and choose Capture View.

    Because you captured the current camera orientation, the trimetric camera is now current and a green check mark replaces the out of date icon .
