True Shading overview

Use True Shading to make realistic product visualization using pre-defined visual effects. You can rotate your model and see the visualization changes in real time. True Shading is useful for creating product presentations.

By activating True Shading, a number of display conditions are applied:

  • Materials

  • Reflections

  • Backgrounds

  • Floor

  • Lights

  • Shadows

You can apply true shading effects to models using either the True Shading toolbar, or the True Shading Editor dialog box.
