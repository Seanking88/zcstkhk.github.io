Place the decal on the middle of the face

  1. In the Name group, in the Decal Name box, type engrave.

  2. In the Image group, click Choose Image File .

  3. Navigate to the parts directory and double-click des07_engrave_text.jpg.

    Note

    If you do not see the proper images, make sure the Files of type filter is set to .jpg

  4. In the Image group, select the Preview check box to see a preview of the decal.

  5. In the Object to Decal group, click Select Object .

  6. In the Object to Decal group, with Select Object highlighted, select the face of the support forging.

  7. In the Placement group, set Anchor Type to Center.

  8. In the Placement group, click Specify Origin and select Point on Face from the drop down.

  9. Click on the face.

    This defines where the decal will be placed.

  10. Click Specify Up Vector, then from the drop down select YC Axis to orient the decal.
