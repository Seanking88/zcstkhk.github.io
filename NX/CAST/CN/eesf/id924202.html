Copy and edit a decal

  1. On the Resource bar, click Materials in Part .

  2. In the Materials in Part palette, right-click logo_name and select Copy.

  3. Right-click in a blank area of the Materials in Part palette, and select Paste.

  4. Right-click on the new pasted entry and select Edit.

  5. In the Edit Decal dialog box, in the Name group, change Decal Name to decal_triangles.

  6. In the Image group, click Choose Image File.

  7. Navigate to the parts directory and select des07_triangles.jpg.

    In the Decal Image dialog box, make sure Files of type is set to JPEG Files (*.jpg)

  8. In the Object to Decal group, click Select Object.

  9. In the Object to Decal group, with Select Object highlighted, select the two front faces of the bottle.

  10. In the Placement group, make sure Anchor Type is set to Center.

  11. Set Specify Origin to Point on Curve/Edge to place the origin of the decal in a similar fashion as the first decal.

  12. In the Scaling group, set Scaling Method to Uniform Scale.

  13. Set Scale to 80.00 and Aspect Ratio to 0.7500.

  14. In the Transparency group, set RGB Tolerance to 70.

  15. In the Illumination group, set Type to Use Decal's Own Material.

  16. Click Edit Illumination.

  17. In the Material Editor dialog box, set Type to Dielectric, then click OK.

  18. In the Displacement group, clear the Enable Displacement check box.
