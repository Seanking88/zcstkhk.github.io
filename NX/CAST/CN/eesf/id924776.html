Reorient and reposition the Raster Image

  1. Start the Modeling application.

  2. With the cursor on the top view image, right-click and select Edit Parameters.

  3. Drag the XC-YC rotation handle (image will rotate about +ZC) until the top view image is in the orientation shown below (parallel to the X-axis).
