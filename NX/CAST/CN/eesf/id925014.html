Edit the System Scene

  1. On the Visualize Shape toolbar, click Scene Editor .

  2. In the Scene Editor dialog box, set the following:

    Background tab:

    • Background = Graduated

    Stage tab:

    • Stage Size = 1000

    • Floor/Wall Settings = Floor

    • Material Type = Reflective

    • Wall Image = Choose from Image Palette

    • TIFF Palette = Wood – select a light wood thumbnail

    • Pattern Repeat Factor = 2

    • Reflectivity = .5

  3. Click Apply.
