Apply an image as a decal

The supported image formats are:

  • TIFF

  • JPG

  • PNG

  • Transparent TIFF

Image Size

  • Determines the size of the decal image.

  • True Size will use the actual size of the graphic file.

  • The other values (128, 256, 512 etc.) will sharpen the image; the higher the value, the sharper the image and the bigger impact on performance.

  • The attainable values depend on your graphics driver.
