Scaling

There are four scaling methods:

Face Size

The decal is fitted, as best as possible, to the size of the face selected for the decal application.

Image Size

The decal is applied using the actual size of the decal image.

Uniform Scale

The decal is applied with uniform height and width based on an specified scale.

Non-Uniform Scale

The decal is applied with a individual scale height and individual scale width.
