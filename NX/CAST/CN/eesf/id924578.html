Refine the illumination of the decal

  1. In the Illumination group, set Type to Use Decal's Own Material.

  2. Click Edit Illumination.

  3. In the Material Editor dialog box, set Type to Dielectric, then click OK.

  4. In the Displacement group, select the Enable Displacement check box.

  5. Set Amplitude to –0.500.

  6. Set Softness to 0.100.

  7. Click OK.
