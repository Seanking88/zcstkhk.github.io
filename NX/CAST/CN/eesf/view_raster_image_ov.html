Raster Image

Use the Raster Image command to import raster images into NX in TIFF format. You can construct curves or sketches directly on top of the raster image.

Note

You must set the rendering style to Studio to see the raster image.

When you import a raster image, you can control the following:

  • Unit of measurement

  • Format size

  • Aspect ratio

  • Height and length dimensions

  • Translucency

Dynamic handles are also available on the raster image plane to help you control size, rotation and positioning of the image.

Where do I find it?

Toolbar

Visualize Shape→Raster Image

Menu
