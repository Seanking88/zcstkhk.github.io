Change the background image

  1. In the 3D Dome Panoramic LDR Image group, click Choose Image File .

  2. Navigate to the parts directory and select des07_sled_hill_3.jpg, then click OK.

    You want the size to be larger than the default stage size.

  3. In the 3D Dome Settings group, clear the Use Stage and Orientation check box.

  4. In the Dome Size box, enter 7000 and press Enter.

  5. Using pan, zoom, and the Image Horizon and Rotation Angle options, set up a dramatic scene.

  6. With the Scene Editor dialog box still open, on the Visualize Shape toolbar, click Start Shade when you have established a suitable scene.

    Once you have a High Quality Image rendering, you can use Capture Studio Image to manipulate the layout and the quality of the captured image, and then file a TIFF formatted image to a directory you specify.

  7. Click Erase Shade .

  8. Close the part.
