Placement of a decal

The placement of a decal is designated using its:

  1. Origin

  2. Up Vector

  3. Normal Vector

The Origin can be positioned either using the Point Constructor or by dragging.

The Anchor Type positions the decal origin in relation to the image.

Anchor Types

The Rotation Angle rotates the Up Vector around the Normal Vector to provide further control with positioning the image on the surface.
