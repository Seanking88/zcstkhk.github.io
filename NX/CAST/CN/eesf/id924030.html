Create a Pencil Shade artistic image

  1. In the Artistic Image dialog box, click Pencil Shade .

  2. Set the following Pencil Shade parameters:

    • Stroke Length =

    • Stroke Density =

  3. Click Start Shade .

  4. Click Erase Shade .

  5. Select the Line Color check box .

  6. Set the Line Color to gray.

  7. Click Start Shade .

  8. Click Erase Shade .

  9. Experiment with different Stroke Length, Stroke Density and Line Color settings.

    You must select the Line Color check box for the Line Color swatch to become active.

  10. Click Reset to Default Options .
