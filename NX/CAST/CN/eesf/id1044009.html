Basic visualization tools

Estimated time to complete this course: 2–3 hours

This course covers how to use basic visualization tools. You will:

  • Apply various drawing effects to enhance or simplify the depiction of your model.

  • Learn how to control the basic lighting of your model.

  • Use cameras to capture viewing characteristics of your current work view.

  • Use sectioning to show cross-sections of your model.

  • Learn to apply decals to your model.

  • Set up various types of rendering scenarios.

  • Use raster images to help in the design process.

  • Set up simple materials, lighting and reflection maps to create visualization scenarios in which you can work.

  • Apply system scenes, edit them using the Scene Editor, and create your own custom scenes.
