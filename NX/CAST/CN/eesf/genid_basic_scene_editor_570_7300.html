Scene Editor

Use the Scene Editor to create a realistic display around your model. NX provides pre-defined scenes for ready use.

Background

There are four types of backgrounds:

Plain

Graduated

Image File

3D Dome

Stage

The environment in which a visualization scene is displayed is called a Stage. It is based on an automatically defined Showroom Environment cube.

Stage Size

Controls the size of the cube.

Stage Offset

Controls the vertical position of the cube.

Define Floor

Aligns the floor to the X, Y, or Z axis and specify a floor plane.

Floor/Wall Settings

Controls the visibility of each wall and what images will be mapped to what walls.

Reflection

You can change the reflections seen on the floor.

2D Background

Uses the background specified in the Background page of the Basic Scene Editor for the reflection onto your model.

Stage Floor/Walls

Uses the floor and/or walls of the environment cube for the reflection onto your model.

Image-Based Lighting

Uses a high dynamic range image (which provides the lighting) for reflection onto your model.

User-Specified Image

Uses any image you specify to be the reflection onto your model.

For instance, you may want a Graduated background but want a specific image, not the graduated background, to be reflected on your model.

Lights

You can make minor adjustments to existing lights, such as intensity and orientation using the Scene Editor dialog box.

The Lights page does not let you:

  • Create/enable new lights

  • Delete/disable lights

Global Illumination

Set up a global lighting schemes based on High Dynamic Range images, stage, and other image-based lighting while controlling shadow catching and color saturation.

Global Illumination overrides any basic or advanced lighting set ups.

Shadows

Set up shadow catchers and inter-object shadowing.
