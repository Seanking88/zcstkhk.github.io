Apply different backgrounds to see the effect on the model

  1. From the Background Drop-down list, select Graduated Dark Grey Background .

  2. Try applying some other backgrounds.

    Graduated Light Gray

    Dark

    Light

  3. Choose Preferences→Background.

    You will create a customized graduated background.

  4. In the Edit Background dialog box, in the Shaded Views area, make sure Graduated is selected.

  5. Set the Top color to red , set the Bottom color to white and then click OK.

  6. On the True Shading toolbar, change the background to Inherit Shaded Background .
