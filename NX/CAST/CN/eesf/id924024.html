Create a Hand Drawn artistic image

  1. In the Artistic Image dialog box, click Hand Drawn .

  2. Using the default settings, click Start Shade .

  3. Click Erase Shade .

  4. Set the following Hand Drawn parameters:

    • Outline Color =

    • Outline Width =

    • Line Deviation =

  5. Click Start Shade .

    When using the Hand Drawn effect, be aware that the background is considered to be the paper onto which the lines are drawn, so it is important to have a background that sets off your outline color.

  6. Click Erase Shade .

  7. Experiment with different Outline Width, Line Deviation, Line Regularity, Line Taper and Outline Color settings.

  8. Click Reset to Default Options .
