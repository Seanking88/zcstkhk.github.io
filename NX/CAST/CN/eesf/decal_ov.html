Decal

Use the Decal command to overlay an image file on a specified body or faces of a body.

In the following animation, an image is overlaid on different locations on the model.

You can:

  • Use TIFF, JPEG, or PNG images to create a decal.

  • Apply an unlimited number of decals in a part.

  • Apply decals to faces, solids, and facetted bodies or faces.

  • Overlay multiple images on a face or textured surface.

  • Overlay images that span multiple surfaces.

  • Make the overlaid image inherit the material and texture of the surface.

  • Specify a material and texture of your choice for the overlaid image.

  • Edit the base surface material and decal material separately.

  • Control the transparency of the overlaid image for stenciling.

  • Overlay images as engravings.

  • View and edit the decals saved in the active part file in the Materials in Part palette.
