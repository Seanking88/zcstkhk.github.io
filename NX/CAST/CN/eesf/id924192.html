Place the first decal

  1. In the Name group, in the Decal Name box, type logo_name.

  2. In the Image group, click Choose Image File .

  3. Navigate to the parts directory and select des07_name.jpg.

    Note

    If you do not see the proper images, make sure the Files of type filter is set to .jpg

  4. In the Image group, select the Preview check box to see a preview of the decal.

  5. In the Object to Decal group, with Select Object highlighted, select the two front faces of the bottle.

     

  6. In the Placement group, set Anchor Type to Center.

  7. In the Placement group, for Specify Origin, select Point on Curve/Edge from the drop down.

  8. Click to select a location on the faces as shown below.
