Change the floor characteristics using the True Shading Editor

  1. On the True Shading toolbar, click True Shading Editor .

    All the True Shading toolbar functions can be accessed in the True Shading Editor, including a few other functions not available on the toolbar.

  2. In the Floor group, from the Specify Plane list, select YC-ZC Plane.

  3. In the Offset box, type 2.

  4. In the Lights group, move the Brightness slider to make the scene lighting darker or brighter.

  5. Click OK.

    Note on the True Shading toolbar, Custom Lighting is in effect. This is because you customized the brightness of the default lighting scheme.

  6. Experiment with the other True Shading options as needed.

  7. Close the part.
