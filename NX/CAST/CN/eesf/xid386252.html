Apply a 3D dome system scene

Note

You must be in the Advanced with full menus role and have access to Advanced Studio Display to complete this activity.

  1. Open des07_sled_assm.

  2. If necessary, display the Visualize Shape toolbar.

  3. On the Visualize Shape toolbar, click Advanced Studio Display .

    If the Advanced Studio Display is turned off in Visualization Preferences, you get a message asking you if you want enable and use Advanced Studio Display.

  4. Select the Don’t display this message again check box, and click Yes.

  5. From the Resource bar, click System Scenes .

  6. Select the Nymphenburg 3D Dome thumbnail from the palette.

    The 3D dome is applied with the default settings of perspective and size. The default size of the dome is approximately three times the size of the model.

    Top view of model with dome

    You will edit some of the parameters of the dome to make the scene more realistic.
