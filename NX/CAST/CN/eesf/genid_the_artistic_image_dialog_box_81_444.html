The Artistic Image dialog box

The dialog box has four parts.

  1. Style preview icons

  2. Current Style name and Reset Defaults button

  3. Style-specific parameters

  4. Start Shade/Erase Shade buttons

The following styles are available.

Cartoon

Ink Print

Color Wash

Lines and Shadows

Pencil Shade

Rough Pencil

Hand Drawn

Stipple

The Start Shade button renders the model using the selected style.
