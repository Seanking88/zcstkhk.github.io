Use the 2D Viewer

  1. In the Display Settings group, clear the Show Manipulator check box.

  2. In the Section Curve Settings group, change the Color to red.

  3. In the 2D Viewer Settings group, select the Show 2D Viewer check box.

    You can measure geometry, zoom, and pan in this view.

    Use the four navigation buttons in the 2D Viewer Settings group to change the orientation of the view.

  4. Select the Shaded Cap Preview in 2D View check box.

  5. Click the Rotate Right, Rotate Left, Reflect X-axis, and Reflect Y-axis buttons to see what they do. When done, use the same buttons to restore the view as shown above.

  6. In the 2D Viewer Settings group, clear the Show 2D Viewer check box.

  7. Click OK.
