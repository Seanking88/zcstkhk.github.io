Use Image-Based Lighting

  1. On the Reflection tab, set Reflection Map to Image-Based Lighting.

  2. On the Global Illumination tab, set the following:

    Environment Image group:

    • Choose Image File = des07_kitchen_probe.hdr (from the parts directory)

      Note

      If you do not see any hdr images, make sure you are filtering the list for .hdr images.

    HQI Settings group:

    • Use Image-Based Lighting in HQI =

    • Intensity = .5

  3. Click OK.

  4. Click Start Shade to test the reflection effect.

    Note the lack of shadowing. This is because when you turned on Use Image-Based Lighting in HQI on the Global Illumination page, the High Dynamic Range image you applied actually lights the scene.

    Any definitions of Basic Lights or Advanced Lights are overridden.

  5. Close the part.
