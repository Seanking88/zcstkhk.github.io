优化大型装配的制图首选项设置

要使内存用量降至最低并加快大型装配制图视图的放置和更新速度，请考虑使用下列首选项。

首选项→制图→常规选项卡：

在独立图纸工作流下，

  • 自动启动投影视图命令 – 关

在基于模型的图纸工作流下，

  • 自动启动投影视图命令 – 关

首选项→制图→预览选项卡：

  • 样式→边界

首选项→制图→视图选项卡：

  • 延迟视图更新 — 开

  • 创建时延迟更新 — 开

首选项→视图→常规选项卡：

  • 表示 = 智能轻量级

  • 分辨率 = 粗糙

  • 公差 — 如果使用精确视图，则放宽公差可提升性能，但这样会导致曲线边的准确度下降。

  • 快照 = 开

  • 轮廓线 — 关

  • 自动更新 — 关

  • 引用 — 开

  • 检查边界状态 — 关

  • UV 栅格 — 关

  • 中心线 — 关

  • 不要使用定义渲染集(需要额外更新。)

首选项→视图→隐藏线选项卡：

  • 隐藏线 — 关

  • 被边隐藏的边 — 开

  • 自隐藏 — 开

  • 小特征 = 隐藏(可能情况下)

  • 干涉实体 = 无

首选项→视图→光顺边选项卡：

  • 光顺边 — 开，且无 CFW 集

首选项→视图→虚拟交线选项卡：

  • 虚拟交线 — 关

首选项→视图→剖视图选项卡：

  • 背景 — 取决于所需的结果，“关”时更快

  • 装配剖面线 — 初始为“开”，然后在创建视图之后为“关”。

  • 如果不涉及剖面线空间变动，则“关”更快

  • 隐藏剖面线 — 关

  • 剖面线 — 关。(剖视图中的剖面线会增加视图的再生成时间。)

首选项→视图→可见线选项卡：

  • 可见线 — 保持所有线为原先的

首选项→视图→螺纹选项卡：

  • 螺纹标准 = 无

首选项→视图→着色选项卡：

  • 渲染样式 = 线框
