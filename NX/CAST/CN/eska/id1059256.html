拉伸草图

  1. 右击草图 (5)并选择拉伸 。

  2. 在拉伸对话框中，设置如下参数：

    方向组：

    • 指定矢量：确保矢量指向实体内部

    限制组：

    • 开始 = 值

    • 距离 = 0

    • 结束 = 值

    • 距离 = 5

    布尔组：

    • 布尔 = 自动判断

    偏置组：

    • 偏置 = 单侧

    • 结束 = 5

  3. 点击确定以创建拉伸。

  4. 在部件导航器中，右击拉伸 (6)。

    注意到使草图为内部的选项不再可用。

    草图 (5)现在含有两个子特征，拉伸 (6)和拉伸 (8)，草图不能成为这些特征中任意一个的内部草图。
