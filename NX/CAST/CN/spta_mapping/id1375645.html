Set up a mapping solution

  1. (Optional) Create one or more mapping constraints in the source solution.

  2. (Optional) Create one or more Mapping Target Set constraints in the target solution.

  3. Solve the thermal or flow (source) model to obtain the results you want to map.

  4. Open the target model Simulation file, then, in the Simulation Navigatorright-click the Simulation node and choose New solution to create the target mapping solution.

  5. In the Solution dialog box, choose the appropriate solver from the Solver list and Mapping from the Analysis Type list.

  6. Fill out the options on both the Mapping Details tab and the Optional Output tab of the Solution dialog box.

  7. (Optional) If you have created one or more Zone constraints in the source solution, create matching Zone Association constraints in the target solution.

  8. (Optional) If you have created one or more Transverse Gradient constraints in the source solution, create matching Transverse Target Set constraints in the target solution.
