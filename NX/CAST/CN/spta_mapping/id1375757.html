Optional output formats

Select the output format to generate in your mapping solution in the Optional Output tab of the Solution dialog box.

  • The default output of a mapping analysis is set to generate in Nastran format.

  • You can control model numbering with the Nastran SID Start Number and Nastran SID Increment options.

Additional output options are also available:

  • Select Ansys, Abaqus, and/or TMG check boxes to produce mapped output in the XML format used by these solvers.
