Help repositioning the cover hole

Here is one technique you can use:

  • Use the Expressions dialog box to examine the expressions that define the location of the hole, and the length and width of the rectangular extrusion.

  • When you have identified the expressions that are used to position the hole, modify them to be half the current values for the length and width of the extrusion.

Note

If you still need a solution, import the expressions in exp4_cover_plate.exp.
