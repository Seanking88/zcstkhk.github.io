Define additional tool pocket junctions

In this activity, you will define additional pocket junctions for your tool rack.

  1. Define additional pockets using the following activities in this tutorial:

    • Define the tool pocket mount junction

    • Add a safe move junction to the pocket

    • Add additional junctions to the pocket

    Note

    • Name each tool similarly to the first. Name the junction identifiers sequentially but keep the simulation numbers the same. For example, name the second tool POCKET_TWO, and its junctions T2, P2_SIM_ENTRY1, P2_SIM_ENTRY2, and P2_SIM_EXIT1.

    • In the Edit Machine Component dialog box, in the Holder Details section, name each pocket sequentially. By default the first pocket holder ID was 1, so name the second 2, and so on.

    When your machine and probe heads are used in inspection programs, probe tools will load to the pockets in the order that the pocket definitions appear in the Machine Tool Navigator.

  2. When finished, save sheffield_rl_50_tr.prt and exit NX.

The machine file is now complete.
