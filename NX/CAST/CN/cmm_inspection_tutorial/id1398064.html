Define line features for your next alignment

In this activity, you will create a line feature that will serve as a design feature for a constructed line feature. You will use these lines in the next step of your alignment strategy.

  1. In the Inspection Navigator, in the FEATURES program group, select the FRONT_FACE inspection feature.

  2. Hold down your Ctrl key, and in INSPECTION_PATHS program group, select the PATH_FRONT_FACE inspection path.

  3. Right-click either selection and choose Cut.

  4. Right-click the PART_ALIGNMENT program group and choose Paste Inside.

    FRONT FACE and PATH_FRONT_FACE appear at the bottom of the PART_INSPECTION program group.

    These elements define and then measure a plane that is perpendicular to the top plane.

  5. Create a constructed line feature that will use the actuals of the two measured planes.

    1. Choose Insert→Constructed Feature.

    2. In the Constructed Feature dialog box, from the Feature Type list, select Line.

    3. From the Construction Form list, select Intersection.

    4. From the Design Feature Mode list, select Create New.

    5. In the Design Feature Name box, type POS_XLINE.

    6. In the Measured Plane or Line group, from the Available Features list, select TOP_PLANE_A.

    7. In the Intersecting Feature group, from the Available Features list, select FRONT_FACE.

    8. In the Operation Name box, type CONST_XLINE, and click OK.

    9. In the Inspection Navigator, from the CONSTRUCTED_FEATURES program group, select CONST_XLINE.

    10. Hold your Ctrl key, and from the FEATURES program group, select POS_XLINE.

    11. Drag CONST_XLINE and POS_XLINE onto the PART_ALIGNMENT program group and save your inspection setup file.
