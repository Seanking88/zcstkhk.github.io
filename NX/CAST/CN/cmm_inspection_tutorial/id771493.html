Load a virtual CMM machine

This example shows you how to load a generic CMM machine, the probe head that connects a probe to the machine, and the probe itself.

  1. If you have not already done so, create an inspection setup file.

  2. On the Navigator toolbar, click Machine View .

  3. In the Inspection Navigator, double-click GENERIC_MACHINE.

  4. In the Generic Machine dialog box, load a virtual CMM by doing the following.

    1. Click Retrieve Machine from Library .

    2. In the Library Class Selection dialog box, from the Machine list, double-click CMM.

    3. In the Search Result dialog box, from the Matching Items list, double-click the preferred machine.

  5. In the Part Mounting dialog box, position the part to be inspected on the CMM by doing the following:

    1. From the Placement group, select Use Part Mount Junction.

    2. Click CSYS dialog .

    3. In the CSYS dialog box, from the Type list, select Absolute CSYS and click OK.

    4. In the Part Mounting dialog box, click Select Part , and in the graphics window, click the solid body of your inspection part.

    5. Click OK.

      An information window reports the status of your machine load, including the location of the machine and related objects.

    6. Close the information window.

      The inspection part has loaded on the CMM machine, and the Generic Machine dialog box is renamed to match your selected CMM machine description.

      Note

      If the inspection part is not visible or is not placed correctly, you can later change its position by starting the Assemblies application and using Move Component in the Assemblies toolbar.

  6. In the renamed dialog box, load the head that will rotate the probe around your inspection part by doing the following.

    1. Click Retrieve Devices from Library .

    2. In the Library Class Selection dialog box, double-click Head.

    3. In the Search Result dialog box, in the Matching Items group, double-click the preferred probe head.

  7. In the renamed dialog box, load the probe that will connect to the probe head by doing the following.

    1. Click Retrieve Tools from Library .

    2. In the Search Criteria dialog box, click OK.

    3. In the Search Result dialog box, in the Matching Items group, select one or more probes and click OK.

  8. In the renamed dialog box, click OK.

    The machine, head, and probe are now loaded and appear in the Inspection Navigator - Machine view. The probe is not visible until you run inspection path or machine simulations.
