View a complete probe tool assembly

In this activity, you will view and examine a complete probe file that represents the probe tool you will soon be constructing.

  1. If you have not already done so, download the tutorial training materials and set up your assembly load options.

  2. Choose File→Open and in the following folder

    [directory]\Renishaw_Probe_Library\Metric\Defined_Probes\

    where [directory] is the directory where you unzipped the library, open LA_2237_0355_C_SP25M_example.prt.

    Although a probe tool can be a single solid body, this tool consists of multiple assembly components, and represents the probe tool you will create in later activities. You can apply these activities to other probe types, including single stemmed probes.

  3. On the Resource bar, click the Assembly Navigator tab, and click individual components to identify them in the graphics window.

    Stylus 1

    Stylus hub

    Stylus 2

    Extension

    Stylus 3

    Force module

    Stylus 4

    Body

    Stylus 5

    SP25

  4. Choose File→Close→All Parts. If the Close All Files dialog box appears, click No - Close.
