Add the tool rack to your CMM machine copy

In this activity, you will modify your machine copy by adding a tool rack as a component.

  1. In NX, open the machine assembly file

    [install directory]\Siemens\NX x.x\CMM_INSPECTION\resource\library\machine\installed_machines\sheffield_rl_50_tr\graphics\sheffield_rl_50.prt

  2. Choose File→Save As, and in the Save As dialog box, in the sheffield_rl_50_tr\graphics folder, save the file as sheffield_rl_50_tr.prt.

  3. Click the Machine Tool Navigator tab .

    Note

    If the tab is not visible, choose Start→All Applications→Machine Tool Builder.

  4. Choose Start and ensure that Assemblies is selected.

  5. On the Assemblies toolbar, click Add Component .

  6. In the Add Component dialog box, in the Part section, click Open .

  7. In the Part Name dialog box, select the tool rack file and click OK.

    [install directory]\Siemens\NX x.x\CMM_INSPECTION\resource\library\racks\ACR1_308_mm.prt

    The selected part file appears in the Component Preview window.

  8. In the Placement section, from the Positioning list, select Move.

  9. Ensure that Count is set to 1, Multiple Add is set to None, and Reference Set is set to Model (“MODEL”).

  10. Click OK.

  11. In the Point dialog box, from the Type list, select Point on Face, and select the point on the machine table where you want the tool rack to lie.

  12. Click OK, and in the Move Component dialog box, in the Transform section, from the Motion list, select Dynamic.

  13. In the graphics window, use the translation and rotation tools to move the tool rack to your preferred position, and then click OK.

  14. Choose File→Save.

The rack is now added to your CMM part model, and is ready to be integrated with the CMM machine.
