Export your probe tool to a library

In this activity, provided you have rights to do so, you will add your probe to the default CMM Inspection Programming library.

  1. In the Probe dialog box, click on the Tool tab.

  2. On the Tool tab, do the following.

    1. In the Description group, using 132 characters or less, type a description.

      This is the name that will appear in the second column of the library’s search results, and will be appended to the Library Reference (see below) to identify the tool in the Inspection Path dialog box.

    2. In the Numbers group, in the Tool Number box, accept the default value of 1.

    3. In the Information group, in the Catalog Number box, using 132 characters or less, type Original Equipment Manufacturer (OEM) part number reference information.

      SP25_SH2_A5000_7610_7630_STAR

    4. In the Library group, in the Library Reference box, leave the box blank to let the application automatically assign a unique name for your probe.

      This is the name that will appear in the first column of your probe library search results, and in the Tool column of the Inspection Navigator. The name also appears in the Inspection Path dialog box, followed by the Description.

  3. In the Library group, click Export Tool to Library .

  4. If an EXPORT TO LIBRARY dialog box appears warning you that a similar tool exists in the library, click Create.

  5. In the Template Attributes dialog box, from the Holding System list, select a holder value of 310(Steep Taper 40).

  6. Click OK.

    NX copies the assembly to the tool library graphics area, creating a new folder under

    [NX install location]\MACH\resource\library\tool\graphics\

    where [NX install location] is the location where you installed NX. NX also creates a tool data record in the tool database that references the probe graphics files. The record contains information similar to the following:

    TP6_35EXT_A-5003-0060-01-A_4MM | 04 | 02 | TP6 x 35EXT x 36EWL x 4mm DIA | TMC0_00041 | Ruby | 1 | 310 |

  7. If an Export Tool to Library dialog box appears, click OK.

Your probe is now exported to the default CMM Inspection Programming library and ready for use. Leave the Probe dialog box open for the next activity.
