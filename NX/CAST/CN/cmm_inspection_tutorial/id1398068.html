Define a measurable point on the last axis

In this activity, you will create a constructed point feature from your original teach point, using the measured values of a third orthogonal plane and the constructed line feature you used in the last activity.

  1. In the Inspection Navigator, in the FEATURES program group, select the LF_SIDE_PLANE inspection feature.

  2. Hold your Ctrl key, and in the INSPECTION_PATHS program group, select the PATH_LF_SIDE_PLANE inspection path.

  3. Right-click either selection and choose Cut.

  4. Right-click the PART_ALIGNMENT program group and choose Paste Inside.

    LF_SIDE_PLANE and PATH_LF_SIDE_PLANE appear at the bottom of the PART_INSPECTION program group.

    The left-side plane is perpendicular to the top and front planes.

  5. Using the teach point as a design feature, create a constructed point feature at the location where the left plane feature and the line feature, used in the Y origin and PCS X rotation, intersect.

    1. In the Inspection Navigator, in the PART_ALIGNMENT program group, right-click TEACH_POINT and choose Insert→Constructed Feature.

    2. In the Constructed Feature dialog box, in the Constructed Feature group, from the Construction Form list, select Intersection.

    3. In the Measured Plane or Line group, from the Available Features list, select POS_XLINE.

    4. In the Intersecting Feature group, from the Available Features list, select LF_SIDE_PLANE.

    5. In the Operation Name box, type CONST_XORG_PT and click OK.

    6. In the Inspection Navigator, from the CONSTRUCTED_FEATURES program group, select CONST_XORG_PT and drag it onto the PART_ALIGNMENT program group.

    7. Save your inspection setup file.

The teach point can now be used to set the PCS X-origin.
