Complete and save the final alignment

In this activity, you will define an alignment command to complete the PCS alignment.

  1. On the Insert toolbar, click Alignment .

  2. In the Alignment dialog box, from the Type list, choose Translate PCS Origin.

  3. In the XP group, from the Method list, select Measured Feature.

  4. From the Select Feature list, choose TEACH_POINT.

  5. In the PCS Name box, type PCS_4HOLE_FINAL.

  6. In the Operation Name box, type TRANS_XORG, and click OK.

    The last degree of freedom is now constrained and the alignment is complete.

  7. On the Insert toolbar, click Alignment .

  8. In the Alignment dialog box, from the Type list, choose Save PCS.

  9. In the Save Parameters group, from the PCS Name list, select PCS_4HOLE_FINAL.

  10. In the Device Identifier list, type PCS_4HOLE_FINAL.

  11. In the Operation Name box, type SAVE_FINAL_PCS and click OK.

    The aligned PCS is now saved and can be recalled if you need to modify it later in your program.

  12. Right-click CMM_PROGRAM, choose Generate, and in the Generate Tool Path dialog box, click Accept Paths.

  13. Double-click CONST_XLINE, click OK, and generate again.

    The program now generates correctly.

  14. Right-click CMM_PROGRAM, choose Post Process, and with the DMIS 5.2 postprocessor selected, click OK.

    If a message reports that a file of the same name exists, click OK.

  15. Save your inspection setup file.

In the Information window, review the DMIS output. For information on the commands you created for part alignment, see the Dimensional Measuring Interface Standard, Part 1, Revision 5.2, ANSI/DMIS 105.2-2009.

Note

If you run simulations on this program you will notice machine limit violations, collisions, and incorrect angles. You can further experiment with correcting these by referring to the CMM Inspection Programming Online Help.
