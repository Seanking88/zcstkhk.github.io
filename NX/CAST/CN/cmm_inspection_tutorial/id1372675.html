打开检测设置文件并链接到 PMI

在这个练习中，您将打开包含要对齐的部件文件的检测设置文件，您将加载所有设备组件，然后将检测设置文件链接到部件文件的产品制造信息(PMI)数据。

  1. 如果您还没有进行这些步骤，请下载训练材料并设置装配加载选项。

  2. 选择文件→打开并在下面的文件夹中打开 4hole_pmi_plane_line_pt.prt：

    [路径]\alignment_2011_06_10\

    其中[路径]是您解压对齐文件的目录。

  3. On the Navigator toolbar, click Machine View .

  4. Load a virtual CMM.

    1. In the Inspection Navigator, double-click GENERIC_MACHINE.

    2. In the Generic Machine dialog box, click Retrieve Machine from Library .

    3. In the Library Class Selection dialog box, in the Machine list, double-click CMM.

    4. In the Search Result dialog box, from the Matching Items list, double-click dea_swift.

    5. In the Part Mounting dialog box, in the Placement group, select Use Part Mount Junction.

    6. Click CSYS Dialog .

    7. In the CSYS dialog box, from the Type list, select Absolute CSYS, and click OK.

    8. In the Part Mounting dialog box, click Select Part , and in the graphics window, click the solid body of the four-hole inspection part.

    9. Click OK.

      An Information window reports the status of your machine load, including the location of the machine and related objects.

    10. Close the Information window. The CMM is now loaded.

  5. Load a probe head.

    1. In the 3 Axis Gantry Cmm dialog box, in the Library group, click Retrieve Devices from Library .

    2. In the Library Class Selection dialog box, double-click Head.

    3. In the Search Result dialog box, in the Matching Items list, double-click Renishaw_PH10M.

  6. Load a probe.

    1. In the Library group, click Retrieve Tools from Library .

    2. In the Search Criteria dialog box, click OK.

    3. In the Search Result dialog box, in the Matching Items group, double-click SP25_SH2_A-5003-0063-01-A.

      The probe does not appear in the graphics window but will display when you run animations.

  7. Click OK to close the 3 Axis Gantry Cmm dialog box.

  8. On the Navigator toolbar, click Program Order View .

  9. On the Operations toolbar, click Link to PMI .

  10. In the Link PMI dialog box, accept the default settings (Create Paths = Yes, Tool = Auto) and click OK.

    An Information window reports the status of all PMI notations that were successfully linked to your inspection program.

  11. Close the Information window and choose File→Save.

The Inspection Navigator now contains sensors, features, inspection paths, tolerances, and outputs that are automatically generated from PMI. You are now ready to set up your program and begin the process of defining part alignment.
