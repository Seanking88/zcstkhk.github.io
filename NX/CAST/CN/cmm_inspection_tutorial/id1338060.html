Begin constructing your probe tool

In this activity, you will create a new inspection setup file from the Probe Tool template.

  1. Choose File→Open.

  2. In the Open dialog box, navigate to

    [directory]\Renishaw_Probe_Library\Metric\SP25M\

    where [directory] is the directory where you unzipped the library.

  3. Select LA_2237_0355_C_SP25M_body.prt and then click OK.

  4. Choose File→New.

  5. In the New dialog box, click the Inspection tab.

  6. In the Templates group, from Units, select Millimeters.

  7. From the Templates list, select Probe Tool.

  8. In the New File Name group, in the Name box, type a name for your probe setup file.

    This is not the name as it will appear when selecting it from the library, but the name of your working file.

  9. Click Folder and in the Choose Directory dialog box, navigate to a location or create a new subfolder within the folder you designated in your load options, and then click OK.

  10. In the New dialog box, click OK.

    You have loaded the first component of the probe assembly.
