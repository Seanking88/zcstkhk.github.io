Use existing head and machine assemblies as templates

In this activity, you will create copies of a probe head and CMM machine that come installed with CMM Inspection Programming. You will use these copies to create a new machine with a virtual tool rack, as well as a probe head that can access that tool rack.

  1. Copy the folder

    [install directory]\Siemens\NX x.x\CMM_INSPECTION\resource\library\device\graphics\Renishaw_PH10M

    where [install directory] is the location where you installed NX and x.x is the software version.

  2. Move up one directory, paste the folder, and rename it Renishaw_PH10M_tr.

  3. Cut and paste the new folder back to the graphics directory.

  4. Copy the folder

    [install directory]\Siemens\NX x.x\CMM_INSPECTION\resource\library\machine\installed_machines\sheffield_rl_50

  5. Move up one directory, paste the folder, and rename it sheffield_rl_50_tr.

  6. Cut and paste the new folder back to the installed_machines directory.

  7. In the folder

    [install directory]\Siemens\NX x.x\CMM_INSPECTION\resource\library

    create a new folder, racks.

  8. Download the tutorial training file ACR1_308_mm.zip and unzip the file to

    [install directory]\Siemens\NX x.x\CMM_INSPECTION\resource\library\racks

You are ready to modify your new probe head assembly.
