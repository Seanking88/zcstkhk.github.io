Change from manual mode to program mode

In this activity, you will set the inspection program to change from manual mode to program plus manual mode, and inform the machine operator of the change.

  1. In the PART_ALIGNMENT program group, right-click MODE_MANUAL and choose Copy.

  2. Right-click the PART_ALIGNMENT program group, and choose Paste Inside.

    PART_ALIGNMENT

     DEFINE_MCS

     TEACH_POINT

     OPER_MSG_TP

     MODE_MANUAL

     TEACH_POINT_PATH

     MODE_MANUAL COPY

  3. Double-click MODE_MANUAL_COPY.

  4. In the Command Parameters group, select the Program check box.

    Program plus manual mode instructs the machine to execute measurements without intervention from the operator unless a problem occurs that the machine cannot solve.

  5. In the Operation Name group, clear the existing name, type MODE_PROGMAN, and click OK.

  6. In the PART_ALIGNMENT program group, right-click OPER_MSG_TP and choose Copy, and then right-click MODE_PROGRAM and choose Paste.

    PART_ALIGNMENT

     DEFINE_MCS

     TEACH_POINT

     OPER_MSG_TP

     MODE_MANUAL

     TEACH_POINT_PATH

     MODE_PROGMAN

     OPER_MSG_TP_COPY

  7. Double-click OPER_MSG_TP_COPY.

  8. In the Insert Command dialog box, in the Program Statements box, clear the existing statement and type:

    TEXT/OPER,'Press OK to begin inspection.'

  9. In the Operation Name box, type OPER_MSG_BEGIN_INSP, and click OK.

  10. Save your inspection setup file.

The program is set to run programmatically based on upcoming parameters.
