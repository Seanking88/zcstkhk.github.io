Modify the probe head assembly

The probe mount component of a standard machine head in CMM Inspection Programming is ordinarily classified as a device holder. In this activity, you will remove this classification from your probe head copy since the tool rack pockets will instead hold the probes.

  1. In NX, open the machine head assembly file.

    [install directory]\Siemens\NX x.x\CMM_INSPECTION\resource\library\device\graphics\Renishaw_PH10M_tr\Renishaw_PH10M.prt

  2. Click the Machine Tool Navigator tab .

    Note

    If the tab is not visible, choose Start→All Applications→Machine Tool Builder.

  3. In the Machine Tool Navigator, expand the PH10M-HEAD node and double-click HEAD_A.

  4. In the Edit Machine Component dialog box, click and expand the Classify Component group.

  5. In the Classify Component section, clear _DYNAMIC_HOLDER and click OK.

    Note

    When using a tool rack, if the probe head is a dynamic holder, the first probe will load on the head and not the tool rack.

  6. Choose File→Save As, and in the Save As dialog box, in the Renishaw_PH10M_tr folder, save the file as Renishaw_PH10M_tr.prt.

  7. Click OK to close the Read-only Part message.

  8. Choose File→Close→All Parts.

The probe head can now be used to access tool rack pockets.
