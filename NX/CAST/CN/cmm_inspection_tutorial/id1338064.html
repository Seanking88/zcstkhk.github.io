Add remaining axial components

In this activity, you will add all remaining components that are coaxial with the probe body.

  1. Rotate the probe so that the area of the assembly, where the remaining extension and stylus components will be attached, is visible.

  2. In the Add Component dialog box, click Open .

  3. In the Part Name dialog box, navigate to

    [directory]\Renishaw_Probe_Library\Metric\Styli\M3\

    and select M-5000-3592-01-M.prt.

  4. Click OK, and in the Add Component dialog box, click Apply.

  5. Select the arc center of the interface point on the SH25 component.

  6. The M extension component is added to the probe setup assembly.

  7. In the Add Component dialog box, click Open .

  8. In the Part Name dialog box, select A-5000-7610-01-A.prt.

  9. Click OK, and then click Apply.

  10. In the graphics window, select the arc center of the interface point on the extension component.

    The stylus hub component is added to the probe setup assembly.

  11. In the Add Component dialog box, click Open .

  12. In the Part Name dialog box, select A-5000-3553-02-B.prt.

  13. Click OK, and then click Apply.

  14. In the graphics window, select the arc center of the interface point on the extension component.

    The first stylus component is added to the probe setup assembly.

All components that are coaxial with the probe body are now assembled. Leave the Add Component dialog box open for the next activity.
