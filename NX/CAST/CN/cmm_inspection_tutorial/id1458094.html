Activity: Defining a virtual tool rack

In this tutorial, you will:

  • Configure a machine head to work with tool racks.

  • Add a tool rack to a virtual CMM.

  • Define tool pocket junctions so that the CMM loads probe tools from the tool rack during simulations.

  • Add your new part files to the CMM databases and test your work.
