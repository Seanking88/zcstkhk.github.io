Download training materials and define load options

When working with libraries, parts, and assemblies that reside in different locations on your hard drive, you must define location paths so that they are found each time you open an inspection file. Do the following to download the appropriate training materials for your tutorials and to define assembly load options.

  1. Click the appropriate link to save the training materials to your local hard drive.

    Tutorial Course Materials

    Applying a plane/line/point alignment strategy

    alignment_2011_06_10.zip

    Constructing a CMM Probe

    Renishaw_2011_03_08.zip
    Defining a virtual tool rack

    ACR1_308_mm.zip

  2. Unzip or move the zipped file to a preferred location (for example, C:\Temp\).

  3. Choose File→Options→Assembly Load Options.

  4. Point to your tutorial files by doing the following.

    1. In the Assembly Load Options dialog box, in the Part Versions group, from Load, select From Search Folders.

    2. Next to Add Folder to Search, click Browse .

    3. In the Choose Search Directory dialog box, navigate to the root of the directory where you unzipped the training materials (for example, C:\Temp\).

    4. Click OK.

    5. In the Assembly Load Options dialog box, in the Show Session Folders list box, select the new path.

    6. Click in the Add Folder to Search box, move to the end of the path, and add three periods:

      [path]...
    7. In the Assembly Load Options dialog box, in the Saved Load Options group, click Save as Default .

  5. Unless you are taking the Constructing a CMM Probe tutorial, in the Assembly Load Options dialog box, point to the NX CMM Inspection folders by doing the following.

    NOTE: If you are taking the Constructing a CMM Probe tutorial, including NX CMM Inspection folders may accidentally pull geometry with similar names from previously defined probe components.

    1. In the Part Versions group, from Load, select From Search Folders.

    2. Next to Add Folder to Search, click Browse .

    3. In the Choose Search Directory dialog box, navigate to the CMM_INSPECTION folder in your NX install location (for example, C:\Program Files\Siemens\NX x.x\CMM_INSPECTION\resource\).

    4. Click OK.

    5. In the Assembly Load Options dialog box, in the Show Session Folders list box, select the new path.

    6. Click in the Add Folder to Search box, move to the end of the path, and add three periods:

      [path]...
    7. In the Assembly Load Options dialog box, in the Saved Load Options group, click Save as Default .

  6. Click OK.

CMM Inspection Programming will now search the library and tutorial directories each time you start the application.
