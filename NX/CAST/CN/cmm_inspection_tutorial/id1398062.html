Add teach point commands

In this activity, you will add commands that instruct the machine operator to manually verify the teach point location.

  1. In Inspection Navigator, right-click the TEACH_POINT feature and choose Insert→Inspection Path.

  2. On the Measurement tab, in the Sub-Operations group, click Add Sub-Operation .

  3. In the Create Sub-Operation dialog box, from the Type list, select Measured Point.

  4. In the Geometry group, click Inferred Point and select the same point that you used for the teach point as shown.

  5. Click OK.

  6. In the Inspection Path dialog box, click the Transitions tab.

  7. In the Operation Name box, type TEACH_POINT_PATH and click OK.

    In the Inspection Navigator, the point measurement appears as the last item in the INSPECTION PATHS program group.

  8. In Inspection Navigator, drag TEACH_POINT_PATH from the INSPECTION_PATHS program group below MODE_MANUAL in the PART_ALIGNMENT program group.

    PART_ALIGNMENT

     DEFINE_MCS

     TEACH_POINT

     OPER_MSG_TP

     MODE_MANUAL

     TEACH_POINT_PATH

  9. Save your inspection setup file.

The machine operator is set to measure the teach point, which serves as a reference to connect the MCS with the upcoming PCS.
