Set your PCS Y origin and X alignment

In this activity, you will use the constructed line feature you created in the last activity to set the PCS Y origin and to align the X-axis of the PCS to that of the CMM.

  1. Set the PCS Y origin to align with the line feature origin on the part.

    1. On the Insert toolbar, click Alignment .

    2. In the Alignment dialog box, from the Type list, select Translate PCS Origin.

    3. In the YP group, from the Method list, select Measured Feature.

    4. From the Select Feature list, select POS_XLINE.

      POS_XLINE is obviously not directly measured. However, because it is the design feature for the constructed line feature CONST_XLINE, its measured location values are mathematically derived from the intersection of the two measured planes between which it lies.

    5. In the PCS Name box, type PCS_Y_ORIG.

    6. In the Operation Name box, type TRANS_Y_ORIG and click OK.

      The PCS is now translated to the line’s Y origin.

  2. Rotate the PCS about its Z-axis to align its X axis with that of the CMM.

    1. On the Insert toolbar, click Alignment .

    2. In the Alignment dialog box, from the Type list, select Rotate PCS.

    3. In the Rotation Parameters group, from the Method list, select Measured Feature.

    4. From the Rotation Axis list, select Z-axis.

    5. From the Select Feature list, select POS_XLINE.

    6. In the PCS Name box, type PCS_XDIR.

    7. In the Operation Name box, type ROT_ALIGN_XDIR and click OK.

    8. Save your inspection setup file.

You have constrained two more degrees of freedom by halting any movement along the Y-axis and aligning the X-axis of the PCS with the X-axis of the machine.
