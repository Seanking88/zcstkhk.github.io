Law Extension enhancements

Estimated time to complete this lesson: 8–11 minutes

What is it?

The Law Extension command is enhanced in the following ways:

  • The Method list has been added to the Spine group. You can now define a spine by specifying a curve or a vector.

  • The Miter group has been added so that you can specify no miter, a sharp miter, or a blend when creating law extensions with adjacent base profile curves.

  • The Laydown Base Profile option has been added to the Settings group. This option ensures that the base profile for the law extension is projected onto the supporting face to avoid any out-of-tolerance issues between the base profile and the constraint face.

  • The Advanced Curve Fit option has been added to the Settings group.

    The following Method options are available when Advanced Curve Fit is selected:

    • Degree and Segments – Specifies the degree and segment of output surfaces.

    • Degree and Tolerance – Specifies the maximum degree and the tolerance of output surfaces.

    • Keep Parameterization – Inherits the degree, segments, pole structure, and the knot structure from the input surface and applies it to the output surface.

    • Auto Fit – Specifies the minimum degree, the maximum degree, the maximum number of segments, and tolerance to control the parameterization of the output surface.

    The Advanced Curve Fit option replaces the Rebuild options of previous releases.
