Define the extension profile and references

  1. Open des19_85_law_extension.

  2. On the Surface toolbar, from the Flange Surface Drop-down list, select Law Extension .

  3. Confirm that Type is set to Faces.

  4. On the Selection bar, from the Curve Rule list, select Single Curve.

  5. In the Base Profile group, with Select Curve highlighted, select the four face edges as shown.

  6. Click the middle mouse button to continue.

  7. On the Selection bar, from the Face Rule list, select Single Face.

  8. In the Reference Faces group, with Select Face highlighted, select the four faces as shown.

  9. In the Reference Faces group, click Reverse Direction to have the extension surfaces point outward.
