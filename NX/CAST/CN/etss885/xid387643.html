Fit freeform, cylindrical, and conical surfaces to facet bodies

  1. Open des19_85_fit_knob_facets.

  2. Choose Insert→Surface→Fit Surface.

  3. In the Fit Surface dialog box, make sure Type is set to Fit Freeform .

  4. With Select Object highlighted, select the top facet of the knob.

  5. Click Apply.

  6. In the Fit Surface dialog box, from the Type list, select Fit Cylinder .

  7. In the Fit Conditions group, clear the Closed check box.

  8. Select the facet body of the first cylindrical indentation.

  9. Click Apply.

  10. Repeat the above procedure to fit and apply three more open cylinders to the other three indentations.

  11. With Type still set to Fit Cylinder, select the center cylinder

  12. Click Apply.

  13. In the Fit Surface dialog box, from the Type list, select Fit Cone .

  14. In the Fit Conditions group, make sure the Closed check box is cleared.

  15. Select the outside surface as shown.

  16. Click Apply.
