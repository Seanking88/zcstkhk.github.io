Blend the first set of fitted surfaces

  1. On the Feature toolbar, click Face Blend

  2. Select the fit freeform surface of the top of the knob as face chain 1.

  3. Make sure that the direction vector points to the center of the blend. Click Reverse Direction if necessary.

  4. In the Face Blend dialog box, in the Face Chains group, click Select Face Chain 2.

  5. Select the large open fit cone surface.

  6. Make sure that the direction vector of the blend points inward. Click Reverse Direction if necessary.

  7. In the Cross Section group, confirm or set the following options:

    • Section Orientation = Rolling Ball

    • Shape = Circular

    • Radius Method = Constant

    • Radius = 10

  8. Click Apply.

  9. With Select Face Chain 1 highlighted, position the cursor over the center hole and select the internal cylinder from the QuickPick dialog box.

  10. Click the middle mouse button to advance.

  11. On the Selection bar, from the Face Rule list, select Single Face.

  12. With Select Face Chain 2 highlighted, select the top freeform fit surface.

  13. Make sure the direction vector is pointing to the center of the intended blend.

  14. In the Cross Section group, in the Radius box, type 5.

  15. Click Apply.

  16. With Select Face Chain 1 highlighted, select the fitted open conical surface.

  17. Click the middle mouse button, then select one of the open cylindrical fit surfaces that make up the finger indentations of the knob.

  18. In the Cross Section group, in the Radius box, type 12.

  19. Click Apply.

  20. Repeat the blending procedure above to create face blends for the other three indentations. Click OK for the last blend to close the Face Blend dialog box.

  21. On the Selection bar, from the Type Filter list, select Facet Body.

  22. Select each fit surface that was used for blending, right-click and select Hide.
