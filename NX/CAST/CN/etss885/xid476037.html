Specify the length, spine, and miter of the law extension

  1. In the Length Law group, in the Value box, type 30 and Enter, or drag the on-screen Value handle to a value of 30.

  2. Rotate around the part to check the extension.

    You want all extension surfaces to be flush with the rectangular end condition.

    The Angle Law for the extension surfaces is set to 90 degrees which is calculated from the duct transition surfaces.

    You must specify a spine to orient the extension surfaces.

  3. In the Spine group, from the Method list, select Vector.

  4. With Specify Vector highlighted, specify a vector in the Z direction.

  5. In the Miter group, from the Method list, select None.

  6. From the Method list, select Sharp.

  7. From the Method list, select Blend to return to the default condition.
