使用 Teamcenter Integration for NX 创建非主模型图纸

使用 Teamcenter Integration for NX 来创建非主模型图纸的过程与在原生环境中创建图纸id过程是类似的。但是，您可能需要从数据库中选择引用(主模型)部件，而不是从文件夹中，您可能还需要知道它与主模型部件的关系。

  1. 点击文件→新建。

  2. 点击新建对话框中的图纸选项卡。

  3. 选择其中一个图纸模板。

  4. 使用默认名称或者在数据集名称输入框中键入想要的值。

    如果您选择的是空白图纸模板，您还需要在关系列表中指定与主模型部件的关系，如果您选择了一个已建立关系的模板，则不会显示该列表。

  5. 工作部件默认显示在对话框的引用部件组。如果工作部件中不包含主模型，点击浏览。

  6. 点击确定以创建非主模型图纸，将打开制图应用模块并显示模板。

  7. 选择文件→保存以保存非主模型图纸。非主模型图纸文件将被添加到数据库中，并与引用部件关联。第一次保存该项时，将打开命名部件对话框，以便您可以在将它保存到 Teamcenter 数据库中之前查看并修改名称、位置以及属性。
