用户定义的焊接对话框

体

选择体

让您选择一个实体来创建用户定义的焊接。

注释

您可以只选择一个实体，但必须位于当前工作部件中。

边

选择边

让您从实体中选择一个或对个边。

这些边将用来计算焊接长度。

相连部件

选择面

让您选择用户定义的焊接中表示相连部件的面。

这些面将用来抽取相连部件的属性。

焊接特性

属性列表

列出其它您可以指派给用户定义的焊接的属性。

这些属性在用户默认设置对话框的焊接助理部分里设置。

参见在线帮助以获取更多关于焊接特性的信息。

提示

要查找某个用户默认设置，选择文件→实用工具→用户默认设置，然后点击查找用户默认设置 。

从对象继承

让您选择一个对象，用户定义的焊接将继承该对象的属性。

设置

指派焊接 PMI

将在您点击用户定义的焊接对话框中的确定后打开焊接符号对话框。

您可以在创建用户定义焊接的同时创建焊接 PMI 符号。
