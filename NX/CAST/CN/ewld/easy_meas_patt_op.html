Parallel, radial sections are created along the YC direction.

Parallel ZC Planes

Parallel, radial sections are created along the ZC direction.

Spacing Method

Available when Plane Method is set to any option except Infer Planes.

Specify one of the following options:

  • Grid

  • Single Plane

Grid

Single Plane

Grid Increment

Available when Spacing Method is set to Grid.

Specifies the spacing between grid planes.

Example

Suppose the specifications are as follows:

  • The pattern path curve extends from X=-5 to X=125.

  • Plane Method = Parallel XC Planes.

  • Grid Increment = 50.0

Then the section planes are defined at XC=0, XC=50, and XC=100.

Plane Location

Available when the Spacing Method list is set to Single Plane.

Specifies the coordinates for locating the plane.

Number Surface Vectors

Specifies the number of surface vectors.

If you specify a value of 1, the surface vector is computed based on the Trim Edge Offset value.

Trim Edge Offset

Specifies the location of the first surface vector.

Back Edge Offset

Specifies the location of the last surface vector.

Hem Method list

Available when Type is set to Hem and Surface.

  • Mid Point — Calculates the surface vector at the midpoint of a curve on the hem face.

  • Normal to Body — Calculates the surface vector at the intersection of the guide curve and the body coordinate perpendicular to the plane.

Maximum Spacing

Lets you set the maximum value for the spacing between the first surface vector and the second surface vector if the number of surface vectors is 2.

If the width of the flange is greater than the value of the Trim Edge Offset plus the value of Maximum Spacing, then the second surface vector is created at a distance that is equal to the value of the Trim Edge Offset plus the value of Maximum Spacing, instead of at the specified Back Edge Offset.

Example

Flange width < Trim Edge Offset + Maximum Spacing.

Flange width > Trim Edge Offset + Maximum Spacing.
The second surface vector is created at a distance that is equal to Trim Edge Offset + Maximum Spacing.

Settings

Tolerance

Specifies the Distance Tolerance. For more information, see the online help for Modeling Preferences.

Length and Width

Specifies the length and width for the base of the solid object that is created.

Height

Specifies the height of the solid object that is created.

Grid Angle Tolerance

Available when Plane Method is set to Infer Planes or Spacing Method is set to Grid in the Parameters group.

Sets the angle tolerance between a path curve tangent and the grid plane used to create a measurement point.

Minimum Flange Width

Lets you set the minimum value for the width of a flange that surface measurement vectors are created on. If the actual width of the flange is less than the specified value, a single surface vector is created in the middle of the flange.

Example

Flange width at the location of the trim vector. At this location, width of the flange is less than Minimum Flange Width.

Only one surface vector is created.
