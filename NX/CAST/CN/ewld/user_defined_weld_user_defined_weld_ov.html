用户定义的焊接

使用这个命令来定义一个实体作为用户定义的焊接，这个命令在您想定义一个无法使用标准焊接特征方法创建的焊接时特别有用。

用户定义的焊接拥有与其它焊接特征相同的特性：

您可以：

  • 使用任何可用的建模方法来构建焊接形状

  • 对实体指派属性，以将它识别为焊接

    实体将在部件导航器中显示为一个焊接特征，并在焊接长度和焊接体积计算中使用。

  • 创建用户定义焊接的同时创建焊接 PMI 符号

  • 从焊接属性过滤器中访问用户定义的焊接

    当您把选择过滤器设置为焊接对象时，用户定义的焊接也会在细节过滤对话框中列出

  • 将用户定义的焊接发布到 Teamcenter中作为单独的焊接对象

  • 使用用户默认设置对话框来对用户定义的焊接定义特有的属性，包括可以从列表中选择值以及指定需要的属性，并定义属性的默认值。

  • 从装配中移除选做相连部件的部件文件

    这将使焊接特征变为保留，并在部件导航器中用保留符号标记。您可以通过选择装配中的新组件，来使焊接特征再次变为可用
