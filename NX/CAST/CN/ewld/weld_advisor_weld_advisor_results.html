焊接顾问结果

当检查完成后，信息将以树的格式显示在结果窗口中，您可以在树中的信息上执行操作，例如，指定排序选项。

要访问这些操作的菜单，右击结果对话框列标题区域的任意位置，并从快捷菜单中选择想要执行的选项。

可用的选项都包含在下表中：

操作

描述

根据点

结果按焊接 ID 排序。

按检查器

结果按检查器排序

按特征集

显示组织为特征集的点，这将提供对点的统一查看。

显示全部

按焊接 ID 顺序列出所有检查的点。

显示通过的对象

仅列出通过所有检查的点。

显示失败的对象

列出至少一项检查失败的点。

保存结果

写入两个属性到各个检查的点中，第一个属性是“标准”，它将列出点通过的检查，第二个属性是“非标准”，它将列出点失败的检查。

当每一次运行保存结果时这些属性将被覆盖。

注释

以下选项在选择一个焊点并右击时可用：

报告

为选定的点生成一个 XML 报告。

报告所有对象

为列表中的所有点生成 XML 报告。

中心对象

将选定的焊点移动至图形窗口中心。

编辑

打开焊点对话框，在此您可以编辑焊点和特征，当您完成编辑并选择确定或者应用后，将返回结果菜单。

删除

删除选定的焊点。
