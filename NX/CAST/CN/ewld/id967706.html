将焊接助理用户默认设置编辑为所需的

  1. 选择文件→实用工具→用户默认设置。

  2. 从用户默认设置对话框的左侧面板列表中选择焊接助理。

  3. 从展开的焊接助理列表中，选择公共。

  4. 查看在常规选项框中可以更改的默认设置，注意到您可以指定 ID 范围以及颜色。

  5. 点击组 ID 颜色选项卡，查看默认指派给各种组 ID 的颜色。

  6. 从展开的焊接助理列表中，选择创建，注意到有很多个选项卡，在这些选项卡中您可以指定标准焊接特征类型的默认设置。

  7. 点击坡口焊选项卡，并注意选择的是何种装配数据链接方法。(参见在线帮助以获取更多有关这两种方法的信息)

  8. 在装配数据的链接方法下方，点击选择 WAVE 链接实体。

  9. 点击其它选项卡并查看各种焊接类型可以更改的默认设置。

  10. 从展开的焊接助理列表中，选择定制焊点，并查看其中的默认设置。

    注意可能的菜单项。

  11. 从展开的焊接助理列表中，选择焊接顾问。

  12. 点击电阻焊点焊选项卡，并查看这个焊接的默认设置。

  13. 继续查看其它选项卡的默认设置。

  14. 从用户默认设置对话框左侧面板中选择制图。

  15. 从展开的制图列表中选择自动焊接符号，并查看其中的默认设置。

  16. 如果您已做出任何更改，记住要点击应用。

    如果您需要更改默认设置时遇到锁定，使您无法继续，询问计算机系统的管理员。
