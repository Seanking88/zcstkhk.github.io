More about resulting curves

  • General – a spline that exactly represents the original chain of curves

  • Cubic – a spline that approximates the original chain using a degree of 3 splines.

  • Quintic – a spline that approximates the original chain using a degree of 5 splines. Using Quintic will result in a smaller number of segments than those created with Cubic.

  • Advanced – a spline that approximates the original curve. This type of curve uses the maximum number of degrees and segment parameters as specified by the curve fit method designated in the Modeling Preferences dialog box.
