Excluding objects from boundary conditions

When you define loads or constraints, use the Excluded option to specify geometry or FE entities to exclude from the boundary condition definition.

For example, you can define a fixed constraint on a polygon face, and exclude the polygon edges.

Use the Excluded option to:

  • Avoid conflicts in constraints applied to shared geometry. You can prevent the software from creating a constraint along edges that are shared between faces.

  • Apply a load to geometry and exclude selected elements. You can then apply a different load to the excluded elements.
