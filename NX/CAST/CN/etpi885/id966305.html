Specify the origin point for the region

  1. In the Origin group, set Specify Point to Point on Face .

    This option will help you select the point that will be the origin of the region you define.

  2. Select the face of the extrude.

  3. In the Origin group, from the Anchor list, select Middle Right.

    The region location adjusts to the new anchor point.

    This is not the location that you want.

  4. In the Origin group, from the Anchor list, select Middle Center.

  5. To associate the region with the face, in the Settings group, click Select Face and then select the face.

  6. In the Settings group, in the Diameter box, type 18 and press Enter.

    The circular PMI region becomes larger.
