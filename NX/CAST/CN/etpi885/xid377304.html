PMI user interface enhancements

What is it?

The user interface for the PMI task environment is simplified to improve usability and make the system more intuitive. Enhancements include:

  • The Show and Hide command now supports all PMI objects.

  • The Center Mark command is now named PMI Center Mark.

  • The 3D Centerline command is now named PMI Centerline.

  • The Region group on the PMI Region Label dialog is now named Settings.

  • The Inherit option is added to the Feature Control Frame, Datum Feature Symbol, Datum Target, and Surface Finish Symbol dialog boxes
