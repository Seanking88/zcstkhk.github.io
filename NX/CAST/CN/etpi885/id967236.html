Find PMI associated to geometry

  1. Choose Information→PMI→Find PMI Associated to Geometry.

  2. On the Selection bar, from the Type Filter list, select Face.

  3. In the graphics window, select the planar face COPY_BODY_CUT(9) for Datum A.

  4. In the Find PMI Associated to Geometry dialog box, select Create Query Results View.

    Note

    Because the view you used to inquire about associated PMI data is a section view, the Query Results view created by the Find PMI Associated to Geometry command is linked to that section view.

  5. Click OK.

  6. If NX displays the Find PMI Associated to Geometry information window, click OK.
