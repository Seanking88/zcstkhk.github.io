Use PMI Region

  1. Open pmi1_material_specs and start the PMI application.

  2. Make the Back view the work view.

  3. On the PMI toolbar, from the PMI Supplemental Geometry Drop-down list, select PMI Region .

  4. At the top of the PMI Region dialog box, click Reset .

  5. From the Type list, select Circular.

  6. In the Preview group, make sure the Preview check box is selected.
