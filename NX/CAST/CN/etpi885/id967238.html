Examine the linked query results view

  1. In the Part Navigator, expand the Query Results View node.

  2. Under the Query Results View, select the Datum Feature Symbol A feature.

  3. Toward the bottom of the Part Navigator, expand the Dependencies group.

  4. Completely expand the Datum Feature Symbol A node and note that the face Face of COPY_BODY_CUT(9) is associated to the Datum Feature Symbol A.

  5. Click the Dependencies header to close the group.
