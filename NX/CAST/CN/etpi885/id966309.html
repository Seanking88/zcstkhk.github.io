Add crosshatching to the region

  1. In the Settings group, under Crosshatch, select Display Crosshatch .

  2. From the Pattern list, select Iron/General Use.

    Note

    If you do not see Iron/General Use in the pattern list, a crosshatch pattern file other than the default may have been selected. In that case, in Crosshatch File, set the path to the default crosshatch pattern file, xhatch.chx. The default location for the file is [NX 8.5 installation folder]\UGII\.

  3. In the Preview group, click Show Result .

    The crosshatch pattern scale is too large to see well.

  4. In the Preview group, click Undo Result .

  5. Under Crosshatch, in the Distance box, type 2.

  6. Click OK.
