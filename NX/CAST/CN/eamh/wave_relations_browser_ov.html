NX 关系浏览器

可以使用 NX 关系浏览器显示以下内容：

  • 部件之间的 WAVE 和部件间表达式相依性。

  • 关于产品接口关系的详细信息。

注释

在本地 NX 中，部件必须已完全或部分加载。

在 Teamcenter Integration for NX 中，无须加载所有部件文件即可浏览关系。

为何使用？

NX 关系浏览器：

  • 允许根据类型和状态来过滤关系。

  • 允许按部件名搜索包含关系的部件。

  • 可供以交互方式遍历相依性路径。

  • 以图形方式显示关系类型以及每种类型的关系的数量。

  • 自动排列节点以最大程度地减小重叠。

NX 关系浏览器包含：

  • 用于查看概述或预览的窗口。

  • 部件窗口

  • 多个窗口，可供查看有关在部件间关系视图窗口中选择的部件节点或关系节点的关系或产品接口的细节。

  • 文件、编辑、查看、窗口、浏览器和帮助菜单。

注释

可以关闭、最小化或重新组织任何窗口，以便更好地查看感兴趣的窗口。窗口菜单允许您重新打开已关闭的窗口。

NX 关系浏览器底部显示当前所选部件的数目及当前所选关系的数目。

如果试图在以下任意情况下与 NX 关系浏览器交互，则会收到一条消息：

  • 由于对浏览器中显示的部件和关系进行更改，而需要刷新 NX 关系浏览器时。如果在浏览器之外（例如，在部件导航器中）进行了更改，则可能需要刷新浏览器。

    注释

    在 NX 关系浏览器之外做出的更改不会自动传播到浏览器，因此必须刷新浏览器才能显示更改。如果在浏览器内进行了更改，则浏览器将进行自我更新，因而不会显示任何消息。

  • 正在执行复杂操作（如部件加载或更新）时。

  • 打开一个阻止 NX 关系浏览器操作完成的对话框时。

如果用支持检查要求结果的 Teamcenter 版本运行 Teamcenter Integration for NX，则可使用 NX 关系浏览器来检查跟踪链接的属性。有关更多信息，请参见检查要求帮助。
