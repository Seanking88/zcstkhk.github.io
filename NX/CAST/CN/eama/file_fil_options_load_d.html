装配加载选项

使用装配加载选项命令可配置将打开的部件组件加载到内存的方式。

例如，可以指定：

  • 使用多个版本加载部件的方式

  • 使用原生 NX 时组件的查找位置

  • 完全加载还是部分加载组件

  • 要加载的默认引用集

  • 装配书签文件的加载方式

也可执行以下操作：

  • 允许替换组件

  • 创建或更新部件族成员

  • 如果某些组件无法加载，则控制要采取的操作

  • 保存和恢复装配加载选项配置

注释

  • 装配加载选项仅影响尚未打开的部件。

  • 将始终从指定的目录中完全加载在打开对话框中指定的装配部件文件。

在原生 NX 中，可以在装配用户默认设置中设置版本规则。

提示

要找到版本默认设置，请选择文件→实用工具→用户默认设置。选择装配→本地标准，然后单击部件名版本选项卡。

如果在打开装配时，较早版本的组件部件已经打开，则将使用打开的版本，而不使用通过装配加载选项指定的版本。
