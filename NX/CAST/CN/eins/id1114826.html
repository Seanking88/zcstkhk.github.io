Installing CAST

  1. Log in as Administrator.

  2. Start the installation — Double-click the NXCAST.pkg package. Finder will start the installation.

  3. Click Continue as needed, then click Install.

    Note

    The default installation location is /Applications/Siemens/NX8.5/nxcast. This location cannot be changed.

  4. Summary — The installation will supply a final completion message. If any errors are indicated, view the installation log by selecting Window→Installer Log OR press Apple Key – L.
