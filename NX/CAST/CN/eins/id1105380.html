Installing CAST (Linux and UNIX)

This section explains the installation of the CAST (Computer Aided Self Teach) Online product on all supported Linux and UNIX workstations. Techniques are included to help you manage your CAST software on a Linux or UNIX system.

Note

Although NX is no longer supported on UNIX, you can still serve the CAST Library from a UNIX web server.

The topics covered are:

  • Installation instructions

  • Using CAST

  • Putting CAST on a web server

The installation of the CAST Online Library is accomplished using the supplied installation script. If you accept the defaults, the CAST files are installed into $UGII_BASE_DIR/nxcast. This directory will contain your CAST executable files.

Tip

The Linux/UNIX installation files are available on the CAST Online Library DVD or by download from the Siemens PLM Software FTP server.
