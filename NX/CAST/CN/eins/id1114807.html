Installing CAST (Mac OS)

This section shows you how to install the CAST Online Library on an Apple Intel machine, using Mac OS X. NX 8.5 must be installed first. This installation is for Apple Intel machines only.

Note

The Mac OS CAST Online Library installation is not available on DVD. It must be downloaded from the Siemens PLM Software FTP server.
