Create an installation log

As a troubleshooting aid or historical reference, you can create an installation log. If you do not wish to create a log of the installation, proceed to the next page.

  • To create a log, enter the following command:

    script /usr/tmp/nxcast.log (you may specify any name for the log file)

This script command will log any information that appears on your screen including your entries. To append to an existing log file, use script –a.

Note

The log will continue until you enter Ctrl+D or exit <CR> to terminate the log session. It cannot be terminated by nxcast_install.
