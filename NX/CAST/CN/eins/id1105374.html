Using CAST

Starting CAST from inside NX

To run CAST from inside NX, choose Help→Training from the NX menu bar.

Starting CAST from Windows

To run CAST, use the CAST icons in the Siemens NX program group, by choosing:

  • Start→Programs→Siemens NX 8.5→NX 8.5 CAST

Due to screen size constraints, we do not recommend using CAST with resolutions less than 1024x768.
