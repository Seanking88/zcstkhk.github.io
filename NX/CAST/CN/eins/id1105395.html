Using CAST

Starting CAST

To access the CAST Library, you can use either of the following procedures:

  • Use the CAST option in the NX Activity Menu (this brings up CAST and NX).

  • Use Help→Training in the NX menu bar.

Note

Due to screen size constraints, we do not recommend using CAST with resolutions less than 1024x768.

Starting CAST manually

The CAST installation always creates a directory or link entry in the UGII_BASE_DIR directory, which is usually named nxcast.

If you prefer, you can start CAST with NX by executing the following command:

  • $UGII_BASE_DIR/nxcast/bin/startcast

You can also use the following command to start CAST without NX:

  • $UGII_BASE_DIR/nxcast/bin/castonly
