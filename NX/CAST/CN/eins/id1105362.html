安装 CAST (Windows)

This procedure installs the CAST Online Library on Windows (Intel) systems. The installation requires approximately 2.2 GB of disk space (386 MB for the html directory; 1.9 GB for the parts).

Follow the instructions below:

  • Log in as administrator or as a user that has administrator privileges.

  • Put the CAST DVD into your DVD drive.

  • If you have autoplay activated, a launch screen appears in a few seconds. If you do not have autoplay activated, from Windows Explorer double-click the AutoPlay.exe file on the DVD drive.

  • Click Install CAST to start the CAST installation.

    The installation prompts you for the appropriate installation information.

Tip

The Windows installation files are available on the CAST Online Library DVD or by download from the Siemens PLM Software FTP server.
