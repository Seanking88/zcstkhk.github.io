Web server recommendations

There are many free web server packages available over the Internet. If you do not currently have a web server set up to run CAST at your site, you should install a web server such as Apache, Microsoft Internet Information Server (IIS), or other available server that will work with your system.

If you already have a server set up at your site, it will probably work with the CAST Online Library.

Note

If you want to install an Apache server, it is free and can be downloaded from http://www.apache.org.
