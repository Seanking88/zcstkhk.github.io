Create the Solid Laminate physical property

You will create a Solid Laminate physical property with a stacking sequence that is defined using the shorthand notation.

  • Simulation Navigator

  • 3D Collectors (expand)

  •   Solid(1)

  •   Edit

  • Type

    Solid Laminate

  • Create Physical (Solid Property row)

  • Name

    Pad Solid Laminate

  • Import Layup Using Shorthand Format

  • Shorthand String

    [0/+-45(--+-5)90']_s

  • Thickness

    0.1

  • Material

    Ortho_Sample

  • OK

     

Notice that:

  • The Stacking Recipe is set to Symmetric with core.

  • You created 15 plies that all have the same thickness and material.

  • There is one ply group that contains 4 plies.

Note

Do not leave the Solid Laminate Modeler dialog box.
