Plies and 3D mesh elements

In the Solid Laminate Modeler dialog box, you define:

  • The stacking recipe.

  • The stacking sequence, by creating plies and ply groups.

The procedures and commands you use are similar to those you use for the 2D zone-based process.

All plies of a solid laminate physical property are defined on each 3D element.

Example

As there are 3 plies in the laminate and 5 layers in the 3D mesh, the model has a total of 15 plies.

Creating 2D and 3D laminates from existing laminates

Copy as Shell Laminate

Use this command to create a 2D shell laminate physical property table from the selected 3D solid laminate physical property table.

Copy as Solid Laminate

Use this command to create a 3D solid laminate physical property table from the selected 2D shell laminate physical property table.

The copied physical property has the same ply layup and stacking recipe as its 2D or 3D counterpart.

Note

These commands are supported only in the Nastran and Abaqus solver environments.
