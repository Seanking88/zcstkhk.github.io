Create 3D laminates

You can use one of three processes to create 3D laminates.

To create 3D laminates using a 3D zone-based process:

Step 1.

Create solid laminate physical properties.
Step 2.

Assign them to the 3D mesh collectors.

To create 3D laminates using a 2D zone-based process:

Step 1.

Create laminate physical properties.
Step 2.

Assign them to the 2D mesh collectors.
Step 3.

Inflate the 2D laminate.

To create 3D laminates using a ply-based process:

Step 1.

Define global layups and draping settings.
Step 2.

Inflate the 2D laminate.
