Create a 2D dependent mesh

2D Dependent Mesh (Advanced Simulation toolbar, Mesh Drop-down list)

  • Select the bottom face as the master face.

  • Select the top face as the target face.

  • Select Target Edge (0)

  • Select the top front edge as the target edge for Loop_1.

  • Loop_2

  • Select Target Edge (0)

  • Select the top circle shown as the target edge for Loop_2.

  • Repeat these steps for all other loops.

  • OK

     

    The 2D Mesh dialog box appears.
