Solid Laminate Modeler

The Solid Laminate physical property is the 3D equivalent of the Laminate physical property, and is available in the Nastran and Abaqus environments. In the ANSYS environment, the Laminate physical property is used for both 2D and 3D laminates.

You can:

  • Assign a Solid Laminate physical property directly to a 3D mesh collector.

  • Inflate a 2D mesh with an assigned Laminate physical property to create 3D meshes, mesh collectors, and an extruded Solid Laminate physical property.

The Solid Laminate Modeler dialog box is the main interface you use to create or modify your Solid Laminate physical property. You can access the Solid Laminate Modeler dialog box in one of the following ways:

  • On the Laminate toolbar, click Solid Laminate Physical Property .

  • Choose Insert → Laminate → Solid Physical Property.

  • In the Simulation Navigator:

    1. Right-click a 3D collector node and choose Edit.

    2. In the Mesh Collector dialog box, set the Type list to Solid Laminate.

    3. Click Create Physical .
