Inflating laminates

You can inflate a 2D laminate that you created using both the ply-based process and the zone-based process in one of these ways:

  • Use the Extrude Laminate command to extrude a 2D laminate over a 2D mesh to create a 3D mesh.

  • Use the Fill Laminate command to create 3D meshes between one or more sets of 2D dependent meshes.

You can use the solid mesh to get better results in the ply out-of-plane direction.

The laminate information is taken from the Laminate physical property either alone in the zone-based process or with global layup information in the ply-based process.

You cannot view the fiber orientations of the 3D extruded elements.

NX creates the following meshes and mesh collectors:

  • A 3D mesh collector for each laminate physical property that is referenced by the inflated elements. The 3D mesh collectors contain meshes for the 3D plies.

    One 3D mesh per ply or ply group for each 2D mesh collector. The number of solid layers created depends on the Single Ply per 3D Layer check box and on number of ply groups. Each solid layer is a separate 3D mesh.

  • A mesh collector that contains ply drop-off meshes. This mesh collector is called Extrusion Resin. The Extrusion Resin mesh collector points to a solid property and to the isotropic Epoxy material from the NX library. You can change this material.

    Drop-off meshes. These meshes are called Resin Elements.
