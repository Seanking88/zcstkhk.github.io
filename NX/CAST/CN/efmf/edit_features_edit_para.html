编辑参数

使用编辑参数对模型中的特征作出更改。

  • 可以在用于创建特征的同一个对话框中编辑该特征的许多参数值、输入对象及参考对象。

  • 在编辑某些特征时，会显示编辑选项而非创建对话框。包括在较早的 NX 发行版中创建的特征，以及在仍使用先前编辑方法的当前发行版中创建的特征（例如，键槽和槽特征）。

  • 使用编辑参数选择多个特征时，通常可以同时更改所有选定特征的表达式与参考。

使用编辑参数来编辑具有隐藏父几何体的特征

对于所含的父几何体是隐藏的或是位于非可见层的特征：

  • 在使用编辑参数编辑特征时，该父几何体临时变为可选。

  • 完成编辑时，原始显示状态恢复为原始的非可见层、或原始的隐藏或抑制状态。

    如果从特征移除父几何体，则它恢复到原始图层及隐藏状态。
