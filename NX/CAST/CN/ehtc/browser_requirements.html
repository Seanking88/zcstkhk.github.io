浏览器和插件要求

CAST 是以 HTML 的格式提供的，它可以在您本地的浏览器显示。

支持的浏览器

  • Windows

    • Internet Explorer 7、8或者9 (32-位版本)*

    • Firefox 3.x 或更高**

  • UNIX/Linux

    • Firefox 3.x 或更高**

  • Mac

    • Safari 3.x 或更高

Internet Explorer 注意事项

  • 如果您在 Internet Explorer 中有显示页面的问题，您可能需要设置您的浏览器选项来允许显示活动的内容。这通常是在使用 Windows XP Service Pack 2 (SP2)时出现的问题。

    要允许在 Internet Explorer 中显示活动的内容，进行下列操作：

    1. 选择工具→Internet 选项。

    2. 点击高级选项卡。

    3. 在安全部分，打开允许活动内容在“我的电脑”的文件中运行。

  • IE9兼容性视图：当打开 http:// 协议或者 file:/// 协议时 HTML 传送完全支持，然而，当视图文件来源于本地安装时，比如 D://，您就可能需要激活兼容性视图，在 IE 9中，操作如下：

    1. 选择工具/兼容性视图设置。

    2. 在兼容性视图设置对话框中，选中以兼容性视图显示所有网站复选框。

**Firefox 注意事项

  • 在 Firefox 3.0.5 的默认设置是在新标签页中打开页面，这会影响全局的搜索，因为新链接不是在一个新窗口而是在一个新标签页中打开，用户可以在工具→选项→标签页来配置 Firefox 打开一个新窗口而不是新标签页。

  • 大多数用户安装并打开我们的首页是通过 http:// 协议，这是完全受支持的，然而，Firefox 有一个默认的安全设置来阻止我们从 UNC path (file:///)正确打开本地帮助，为了改变这一设置，您必须更改 security.fileuri.strict_origin_policy 首选项的值：

    • 在地址栏中键入 about:config

    • 在搜索输入框中键入 security.fileuri，如果 security.fileuri.strict_origin_policy 首选项的值是设置为 true，便将其改为 false。(双击以修改)。

    • 重启浏览器
