创建部件间链接

使用创建部件间链接命令可在定义特征时创建 WAVE 链接对象。

下面是一些包含创建部件间链接选项的特征创建命令：

  • 草图

  • 拉伸

  • 回转

  • 变化扫掠

为定义特征而选择的几何体将作为链接几何体进行复制。使用选择条上的选择范围选项，您可以过滤父部件的选择范围：

  • 整个装配。

  • 在工作部件和组件内。此选项与子装配及其组件配合使用。

  • 仅在工作部件内部。

选择条上的选择范围 (1) 和创建部件间链接 (2) 选项有助于您在装配的关联中进行设计。

您可以从其他组件中直接选择几何体并在许多建模命令执行过程中自动创建 WAVE 链接，具体取决于选择范围和创建部件间链接的设置。这将在部件之间建立关联关系，使您对一个部件所做的更改可以驱动另一个部件的设计。

选择范围选项确定可以选择用来创建 WAVE 链接的部件间几何体。

  • 整个装配可用于选择工作部件之外的几何体。

  • 如果工作部件是装配，则在工作部件和组件内仅允许从工作部件的子项内选择部件间几何体。

  • 仅在工作部件内仅允许在创建特征的过程中仅从工作部件中选择几何体。

创建部件间链接选项确定所选部件间几何体的 WAVE 链接是关联还是非关联。要为所选几何体创建关联的 WAVE 链接，请打开创建部件间链接。

选择部件间几何体时，支持对以下选项应用“选择意图”规则：

  • 在相交处停止

  • 跟随圆角

  • 特征内成链
