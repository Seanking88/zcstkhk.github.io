关联设计选择范围

“选择”条上的选择范围(1)和创建部件间链接(2)选项有助于您在装配的关联中进行设计。

使用选择范围选项可指出您需要的选择范围。可在以下范围内选择对象：

  • 整个装配

  • 仅当前工作部件

  • 工作部件及其组件

从工作部件以外的部件中选择几何体的一些原因如下：

  • 从相邻组件选择面或边以定义工作部件中某个拉伸特征或回转特征的轮廓。

  • 选择一个面以定义工作部件中某个拉伸的限制。

使用建模命令创建 WAVE 链接

您可以从其他组件中直接选择几何体并在许多建模命令执行过程中自动创建 WAVE 链接，具体取决于选择范围和创建部件间链接的设置。这将在部件之间建立关联关系，使您对一个部件所作的更改可以驱动另一个部件的设计。

选择范围和创建部件间链接设置对 WAVE 链接有以下影响：

  • 如果从选择范围列表中选择整个装配并打开创建部件间链接按钮，则从工作部件及其子项以外选择几何体时，将创建 WAVE 链接。

  • 如果从选择范围列表中选择整个装配并关闭创建部件间链接按钮，则可以从工作部件以外选择几何体，但不会创建关联的 WAVE 链接。

  • 如果从选择范围列表中选择在工作部件和组件内部，行为将与选择整个装配时一样，但只能从工作部件的子项（其中工作部件是装配）中选择部件间几何体。

  • 如果从选择范围列表中选择仅在工作部件内部，则只能在特征创建过程中从工作部件中选择几何体。

选择部件间几何体时，支持对以下选项应用“选择意图”规则：

  • 在相交处停止

  • 跟随圆角

  • 特征内成链
