Generate stress contour results

Next, you will evaluate Stress Response Results for all elements on the model, and select the time point in the response function where stress is highest. Use the following steps, referring back to the previous page for details if necessary:

  1. In the Evaluate Response Results dialog box, select Stress as the Requested Result (clear the Displacement check box).

  2. Select all the elements in the model.

  3. Select From XY Graph as the Method.

  4. Under Point Value, click the button.

  5. In the Equation Selection list, select 2_(16E_VONMTOP) and click OK.

    Note

    The file name may be different based on the element that you selected when you generated the elemental stress function in a previous step (the number 16 in the above file name is the element ID).

  6. In the plotted elemental stress function, select the time point where stress is highest, and then click OK to generate the response results.

In the next step, you will view the contour plot of these stress results as well as the displacement results you generated earlier.
