Create the FEM and Simulation files

  Simulation Navigator

  •   rod.prt

  •   New FEM and Simulation

  •   Create Idealized Part

  • The New FEM and Simulation dialog box lists the three new files that are automatically created: the Simulation, FEM and idealized part files.

  • The Solver Environment section lists NX NASTRAN as the solver. The Analysis Type is Structural.

  • OK

     New FEM and Simulation dialog box

    The Solution dialog box appears.

  • The default Solution Type is SOL 101 Linear Statics - Global Constraints.

  • OK

      Solution dialog box

  • The Simulation Navigator shows that the Simulation file is the displayed part, and the FEM file is the work part.
