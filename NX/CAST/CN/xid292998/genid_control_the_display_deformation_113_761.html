Control the deformation display

The contour plots you created in the previous steps are displayed on deformed model geometry. To see the deformation, you can overlay the undeformed model geometry.

  Edit Post View (Post Processing group)

  • Display

  • Show undeformed model

  • Apply

      Post View dialog box

The undeformed geometry is translucent. However, your NX settings may depict undeformed geometry as opaque.

Opaque undeformed geometry will likely obscure portions of the contour plot. You can change opaque undeformed geometry to translucent as follows.

  • Edges & Faces

  • Faces (Undeformed Display)

    Translucent

  • OK

      Post View dialog box

  • Note

    If the undeformed shape still appears opaque, complete the following steps.

  Menu 

 

  • Preferences→Visualization

  • Visual

  • Translucency

  • OK

      Visualization Preferences dialog box
