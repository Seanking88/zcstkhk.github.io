Fix element connections

  Zoom (Top Border bar)

Zoom in on the problem area shown.

When you zoom in closely, you should see that there are two edges that are not stitched.

Use the Stitch Edge command to repair the free edges. Stitch Edge stitches together free edges within a solid or sheet body or between different bodies. The software stitches the edges together and creates a single common edge.
