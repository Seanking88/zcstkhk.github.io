Set the customer default to load parts

Assembly FEMs may contain many component FEMs, and can take up large amounts of space on your system. By default, in Advanced Simulation, the parts and idealized parts associated with the component FEMs are not loaded automatically. You must explicitly choose to load these parts.

In this activity, you must load all the parts associated with the component FEMs. This allows the solver to access midsurface thickness data, which is stored in the idealized part.

First, you will change a customer default setting to allow parts associated with component FEMs to be automatically loaded.

Start NX.
