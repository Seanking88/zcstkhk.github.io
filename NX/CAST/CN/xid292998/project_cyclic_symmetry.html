Cyclic symmetry project

Cyclic symmetry occurs when a structure and loading are repeated about an axis. Automatic Coupling in Advanced Simulation allows you to easily create cyclic symmetric models by automatically coupling the displacements occurring on cyclic symmetric surfaces.

Objective

A marine propeller is shown below. Create a cyclic symmetric model and use the SOL 101 Linear Statics - Global Constraints solution sequence to examine stresses in the propeller when it spins at 5000 rpm.

Notes

To model the structure using cyclic symmetry:

  1. Open the part file and start Advanced Simulation. Create idealized part, FEM, and Simulation files.

  2. After making the idealized part the displayed part, use the Promote command to promote the body.

  3. The propeller geometry is repeated three times. Use the Split Body command to isolate one instance of the repeated geometry.

  4. After making the FEM the displayed part, turn off the display of the unused geometry.

  5. Define the material and the physical properties and create a 3D mesh collector.

  6. Use the 2D Dependent Mesh command to create a seed mesh of CTRIA6 elements on the cyclic symmetric faces. Select Symmetric as the 2D dependent mesh type and select Cylindrical as the Reference CSYS. When the 2D Mesh dialog box opens, select CTRIA6 as the element type, enter an element size of 0.15 in, and clear Export Mesh to Solver.

  7. Use the 2D seed mesh to create a solid mesh of CTETRA10 elements. In the 3D Tetrahedral Mesh dialog box, enter an element size of 0.15 in. Assign the mesh to the mesh collector you created previously.

  8. Use the Assign Nodal Coordinate System command to define a cylindrical CSYS as the nodal displacement CSYS. The Assign Nodal Coordinate System command is found under the Nodes and Elements tab in the Nodes group.

  9. After making the Simulation the displayed part, constrain the nodes on the bore of the propeller in the tangential and axial directions. To avoid conflicting constraints, exclude the polygon edges of the bore that are also used to define edges of the dependent face.

  10. Use the Automatic Coupling command to couple the DOF of symmetric node pairs.

  11. Optionally, resize the constraint symbols.

  12. Use the Rotation command to apply a centrifugal load for an angular velocity of 5000 rpm.

  13. Solve the model.
