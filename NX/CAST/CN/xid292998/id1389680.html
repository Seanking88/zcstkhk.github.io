    Note

    The number 38 indicates the number of normal modes that were output. This means that the MEFFMASS threshold filtered out 22 of the 60 modes that you requested.

  • Response Simulation Details View

     

    This panel displays the dynamic characteristics (frequency, mass, damping, and stiffness) for each normal mode solved in the response dynamics solution.

    Modes with an asterisk character (*) in the # Mode column are considered "active" in your response evaluations.

    Typically, you want the total effective mass in the direction of the applied force to be 80% or higher. In this model, you apply the loads in only the Z-direction. The total effective mass in the Z-direction is approximately 87%.

    Tip

    If you want to include more mass, rerun the normal modes solution with either or both of the following changes:

    • Request that the software calculate more modes.

    • Decrease the MEFFMASS threshold so the software retains more modes for the response simulation.

    For this example, you do not need to rerun the solution.
