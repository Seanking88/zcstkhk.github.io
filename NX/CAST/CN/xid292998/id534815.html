Define element thickness with fields

In previous steps, you defined the element thickness using the midsurface. You then modified the corner node thickness of selected elements. Another way to define the element thickness is to use a spatial field to vary the thickness of elements.

In this activity, the provided FEM includes two meshes, 2d_mesh(1) and 2d_mesh(2), that have been copied and translated to create two additional meshes, 2d_mesh(1)_translated(1) and 2d_mesh(2)_translated(1).

You will use spatial fields to define the thickness of the copied elements. First, convert the midsurface-based thickness of the original elements to a field.
