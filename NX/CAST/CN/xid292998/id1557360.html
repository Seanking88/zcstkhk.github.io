Create a mapped mesh

Begin creating the 2D mapped mesh on the part starting with the surfaces on the inside of the part. The order of surfaces chosen can affect the resultant mesh. You will start with the outer portions of the face shown, and work your way towards the middle.

  • Apply

     

Apply generates the mesh and keeps the 2D Mapped Mesh dialog box open for the next step.

There should be no triangular elements in the three-sided surface.

2D Mapped Mesh automatically adds mesh controls, represented by the blue symbols. They are used to control local mesh densities, and can help control a mapped mesh if edges do not line up as planned.
