Create new FEM and Simulation files

You can easily change the properties of a mesh by moving it to a different collector. In the following steps, you will define a new material, Graphite. Then you will create a new mesh collector with Graphite as the material and then move the shaft mesh to the new mesh collector. To more easily compare the analysis results, you will make these changes to a copy of the FEM and Simulation files.

The Simulation file should still be active before starting these steps.
