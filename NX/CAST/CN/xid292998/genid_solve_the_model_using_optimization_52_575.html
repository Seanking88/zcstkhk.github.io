Warning

Do not close the spreadsheet until the solution finishes. The solution is finished when the spreadsheet displays the message: Small change in design, run converged.

The solve can take several minutes, as the software optimizes the part.

Note

In some cases, you may receive a message stating This action cannot be completed because the other application is busy. Choose 'Switch To' to activate the busy application and correct the problem. If you see this message, click Retry in the message box; this should cause the spreadsheet to be populated.

When the job is complete, examine the spreadsheet. The spreadsheet shows all attempted iterations and displays failed design constraints results in red.

Tip

If you want to access the spreadsheet later to examine the results, save the file before closing it. To reopen the saved spreadsheet, in the Simulation Navigator, right-click Setup 1 and choose Optimization Spreadsheet.
