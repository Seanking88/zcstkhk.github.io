Consolidate the 2D meshes

Only one 3D mesh collector was created, so no consolidation is necessary for the 3D meshes.

Next, examine the 2D mesh collectors. You will see that the 2D meshes use three physical property tables:

  • SHELL 4MM IDEAS PTABLE

  • SHELL 3MM IDEAS PTABLE

  • SHELL 2MM IDEAS PTABLE

Rename the Thin Shell(1), Thin Shell(3), and Thin Shell(5) mesh collectors to 4mmShell, 3mmShell, and 2mmShell respectively.

Consolidate the 2D meshes into three collectors. Organize them according to their physical properties. The consolidated meshes should look like this:

2D Collectors
4mmShell
2d_mesh(1)
2d_mesh(2)
3mmShell
2d_mesh(3)
2d_mesh(4)
2mmShell
2d_mesh(5)

You can then delete the empty mesh collectors.
