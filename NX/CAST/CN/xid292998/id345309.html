Create a convection boundary condition

Apply a convection boundary condition to all the external surfaces of the exhaust manifold except the mounting face. Because all the external surfaces of interest transition smoothly into one another, you can efficiently select them using the Tangent Faces selection method option.

  Convection (Loads and Constraints group→ Constraint Type list)

  • Method (Top Border bar)

    Tangent Faces

  •  

  • 484 Polygon faces are selected.

  • Properties

     

  • Ambient Temperature

    250 °F

  • Convection Coefficient

    100 W/m^2-C

    Note

    Make sure that you change the temperature units to F, and the convection coefficient units to W/m^2-C.
