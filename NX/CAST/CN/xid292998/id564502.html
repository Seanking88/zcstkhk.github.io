Create mesh mating conditions

Some mesh mating conditions have been created automatically, because you selected the Keep Imprinted Edges option for the Split Body command. The mesh mating conditions (MMCs) are defined where coincident face pairs exist due to the split body operations. Mesh mating converts the face pair into a single face shared by two volumes.

Next, you will create additional MMCs to connect the meshes between the bodies in the assembly.

  • The MMC Collection now includes MMCs with names including the word "auto". These are the MMCs that were automatically created.

    The split body MMCs and automatic MMCs are created in different situations. The split body MMCs are defined between solid bodies that represent the same component. The automatic MMCs are defined between solid bodies that represent different components. Split body MMCs keep a component connected to itself, while automatic MMCs define connections between assembly components.
