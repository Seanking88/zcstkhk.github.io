Insert lines to model the rails

Use the Line command to insert associative geometry you can mesh with the 1D Mesh command.

Create a series of four line segments that connect the midpoints and arc centers highlighted in the following image.

  Menu 

 

  • Insert→Curve→Line

  • Enable Snap Point, Mid Point, Arc Center (Top Border bar)

    (All other Snap Point options should be deselected.)

  • Select pairs of points to create a series of four line segments along the rail, as shown in the animation.

    Apply

     after selecting each pair of points

  •   Rotate the part to view the other rail.

  • Repeat these steps to create an identical series of line segments on the other rail.

Leave the Line dialog box open for the next step.
