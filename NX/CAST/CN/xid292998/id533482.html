Generating a shell mesh on a strut

In this example, you will do the following to generate a shell mesh on a strut:

  1. Generate a midsurface.

  2. Clean up the geometry of the midsurface.

  3. Generate a 2D mesh on a midsurface.

  4. Define element thickness for selected elements.

  5. Copy and translate the elements on the midsurface.
