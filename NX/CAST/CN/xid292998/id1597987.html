Resize the cylinder to fit

In this step, you will modify the length of the blue cylinder to eliminate the interference between the two parts.

The Move Face dialog box is still open from the previous step.

  •   Static Wireframe (Top Border bar→Rendering Style Drop-down list)

  •   the circular end face of the blue cylinder

    Tip

    You can also use QuickPick to select the face, which is named Face of Extrude(9).

  •   Vector Dialog (Specify Distance Vector)

  • Type

    Face/Plane Normal

        the same circular face as the previous step


  • OK

     

  •   Drag the Distance arrow upward 5 mm to align the end face of the blue cylinder with the end face of the red frame.

  • Apply

     

  • Shaded with Edges (Top Border bar→Rendering Style Drop-down list)
