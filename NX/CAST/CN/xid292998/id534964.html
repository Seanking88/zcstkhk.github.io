Workflow for analyzing a bladed rotor

In this course, you will do the following workflow to solve a model of an integrally bladed rotor (IBR).

  1. Define mesh mating conditions to connect individual meshes together.

  2. Create symmetric 2D dependent meshes on the sides of the blade.

  3. Create hexahedral meshes on polygon bodies in the model. Some of the hexahedral meshes use the 2D dependent meshes as seed meshes.

  4. Create a tetrahedral mesh, using pyramid elements to transition from the tetrahedral to hexahedral elements.

  5. Define centrifugal load on the model.

  6. Define automatic coupling to automatically create coupled degrees of freedom between symmetric meshes on the blade.

  7. Constrain the model and remove redundant constraints.

  8. Solve the model and display results.
