Create an acceleration pulse function

Use the Function Tools for Response Simulation utility to create a 1 mm/sec^2 acceleration pulse function for use in your dynamic excitation.

Note

For your convenience, the completed pulse function is also included in the provided response_functions.afu file.

Response Simulation

  Function Tools for Response Simulation (Response Simulation group)

Note

This utility requires the Java Runtime Environment (JRE) version 1.7. You can install the JRE from www.oracle.com.

If you encounter Java memory errors while trying to open the Function Tools for Response Simulation utility, set this environment variable in your ugii_env.dat file: UGII_JVM_OPTIONS=-Xmx64M. Then, restart NX.
