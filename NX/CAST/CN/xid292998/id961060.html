Split the solid body

Before attempting to mesh the serving tray support bracket, you first need to split it into sweepable shapes.

  Split Body (Geometry Preparation group)

  •   the solid body

  • Tool Option

    Extrude

  •   Curve (Select Curve)

  •  

  • Selection Scope (Top Border bar)

    Entire Assembly

  •    

  •   Profile (Curve group) to stop drawing the curve

  •   Profile (Curve group) to resume drawing

  •    

  •  Finish Sketch (Sketch toolbar)

  • Simulation Settings

     

  •   Create Mesh Mating Conditions

    Note

    When you select this option, the software will automatically create a glue coincident mesh mating condition between the bodies when you switch from the idealized part to the FEM file. The mesh mating condition ensures that any meshes are continuous from one body to the other.

  •   Check for Sweepable Body

  • The part displays in red, which indicates that you cannot generate a swept mesh on the body.

  • Apply

     

  • After you click Apply, the part has three bodies. The hinge bodies are displayed in green, which indicates that you can generate a swept mesh on them. The main bracket body is displayed in yellow, which indicates that you might be able to generate a swept mesh on it.

  • Cancel

     

After you finished sketching with Split Body, the Part Navigator tab opens. You will return to the Simulation Navigator tab in the next step.
