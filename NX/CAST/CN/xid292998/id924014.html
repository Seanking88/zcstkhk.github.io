Create a graph template

Before you change the formatting of a graph, you should create a graph template. This allows you to create and save unique formatting for the graph. If you do not create a graph template, any changes you make to graph formatting are applied to the default graph template.

 XY Function Navigator

  • Default Graph Template (under XY Graph Templates)

  • Save As

  • File name

    Strain Displacement Graph Template

  • OK

     

    The new template appears in the XY Graph Templates list in the XY Function Navigator. Because formatting changes are applied to the active graph template, make Strain Displacement Graph Template the active graph template.
