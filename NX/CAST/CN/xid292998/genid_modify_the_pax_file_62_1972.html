Modify the .pax file

In this step you will modify the ugs_cae_templates_custom.pax file to create a new tab in the File New dialog box.

The .pax file has a long list of entries. Each entry starts with a tag such as <PaletteEntry id="d1">, and ends with a </PaletteEntry> tag.

  1. Open the ugs_cae_templates_custom.pax file in a text editor.

  2. In the first <Presentation> element, change the FileNewTab= attribute value to "Custom".

    When you next run NX, a Custom tab appears on the File New dialog box.

  3. Delete all of the PaletteEntry tags except for the one labelled id="d2". This entry defines settings for the FemNxNastranMetric.fem file.

    Your custom template will be based on this palette entry.

  4. Change the id= attribute value for the PaletteEntry element to "d1".

  5. Change the location= attribute value for the PreviewImage element to "adv_fem_template_custom.jpg".

  6. Change the content of the Filename element to FemNXNastranMetric_custom.fem.

  7. Save and close the .pax file.

Your file should look like the one shown below. The areas that have been changed are shown in boldface type.

Note

This text has been edited and formatted to better fit this window. The formatting in your .pax file may be different. 

<?xml version="1.0" encoding="utf-8"?>
...
  <Presentation 
    name="Simulation templates" 
    bitmap="sheet_and_block.bmp" 
    FileNewTab="Custom" 
    application="All" 
    UserMasterModel="No"/>
    <PaletteEntry 
      id="d1">
    <References/>
    <Presentation 
      name="NX Nastran" 
      description="Empty NX Nastran Fem">
        <PreviewImage 
          type="UGPart" 
          location="adv_fem_template_custom.jpg"/>
    </Presentation>
    <ObjectData class="CaeFemTemplate">
      <Filename>
       FemNxNastranMetric_custom.fem
      </Filename>
      <Units>
        Metric
      </Units>
    </ObjectData>
  </PaletteEntry>

</Palette>

The user interface changes are complete. In the next step you will set up the new FEM file template.
