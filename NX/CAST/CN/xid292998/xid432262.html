Modify element connectivity

Modify the connectivity for the last spider element created, to increase the number of legs.

Nodes and Elements

  Connectivity (Elements group)

  • Elements to Modify

     

  • Single Element

  • Select the spider element.

     

  • Type

    Modify Leg Nodes

  • Select the three nodes shown (or select nodes in similar positions) to add three legs to the spider element.

         

  • OK

     

You can continue adding legs to the spider element, or removing others by holding down shift to deselect. The final steps of this activity will add a constraint and loads.
