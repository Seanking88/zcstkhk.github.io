Generate a hexahedral mesh on a bracket - alternate workflow

Another way to generate a swept mesh is to use the Until Target swept mesh type to select multiple, connected source faces and sweep to a single target face. This analysis workflow is as follows:

  1. Begin with a solid-based CAD assembly, either created in NX or imported into NX.

    Note

    In the following example, the model has already been simplified for analysis. The geometry from the assembly components has also been copied into the work part using WAVE Geometry Linker.

  2. In Advanced Simulation, create a FEM.

  3. Generate mesh mating conditions.

  4. Apply edge densities to control the number of elements through the thickness of the components.

  5. Create 2D dependent meshes to use as seed meshes to control the elements in the swept mesh.

  6. Create mapped meshes as seed meshes to control the elements in the swept mesh.

  7. Generate swept meshes on the bodies in the assembly, selecting multiple source faces and a single target face. The swept mesh generates hexahedral elements.
