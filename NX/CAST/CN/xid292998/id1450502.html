Add viscous damping

You will add viscous damping to the normal modes.

  Motion Navigator

  •   fore_lca-demo_assy_flex_sim1-solution_1 (under the Flexible Links node)

  •   Edit Damping Factor

  • % Viscous

    4.0

  • OK

     

  • Mode Shape Details View

     

  •   hold the mouse over any of the column headers

  •   Columns→Damped Frequency (make sure Damped Frequency is selected)

  • Compare the values in the Frequency column to the values in the Damped Frequency column that you just added.

Because you applied the command to the flexible body node under Flexible Links, the software applies the damping to the entire system. You can define damping for individual modes by right-clicking the mode in the Mode Shape Details View panel and choosing Edit Damping Factor.
