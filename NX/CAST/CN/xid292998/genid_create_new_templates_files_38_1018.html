Create new templates files

The files delivered with NX reference many templates. You will copy some files to create new ones that will be edited to reference only your specific files.

The first file you need to copy is ugs_cae_templates.pax. This file controls a tab in the File New dialog box. Copying this file will allow you to create a new tab.

  1. In your new templates directory, copy ugs_cae_templates.pax.

  2. Rename the copy ugs_cae_templates_custom.pax.

  3. Turn off the Read only property attribute for this file.

The next file to copy is the default FEM file template FemNxNastranMetric.fem. You will modify this new FEM file to add in some default features so that you can avoid performing those steps with future models.

  1. In your templates directory, copy FemNxNastranMetric.fem.

  2. Rename the copy FemNxNastranMetric_custom.fem.

  3. Turn off the Read only property attribute for this file.

Finally, copy adv_fem_template_custom.jpg from your activity parts directory to your new templates directory.
