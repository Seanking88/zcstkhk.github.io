Assembly FEM project

In Advanced Simulation, assembly FEMs are useful for large models where analysts work in collaboration or where multiple instances of individual components are used. The following exercise demonstrates the workflow to create an associative assembly FEM for a simple assembly.

Objective

The assembly shown below is used in an automotive car jack. It is constructed by welding two identical arms to a spacer plate. Create an associative assembly FEM of the assembly and then use the SOL 101 Linear Statics - Global Constraints solution sequence to perform a structural analysis of the assembly when it is compressed by 1000 lbf.

  • Arm (arm.prt)

  • Spacer plate (spacer_lower.prt)

Notes

A summary of how to complete each step is provided below.

  1. Create component FEMs.

    1. Open the part and start Advanced Simulation. Create a FEM file named spacer_lower_fem1.fem.

    2. Define the material and the physical properties and create a 3D mesh collector.

    3. Create a CTETRA10 mesh using automatic sizing. Assign the mesh to the mesh collector you created.

    4. Save and close the FEM.

    5. Repeat the previous steps to mesh arm.prt. For this part, name the FEM arm_fem1.fem.

  2. Map the component FEMs to the assembly FEM.

    1. Open the lower_arm_assy.prt and start Advanced Simulation.

    2. To create a new assembly FEM, in the Simulation Navigator, right-click lower_arm_assy.prt and select New Assembly FEM. Name the new file lower_arm_assyfem1.afm.

    3. In the Simulation Navigator, expand lower_arm_assy.prt to reveal the components comprising the assembly.

    4. In the Simulation Navigator, right-click spacer_lower.prt and select Map Existing.

    5. In the Map CAD Component To Existing FEM dialog box, browse to and select spacer_lower_fem1.fem.

    6. In the Simulation Navigator, right-click arm.prt and select Find All Matching Components.

    7. In the Simulation Navigator, right-click either of the arm.prt instances and select Map Existing.

    8. In the Map CAD Component To Existing FEM dialog box, browse to and select arm_fem1.fem

  3. Resolve label conflicts.

    1. In the Simulation Navigator, right-click lower_arm_assyfem1.afm and select Assembly Label Manager.

    2. In the Assembly Label Manager dialog box, click Automatically Resolve.

  4. Connect the component FEMs in the assembly FEM. Depending on the connection method, you can do this in the FEM or in the Simulation file. In this case, you will use the Surface-to-Surface Gluing command, which is accessed from the Simulation file.

    1. In the Simulation Navigator, right-click lower_arm_assyfem1.afm and select New Simulation. Name the new Simulation file lower_arm_assysim1.sim.

    2. To simulate the welds that join the parts, use the Automatic Pairing feature in the Surface-to-Surface Gluing command.

  5. Apply constraints and loads.

    1. Fix the nodes on the surface of the two lower holes.

    2. Apply 500 lbf bearing loads to each of the two upper holes. Direct the loads to compress the structure.

  6. Solve the model.

Expected Results

To access the analysis results, in the Simulation Navigator, double-click Results.

To view a contour plot of the von Mises stress, in the Post Processing Navigator, expand Solution 1. Next, expand Stress-Element-Nodal and double-click Von-Mises. The resulting contour plot should look similar to the one shown below.
