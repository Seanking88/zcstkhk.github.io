Display element thickness

  Simulation Navigator

  •   2D Collectors

  •   Thickness Information

  • Location to Probe Thickness Value

     

  • Type

    Element

  •  Select an element on the BOOM_RIGHT component.

  • Apply

     

  • An Information window opens. It lists the thickness values at each node of the element. For example:

    Element ID: 1525
    Node ID: Thickness
    1513: 36.000000
    1506: 36.000000
    1507: 36.000000
    1514: 36.000000

  •   the Information window

  •  Move the cursor over an element on the BOOM_RIGHT component.

    The element information is displayed above the graphics window.

  •  Move the cursor over an element on the BOOM_TOP component.

    The thickness values are 36.0 at each node.

  •  Move the cursor over an element on the BOOM_EAR component.

    The thickness values are 25.0 at each node.
