Examine the Simulation file

The Simulation file contains predefined glue connections between the shaft and the grip, ferrule, and club head. These glue connections appear in the Simulation Navigator.

Simulation Object Container
Face Gluing(1)
Face Gluing(2)
Face Gluing(3)

The Simulation file also contains a predefined fixed translation constraint applied to the grip. This constraint also appears in the Simulation Navigator.

Constraint Container
NoTrans(1)
