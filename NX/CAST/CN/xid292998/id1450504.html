    The solve process may take several minutes to complete. After the RecurDyn solve finishes, NX Nastran is called automatically to run the SOL 103 - Flexible Body solve again. NX Nastran uses the RecurDyn results file and the original NX Nastran input file to recover the deformation, displacement, stress and other results for the flexible link.

    Do NOT close any solve windows until all NX Nastran command prompt windows have closed and the message "Nastran results recovery completed" appears in the NX status line.
