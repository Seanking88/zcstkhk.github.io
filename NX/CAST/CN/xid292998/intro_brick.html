Introduction

These examples show you two ways to mesh a bracket using hexahedral elements (brick elements).

In the first example, you will learn:

  • How to simplify the CAD model.

  • How to split the body to prepare for swept meshing.

  • How to use edge densities to control the number of elements through the thickness of the components.

  • How to use mapped meshes as seed faces for a swept mesh.

  • How to generate swept meshes through the bodies in the assembly.

In the second example, you will use the same model but will use a different workflow. You use the 3D Swept Meshing command to sweep a hexahdral mesh from multiple, connected source faces to a single target face. This eliminates the need to split the bodies in Modeling. In this example, you will learn:

  • How to use edge densities to control the number of elements through the thickness of the components.

  • How to use mapped meshes as seed faces for a swept mesh.

  • How to generate swept meshes through the bodies in the assembly.
