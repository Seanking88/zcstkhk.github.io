  •   Edit (Output Requests)

  • Modal Effective Mass

    You can apply a threshold to filter out modes with small modal effective mass. This practice can save disk space because fewer modes are stored, and can save computation time because fewer modes are used in the response computation.

  •   Enable MEFFMASS Request

  • Threshold

    0.01

    Modes that have an effective mass fraction greater than the Threshold in at least one translational or rotational direction will be output for use in the response simulation when you solve the modal solution. All other modes will be filtered out. Threshold values can be set between 0.0 and 1.0. A reasonable value is 0.01, which will save modes that have an effective mass greater than 1% in any translational or rotational direction.

    This method is an alternative to using the manual method of filtering out insignificant modes that is shown in the Response Simulation - Solution Setup activity.

  • Note

    You will also need output for displacement, SPC forces, and stress. These output types are turned on by default.
