Create a 2D mesh dependency

For cyclic symmetry, the radial, tangential, and axial displacements relative to the axis of rotation are identical at equivalent positions on the repeated faces (the faces created during the trimming operation). To model this behavior:

  1. Create a 2D dependent mesh on the repeated faces. Doing so locates the nodes on each face at equivalent positions.

  2. Use the 2D dependent mesh to seed a 3D mesh of the body.

  3. Change the nodal displacement coordinate system to cylindrical.

  4. Couple the degrees of freedom (DOF) for each pair of equivalently positioned nodes lying on the repeated faces.
