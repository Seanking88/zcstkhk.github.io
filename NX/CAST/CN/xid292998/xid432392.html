Update the mesh

NX updates the spider elements when you update the mesh.

  Update (Context group)

  • Yes

     Mesh Update message box

  • Scroll through the Information window. Notice that the manually created spider element will be deleted from the model when the mesh is updated.

  •   the Information window

  • The geometry-based spider elements are updated according to their recipes. The manually created spider element is no longer visible. The mesh that contains the manually created spider element is still present in the Simulation Navigator under 1D Collectors→ RBE2 Collector(4), but is no longer listed under Connection Collectors→ Connections. If you want this spider element, you must recreate it using either the geometry-based or manual method.
