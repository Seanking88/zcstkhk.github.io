Create a new triangular element

Create a new element using the nodes at the top of the open section of the mesh. In this step, create a CTRIA3 element.

When you create 2D elements by selecting nodes, the element normal is determined by the order in which you select the nodes, according to the right-hand rule. To ensure that the elements you create are consistent with existing elements in mesh, select nodes in counterclockwise order.
