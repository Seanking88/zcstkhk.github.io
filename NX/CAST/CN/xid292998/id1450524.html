Optional: Define additional flexible bodies

For simplicity, you have defined only one of the suspension components as a flexible body. If time permits, you can repeat the steps in this activity to define the remaining suspension components as flexible.

  1. In Advanced Simulation, create a FEM file, Simulation file, and SOL 103 Flexible Body solution for each of the other suspension arms.

  2. In Advanced Simulation, solve each solution to generate the .rfi file for each flexible body.

  3. In Motion Simulation, create a flexible link for each additional flexible body.

  4. Add all of the flexible links to the existing motion solution.

  5. Solve the motion solution. Note that NX Nastran will run automatically to solve the modal solution again for each flexible body. The entire solve may take several minutes to complete.

  6. Animate the results.

    Each flexible body has its own post view, which you can edit individually. In the Motion Navigator, under the Animation node for the solution, right-click the flexible link to edit and choose Edit Post View.

    Also, when you have multiple flexible links, color contour plots are displayed only for links of a single result type. To specify the flexible link to display, in the Motion Navigator, under the Animation node for the solution, right-click the flexible link and choose Set Legend Visible. The contour legend will indicate the time increment and min/max result values for all links of this result type. The contour plots for links of other result types will be displayed in gray.

    In the following example, the post view for the blue suspension arm was set to Stress - Element-Nodal and the post views for the other flexible links were set to Displacement - Nodal. The blue suspension arm was set to Set Legend Visible, so only its contour plot is displayed in the animation.
