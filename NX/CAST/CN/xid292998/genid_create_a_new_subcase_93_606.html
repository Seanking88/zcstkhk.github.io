Create a new subcase

Rename the existing subcase that contains the pressure load. Then create a new subcase that uses the same constraints as the first subcase but does not include loads. Nonlinear solutions are path dependent. During the solution, the second subcase will begin with the final load condition from the first subcase. Then the load will be incrementally ramped down to the no-load condition.
