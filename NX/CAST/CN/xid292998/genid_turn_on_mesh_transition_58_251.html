Turn on Transition Element Size

You can use Transition Element Size to improve the element size transitions between the mesh controls and the global element size.

Tip

In large models, use of Transition Element Size can increase the size of the overall model. Therefore, as a best practice, use this option only in areas where you want to account for stresses more accurately.
