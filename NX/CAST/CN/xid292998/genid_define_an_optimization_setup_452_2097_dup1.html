Define an objective

The Geometry Optimization dialog box is still displayed from the previous step.

  • Category

    Model Objectives

    The Model Objectives option applies the optimization objective to the entire model.

    Note

    Alternatively, you can apply the optimization objective to only one type of mesh in your model. For example, if your model contained 1D and 3D meshes, and you selected 1D Objectives, the objective would apply only to the 1D meshes.

  • Type

    Weight

  • Minimize

  • Unit

    mN

    The optimization results are output in these units.
