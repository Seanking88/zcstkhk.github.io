Introduction

This course shows you how to analyze a segment of an integrally bladed rotor (IBR), also known as a blisk in the aerospace industry. You will learn how to:

  • Create 2D dependent, symmetric meshes on several faces.

  • Create 3D swept meshes, using the 2D meshes as seed meshes.

  • Use pyramid elements to transition between 3D tetrahedral and hexahedral meshes.

  • Create a cyclic symmetry model. Cyclic symmetry occurs when geometry and loading is repeated about an axis of rotation.

  • Use automatic coupling to create coupled degrees of freedom.
