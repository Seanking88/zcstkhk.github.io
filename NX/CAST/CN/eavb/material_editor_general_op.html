Option

Description

Material Color swatch

Displays the current color of the selected material.

When you click the swatch, the Color dialog box opens and you can select a different color.

Translucency

Not available for Glass type.

Specifies how much light can pass through the material.

You can use either the slider or type a value in the box.

Background Material

Renders polygons in high quality images and helps improve performance.

Type

Lets you choose how light interacts with the surface of a material.

The material reflectance options range from dull material finishes to polished and shiny materials. The reflectance combines with the material's color to produce the general appearance of the object.

Constant type

Provides constant color by ignoring the effect of all light sources. Produces the same result as an environment that includes only a single white ambient light with an intensity of 1.0.

Matte type

Provides a dull matte appearance. This type is suitable for non-glossy materials such as brick or fabric.

You can modify the following parameters:

  • Ambient: Specifies the effect of global (indirect) lighting on the selected material.

  • Diffuse: Specifies the brightness of the object from direct lighting.

Metal type

Provides a specular metallic appearance. This type is suitable for most metallic materials like steel and brass.

In addition to the ambient parameter, you can modify the following:

  • Specular: Specifies the brightness of highlights in the selected material.

  • Roughness: Specifies the sharpness of the specular highlights in the selected material.

    Higher the value, rougher the surface. That is, when roughness set to 0, all light will bounce off the object, like from a perfect polished surface. When roughness is set to the highest value, the material will be so rough that all light will be scattered on its surface, causing all reflections to disappear.

Phong type

Represents a popular computer graphics solution for shiny surfaces. This type is suitable for polished ceramics.

In addition to the ambient, diffuse and specular parameters, you can modify the following:

  • Exponent: Controls the intensity of the base color of the material.

  • Specular Color: Displays the current specular color of the selected material.

    Click the swatch to open the Color dialog box and select a different color of the specular highlight for the selected material.

Plastic type

Similar to Phong and can also be used for plastic or varnished surfaces. But unlike Phong, you can also modify the roughness of the surface.

Conductor type

Similar to Metal, but is a more accurate representation of metals. Uses ray tracing to generate secondary reflections. Fresnel filtering is incorporated for accurate modeling of the light interaction with a metal surface.

This type is suitable for accurate simulation of metallic surfaces.

In addition to the ambient, diffuse, specular and roughness parameters, you can modify the following:

  • Mirror Factor: Specifies the mirror factor, that is, how much light should be reflected from the surface.

  • [color]: Specifies the index of refraction for that particular color.

  • Absorption [color]: Specifies the index of absorption for that particular color.

You can obtain appropriate values for the refraction and absorption characteristics from published data in handbooks of optical constants.

Dielectric type

Similar to Glass, but as with Conductor, is a more accurate representation of glass.

Uses ray tracing to produce the reflected and refracted light.

Uses Fresnel filtering to calculate the interaction of light with the material.

In addition to the ambient, diffuse, specular and roughness parameters, you can modify the following:

  • Transmission: Specifies the amount of light that can pass through the object.

  • Mirror: Specifies the amount of light that is being reflected by the material.

  • Refraction : Specifies the index of refraction for glass objects.

Environment type

Incorporates the environment in a material appearance when you use an environment map as a background. You can specify how much of the map you want to use in the material appearance, as shown in the following animation.

This type is suitable for reflective materials such as shiny metals.

In addition to the ambient, diffuse, specular and roughness parameters, you can modify the following:

  • Angle Scale: Specifies the scale of angle for the environment map in the material.

Glass type

Supports an approximation of glass-like materials that have both reflective and transmissive properties. Uses ray tracing for secondary reflections and the refraction of transmitted light.

This type is suitable for approximating most glass surface finishes.

You can modify the specular, transmission, mirror, roughness and refraction parameters.

Mirror type

Supports secondary mirrored views through ray tracing. This type is suitable for representing polished mirror-like surfaces.

You can modify the ambient, diffuse, specular, mirror and roughness parameters.

Wrapped anisotropic (HQI only) type

Represents an approximation of surface with parallel scratches or ridges, such as brushed metal. Simulates the directional reflectance properties of surfaces covered in parallel scratches or grooves. Surface scratches are approximated by thin cylinders.

This type is suitable for simulating polished metal or grooved surfaces.

In addition to the ambient, diffuse, specular, specular color, and roughness parameters, you can modify the following:

  • Cylinder Distance

  • Floor Height

Wrapped circular anisotropic (HQI only) type

Represents an approximation of surface with many small areas of concentric rings of scratches or ridges repeated over the surface. Simulates the directional reflectance properties of surfaces covered in a repeating pattern of concentric scratches or grooves. Surface scratches are approximated by thin cylinders.

This type is suitable for simulating polished metal or grooved surfaces.

In addition to the ambient, diffuse, specular, specular color, roughness parameters, cylinder distance and floor height, you can modify the following:

  • Width

  • Height

Translucent Plastic (HQI only) type

Produces a glossy plastic-like reflectance with translucency. Similar to Phong, with translucency.

Provides reflectance in the same way as for Plastic, but an additional diffuse term is added which represents the diffuse contribution from lights on the reverse side of the surface.

This type is suitable for shiny or highly polished translucent materials such as translucent plastic.

You can modify the ambient, diffuse, specular, roughness and specular color parameters.

Multilayer paint (HQI only) type

Simulates the effects of multi-layer paint used in the automobile industry. The reflectance effect is achieved by a combination of several different reflectance models.

Car Paint type

Simulates the effects of paint used in the automobile industry with base color, optional metallic flakes and a separate lacquer layer.

You can modify the following parameters:

  • Base layer

    Sets the underlying color of the paint pigment.

  • Lacquer layer

    Sets the clear topcoat applied over the paint layer.

    Reflection Mode lets you switch between the two different methods of calculating reflections in software rendering.

    Environment is the default mode, and only available during Advanced Studio rendering. Reflections are based on the environment.

    Mirror reflections are based on mirror reflectance shaders.

  • Flake layer

    Sets the metallic flaked embedded in the lacquer to create more specular reflections.

  • Eye flake layer

    Creates an extra set of specular reflections.

Note

You can use Car Paint only in:

  • Advanced Studio real-time display mode.

  • High Quality Image static rendering.
