Material Editor — Texture Space tab

The options on this page let you specify how to apply, or map, a wrapped texture to geometry.

Any texture type that includes the word wrapped uses a texture space to determine how to position and size the texture onto geometry.

Options

Description

Type

Specifies the type of texture space you want to use.

Note

The Type options are available only if you specify a wrapped texture type in any of the other parameter pages.

Arbitrary Plane type

Specifies a planar projection.

You can modify the following parameters:

  • Center Point: Opens the Point dialog box to let you specify the origin of the texture space.

  • Normal Vector: Opens the Vector dialog box to let you specify the normal or primary axis for the texture space.

  • Up Vector: Opens the Vector dialog box to let you specify the reference axis for the texture space.

  • Scale: Controls the overall size of the texture space.

  • Aspect Ratio: Controls the ratio between the height and the width of the texture space.

  • Update Texture to Camera Direction: Updates either the texture space normal vector or both normal and up vectors based on the camera direction. Use this option to rectify incorrect 2D planar texture mapping.

Cylindrical type

Specifies a cylindrical projection.

You can modify the following parameters.

  • Center Point: Opens the Point dialog box to let you specify the origin of the texture space.

  • Axis Direction: Opens the Vector dialog box to let you specify the normal or primary axis of the cylinder or sphere.

  • Origin Point: Opens the Point dialog box to let you specify a reference point for the upper left corner of the texture.

  • Scale Around Axis: Controls the overall size of the texture space around the axis of the cylinder.

  • Scale Along Axis: Controls the overall size of the texture space along the axis of the cylinder.

  • Draw Feedback Vector check box: Displays a vector on the graphics window, denoting the direction in which the pattern dynamically moves, after you select any of the following vectors:

    • : Moves the pattern along the Y axis in a range from -5 to 5 scale units.

    • : Rotates the pattern on the XY plane in a range from -180 to 180.

Spherical type

Specifies a spherical projection.

The parameters you can modify are the same as for Cylindrical except for the Scale options.

  • Latitude Scale: Controls the overall size of the texture space along the latitude of the sphere.

  • Longitude Scale: Controls the overall size of the texture space along the longitude of the sphere.

WCS Auto Axis type

Selects the X,Y or Z projection based on the surface normal.

You can modify the scale and draw feedback parameters.

Uv type

Maps from UV coordinates for the geometry. Assigns the parametric coordinate system to the texture space.

You can modify the following parameters.

  • U Scale: Specifies the ratio between the height and the width of the texture space.

  • V Scale: Specifies the ratio between the height and the width of the texture space.

  • Draw Feedback Vector check box: Displays a vector on the graphics window, denoting the direction in which the pattern dynamically moves, after you select any of the following vectors:

    • : Moves the pattern along the X axis in a range from -5 to 5 scale units.

    • : Moves the pattern along the Y axis in a range from -5 to 5 scale units.

Camera Direction Plane

Automatically adjusts the texture space vectors — normal vector or normal and up vectors — based on the camera direction of the view.
The Camera Direction Plane option is similar to the Arbitrary Plane option for texture space mapping.
When you select either option, texture is mapped onto a user defined plane that is defined with a center point, an up vector, and a vector normal to the plane. The difference is that if you use the Camera Direction Plane option, the texture space vectors adjust automatically based on the camera direction of the view. The texture orientation and position update dynamically as you rotate the part. If you use the Arbitrary Plane option, you must click Update Normal to Camera Direction to rectify the texture space mapping.

Note

When you update the texture space based on camera direction, the texture orientation and positioning update dynamically as you rotate the part. This could degrade display performance, especially when the part is large. To delay the update, use the Disable Dynamic Update Camera Direction Plane Texture Space option available in the Visualization Performance Preferences dialog box.

You can modify the following parameters.

  • Center Point: Opens the Point dialog box to let you specify the origin of the texture space.

  • Up Vector: Opens the Vector dialog box to let you specify the reference axis for the texture space.

  • Scale: Controls the overall size of the texture space.

  • Aspect Ratio: Controls the ratio between the height and the width of the texture space.

  • Normal Vector and Normal and Up Vectors: Updates either the texture space normal vector or both normal and up vectors based on the camera direction.
