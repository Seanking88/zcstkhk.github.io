材料/纹理参数区域

The Material/Texture parameter area has five property pages:

  • General — Specifies the general properties of all materials such as color and light.

  • Bumps — Specifies bump map textures and displacement parameters.

  • Pattern — Specifies image texture maps and other parameters that modify color.

  • Transparency — Specifies parameters that make the surface translucent in some way.

  • Texture Space — Specifies parameters that determine how a texture wraps around an object.
