Check-Mate

预计完成此课程需要：1–2 小时

在这个课程中，您将：

  • 了解 Check-Mate 的用户界面，包括 Check-Mate HD3D 工具

  • 设置并运行测试

  • 解决发现的错误

  • 创建并运行定制的检查包

  • 生成日志文件，使用外部的 Check-Mate 查看器查看报告，并使用质量仪表板报告生成器创建报告格式

系统需求

为了进行这个课程中的练习，您必须拥有：

  • Check-Mate 许可证

  • Knowledge Fusion Author 许可证

  • Java Runtime 1.7.0_03 (最低)

    注释

    在尚未本地安装 NX 的 Microsoft Windows 平台中，可能需要创建 UGII_JAVA_HOME 环境变量，以便质量仪表板应用模块可以找到 JAVA 运行时环境文件。
