Check-Mate HD3D 工作流

如果您是运行 Check-Mate 测试的设计者，并且使用这些结果来检查模型中有问题的区域，则所需的所有 Check-Mate 功能均可以从资源条上的 HD3D 工具选项卡中进行访问。

Start with an open part, assembly, or drawing being designed.

On the HD3D Tools tab, double-click Check-Mate.

In the Settings group, click Set Up Tests .

On the Parts tab, select the part file and the levels of the assembly to check.

On the Tests tab, from the Categories list, select the tests to run. Select from standard tests or from your local company profiles.

Click Add To Selected to move the selected tests to the Chosen Tests list.

Click Close.

To run the chosen tests, in the Check-Mate HD3D tool, in the Controls group, click Execute Check-Mate Tests .

To sort by the result status, in the Check-Mate HD3D tool, in the Results group, click the Result column heading.

To filter results, use the options in the Settings group.

To review any problem areas, click the result objects in the Results group. Double-click to see information in the Check-Mate Result window.

View a summary of the Check-Mate results for each assembly component in the Assembly Navigator:

  1. Right-click in the Assembly Navigator and select Columns→Check-Mate Result.

To edit a feature in an assembly component:

  1. In the Check-Mate HD3D tool, right-click an error and choose Make Work Part.

  2. Right-click and choose Select Associated Objects.

  3. Edit the feature using the Part Navigator.

Continue to work with the model and review tests.

Press Ctrl+3 to temporarily hide HD3D tags in the graphics window.

Open other parts to run the same tests.

Deactivate Check-Mate when finished, using Deactivate at the top of the HD3D Tools Manager.
