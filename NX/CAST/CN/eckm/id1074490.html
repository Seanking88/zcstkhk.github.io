设置搜索文件夹

  1. 在质量仪表板 — 报告生成器对话框中，点击设置搜索路径 。

  2. 在搜索路径对话框的设置路径列表中，确保选择的是本地日志。

  3. 浏览至您在练习 创建 Check-Mate 日志文件 中创建的文件夹，然后点击打开。

    Path

    [Check-Mate 练习文件]\logs

    输出文件夹必须为可写。如果您在路径输入框中手动键入名称，那么您可能需要包含驱动器盘符。

  4. 在搜索路径对话框中点击 。

    文件夹名称将被添加到本地日志文件夹列表中。

  5. 如果搜索路径列表中包含多个路径，请选择您刚刚添加的文件夹，然后点击 以将您的文件夹移动到列表顶端。

    您也可以将其它文件夹从列表中删除。

  6. 指定图像文件夹。

    1. 在设置路径列表中，选择图像。

    2. 浏览至您刚刚创建的 my_reports 文件夹并打开。

    3. 在搜索路径对话框中点击 。

    4. 如果搜索路径列表中包含多个路径，请选择您刚刚添加的文件夹，然后点击 以将您的文件夹移动到列表顶端。

      您也可以将其它文件夹从列表中删除。

  7. 指定报告文件夹。

    1. 在设置路径列表中，选择报告。

    2. 浏览至您刚刚创建的 my_reports 文件夹并打开。

    3. 点击 。

    4. 如果搜索路径列表中包含多个路径，请选择您刚刚添加的文件夹，然后点击 以将您的文件夹移动到列表顶端。

      您也可以将其它文件夹从列表中删除。

  8. 点击确定。

  9. 在质量仪表板 — 报告生成器对话框中，选择文件→属性。

  10. 在属性对话框中，点击目录选项卡。

  11. 在图像工作文件夹输入框中，输入路径 [Check-Mate 练习文件所在目录]\my_reports\charts。

    图像工作文件夹是您在查看显示的报告时图表图像临时存放的文件夹。

  12. 在图表图像子文件夹输入框中，键入 charts。

    当您以 HTML 格式导出报告时，您需要为 HTML 文件指定一个位置，报告生成器将把所有的图表图像都存放在 HTML 文件所在位置的子文件夹中，子文件夹的名称即在图表图像子文件夹输入框中定义。

  13. 点击确定。
