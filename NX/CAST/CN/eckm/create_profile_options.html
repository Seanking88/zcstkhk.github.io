创建/编辑检查包选项

选项

描述

检查包名称

为检查包设置一个独一无二的名称。

浏览

启动检查包对话框，您从类别列表中选择一个检查包，然后编辑该检查包，或者使用该检查包作为模板来创建新的类别。

列表框将使用如下所述的颜色编码：

蓝色 - 表示检查包

黑色 - 表示检查器

灰色 - 表示类别

显示名称

您输入的名称会在类别框中显示。

测试类别

允许您输入在其中显示检查包的类别名称。类别可以是使用句号 (.) 字符以分隔多个类别级别的组合名称。可以根据需要在类别名称中使用空格和其他字符。例如，如果输入：

Engineering.File 检查器

那么该检查器或检查包会在“文件检查器”子目录的“工程”下可用。

何时检查

提供以下可用于指定执行检查的条件的选项：

  • 手工 - 仅在单击执行 Check-Mate 测试或执行当前测试命令时运行测试。

  • 更新之后 - 在手动模式下以及在任何部件的 NX 更新完成之后运行测试。

  • 部件保存 - 在手动模式下运行测试，并在保存部件前再次运行测试。

允许运行时更改选项支持您在运行过程中更改“何时检查”选项。

快速参考

打开 Check-Mate 检查器的类和函数参考文档。

文档

让您向检查包添加可向用户提供此检查包常规信息的特殊附注。

在部件中保存日志

将来自当前检查包的数据写入部件文件。

允许运行时更改选项支持您在运行过程中更改“在部件中保存日志”选项。

类别

显示要添加到检查包的检查器树或检查包树。您可以使用以下任何一种选择方法：

右键单击类别名称并选择全选类别中的项。

按 Ctrl 并同时单击以选择多个检查器。

按 Shift 并同时单击以选择一系列的检查器。

可以通过双击树中的条目或突出显示选项并单击绿色箭头添加检查器或检查包 。

显示信息

在信息窗口中显示有关检查器的常规信息。必须首先在类别列表框中选择检查器或检查包，然后再单击显示信息。

添加到列表

从类别列表窗口将选定的检查包或检查器添加到选定的检查包和检查器列表窗口。

选定的检查包和检查器

列表窗口，显示从类别窗口选择的检查包和检查器。

移除测试

从选定的检查包和检查器列表窗口移除检查器或检查包。

定制测试

显示检查包的定制对话框。作为检查包作者，可以选择启用或禁用 Check-Mate 用户用于更改检查包属性的选项。选定一个参数后，该参数将在定制过程中对 Check-Mate 可用。否则，该参数为隐藏。

保存后加载

将新创建或新编辑的检查包或检查器自动保存至选择框。

创建菜单文件和工具条文件

创建菜单文件 (.men) 和工具条 (.tbr) 文件。可以创建用于访问检查包的快速访问选项。

指定工具条的信息

启动工具条规范对话框。

检查包另存为

显示文件选择对话框，并支持您更新现有检查包或以新名称保存现有检查包。保存函数的输出目录应为在知识融合中定义的用户 DFA 目录。撰写文件后，会运行语法检查，且会向 NX 重新加载类。列表窗口会报告所有错误。

重置

清空所有字段，以便可以新建检查包。
