Check-Mate 日志文件

日志文件包含 Check-Mate 检查的结果数据。在设置测试对话框的运行选项选项卡中，您可以选择将 Check-Mate 测试结果保存到下列位置的其中一个：

  • 当前部件文件

  • 外部日志文件

    如果您使用的是本地原生 NX，那么外部日志文件将被保存在本地文件系统中；如果您使用的是 Teamcenter Integration for NX，那么外部日志文件将被保存到 Teamcenter 数据库中。

    外部日志文件的文件扩展名为 XML，NX 将同时生成包含结果数据的 XML 文件，以及包含用于显示结果数据的样式表单的 checkmate_log.xsl 文件。

此外，如果编创的检查器中包含保存参数，则用于生成日志文件的选项可以在检查器的定制对话框中找到。

您可以使用质量仪表板实用工具 Check-Mate 查看器以及质量仪表板报告生成器来查看日志文件内容，两者均是基于 Java 的应用程序，在 NX 外部运行，并且不需要任何特殊许可证。

提示

要定义外部日志文件的默认文件夹，请设置环境变量 UGII_CHECKMATE_LOG_DIR。
