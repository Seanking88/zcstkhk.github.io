For vector and tensor quantities, NX will read the components from the solver results file and then calculate various derived quantities like magnitude for a vector quantity or the principal values for a tensor quantity.

For elemental results, a single value of the result quantity calculated at the centroid of the element is plotted. Consequently, a contour plot of elemental results can have a checkerboard appearance because the color is uniform across each element and is typically discontinuous across shared element edges.
