Layout Manager

The Layout Manager toolbar provides access to the commands that control the arrangement of viewports and the assignment of displays to viewports.

Tip

If you do not see the Layout Manager toolbar, right-click in the toolbar area and choose Layout Manager.

  • Use these buttons to specify the number of viewports you want.

  • Use Layout Settings to control which display appears in which viewport.

    Use the Viewport Settings dialog box to determine in which viewport new post views or graphs are displayed. Use the User Selected Viewport option to specify a viewport by clicking it.

  • Use View Synchronize to give all viewports the same orientation.

  • Use Return to Model to remove a post view or graph from the current viewport.
