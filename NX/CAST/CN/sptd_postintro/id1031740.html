Saving display settings as templates

Post view templates let you apply multiple post view settings in a single operation. Post-view templates include:

  • User-defined templates. You can save your current post view as a post view template. This template contains all current post view settings. When you apply the template to another post view, it overwrites all settings for the other post view.

  • System templates. NX provides a number of standard post view templates you can use to quickly apply common contour, deformation, animation, and performance settings.

  • Shared templates. An NX administrator can define a library of customized templates. You can apply these templates to the post views that you create, to ensure consistent and compliant post displays.
