Accessing data components

In the Post-Processing Navigator, to access individual data components:

  1. Expand the result data set node.

  2. Expand the desired step, subcase, mode, or iteration if applicable.

  3. Expand the result type node for the quantity of interest.

Once you create a post view, you can use Set Result to modify and refine your results selection. Depending on your model, result type, and plot type, you can control results averaging for element-nodal results, result locations for beam and shell elements, the coordinate system used for displaying results, scaling, and many other parameters.
