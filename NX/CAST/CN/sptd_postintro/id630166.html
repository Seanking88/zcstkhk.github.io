Importing results

You can import and access results for solves performed outside of the current set of solutions. The following file formats are supported for importing results:

Nastran (.op2)

I-DEAS results file (.unv)

Structures P.E. (.vdm)

I-DEAS Bun file (.bun)

ANSYS Structural (.rst)

ABAQUS ODB files (.odb)

ABAQUS Thermal (.rth)

RS2 files (.rs2)

ABAQUS (.fil)

LS-DYNA state databases (*.*)
