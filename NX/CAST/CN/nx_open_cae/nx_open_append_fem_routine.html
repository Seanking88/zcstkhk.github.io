Combine Visual Basic source code files

Open createFem.vb in a text editor, or a development environment.

This file will become our first application file, but you need to save it under a different name to preserve the original file. Save the file as createAppendApp.vb.

Note

Line numbers referenced in the activity refer to the delivered control files. The corresponding lines in the files you record will vary due to differences in the steps performed during the journal creation.

and copy that into the createAppendApp.vb file. Paste these lines before End Module line.

Tip

When pasting code, be careful that you do not end up with two lines stuck together such as: End SubSub Main. If this happens, separate them by adding a carriage return to drop Sub Main to the next line.

Note

In the newly combined code, you have created a conflict by having two subroutines named Main. You will resolve this conflict later.
