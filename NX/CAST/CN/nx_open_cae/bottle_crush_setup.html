Setting up WinMerge

You can use any differencing tool to compare your application file to the provided answer files. If you choose to use WinMerge, there are some additional configuration steps you can take to make comparing the files more efficient:

To have WinMerge ignore leading spaces in lines for comparison, change the following option:

  Edit→Options

  • Categories

    Compare

  •   Ignore all

  • Before leaving the Options dialog box, you may want to change the Line Difference Coloring option to Character level so that it will show individual character differences, making editing quicker.

  • Categories

    Editor

  • Note

    The Character level option is in the Line Difference Coloring group.

  •   Character level
