Planning an NX Open for CAE application

When planning an NX Open for CAE application, you should approach it like any other software development project. When automating especially complex tasks, you may want to assemble a team consisting of contributors skilled in the following areas:

  • General CAE analysis types and workflows.

  • The NX user interface, and specifically Advanced Simulation workflows and data structures.

  • NX Open skills and experience.

  • Application programming.

As with any other software development project, be sure you understand your functional and UI requirements, the target user, use cases, and the inputs, processing, and outputs for your intended application.
