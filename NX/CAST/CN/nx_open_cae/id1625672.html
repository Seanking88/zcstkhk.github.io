Copying the activity files

Copy the supplied parts and files to the appropriate locations in your NX installation.

  1. Locate the parts file that you downloaded at the beginning of this course.

  2. Extract the files to the directory where NX will be started. Be sure to select the Use folder names check box in the Extract dialog box.

  3. Create a directory named startup in your %UGII_USER_DIR% directory.

  4. Copy the contents of the Templates directory to %UGII_USER_DIR%\startup.

  5. Create a directory named applicationin your %UGII_USER_DIR% directory.

    Note

    Any dialog boxes used with NX can be placed in this folder. NX searches for dialog boxes in the folder where NX was started first, and then it looks in the application directory.
