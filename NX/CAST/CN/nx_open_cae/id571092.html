Set mesh size and element thickness

You created a prompt for the element size earlier in the routine. The two commented-out alternatives are shown here. The first one shows setting the element size using the auto element size divided by 2.5, and the second is simply set to 6.0.

For simplicity, the mesh thickness will be set as 10% of the mesh size.

Note

The automatic mesh size created by the function shown below may differ slightly than what would be produced with the automatic element size button on the meshing form.

Add the following lines to your routine, below those added in the previous section.

            ' Get auto element size, set mesh size, thickness
            Dim myAutoElementSize As Double = Nothing
            theUfSession.Sf.GetAutoElementSize(1, myCAEBody.Tag, myAutoElementSize)
            ' Alternate element size options: 
            ' Using a calculation on auto element size
            ' Dim myElementSize As Double = myAutoElementSize / 2.5
            ' 
            ' or a hard coded value
            ' Dim myElementSize As Double = 6.0
             Dim myThickness As Double = myElementSize * 0.1
