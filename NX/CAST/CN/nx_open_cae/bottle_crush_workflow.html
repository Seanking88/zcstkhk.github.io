Creating an application to automate a CAE workflow

This section of the course describes how to create an application to automate a CAE workflow. You will use the following process:

  1. Use journal files to record and play back the steps for selected tasks in the user workflow.

  2. Create custom dialog boxes and save them as Visual Basic files.

  3. Combine journal files and dialog box definitions into applications using a third-party editor.

  4. Edit the resulting applications to enable the custom dialog boxes, generic selection, and other customizations as needed.

  5. Create custom toolbars and buttons to launch your applications.

  6. Test and debug the customized application.

The activities in the last section of the course follow this process.
