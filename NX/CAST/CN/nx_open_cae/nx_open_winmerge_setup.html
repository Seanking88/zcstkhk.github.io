Compare your application to the target

Siemens PLM Software provides an example of a completed application for this course, named createAppendAppAns.vb.

Before proceeding, it is a good idea to open your application and the completed application file in a differencing and merging tool to make sure that editing errors are caught.

Tip

If you are using WinMerge: Start WinMerge and choose File→Open. In the Select Files or Folders dialog box, specify createAppendAppAns.vb on the left side and createAppendApp.vb on the right.

Note any differences between the two files, and merge differences in the target application to your application file as needed.

Tip

WinMerge can be used to make the file edits instead of typing.
