Combine vb source code files

In a text editor, open bottlemesh.vb.

This file will become our second application file. This file will be saved under a under a new name to preserve the original file. Save this file as bottleMeshApp.vb.

Open bottleMeshDb.vb. Select everything from line 45 (Public Class bottleMeshDb) to End Class. Copy these lines into bottleMeshApp.vb and paste them before the End Module line.

Tip

When pasting code, be careful that you do not end up with two lines stuck together such as: End SubPublicClass. If this happens, separate them by adding a carriage return to drop Public Class to the next line.
