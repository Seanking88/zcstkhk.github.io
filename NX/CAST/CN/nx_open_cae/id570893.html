Seed CycleObjsInPart and set up the Do loop

This project meshes every body in the part with CQUAD4 elements, using a Do loop to cycle through all of the surfaces. The NXOpen.Obj.CycleObjsInPart() function is used to cycle through every object in the part. This function requires the tag of the part to be processed, and the type of object to be located (in this case UF_caegeom_type). The tag of the object found is returned. (“objectTag”). If “objectTag” is NULL, that triggers CycleObjsInPart to start the process. The process finishes when objectTag becomes NULL again, or there are no more objects to cycle through. The loop continues as long as objectTag is NOT NULL.

Add the following lines below those added in the previous section.

    Do
        Dim objType As Integer = 0
        Dim objSubType As Integer = 0
        theUfSession.Obj.AskTypeAndSubtype(objectTag, objType, objSubType)
        If objSubType = NXOpen.UF.UFConstants.UF_caegeom_body_subtype Then
            myCounter = myCounter + 1 'used to set the name of the meshcollector
            ' Get the CAEBody object from it's tag - the current returned tag from CycleObjsInPart
            Dim myCAEBody As CAE.CAEBody = NXObjectManager.Get(objectTag)

Note

The If test and the Do loop will be completed later in the activity.
