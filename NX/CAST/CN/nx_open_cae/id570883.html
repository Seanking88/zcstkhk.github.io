Set up the NX Open Wizard for Visual Studio

Microsoft Visual Studio 2010 or Microsoft Visual Basic Express 2010 is recommended as the development environment for this activity. NX provides wizards for these environments to facilitate building NX Open projects.

There are three NX Wizards integrated with Visual Studio. They are: NX Open wizard for C and C++ programs, NX Open VB wizard for Visual Basic programs, and NX Open C# Wizard for C# programs. You will set up only the Visual Basic wizard. If Visual Studio is installed prior to NX, the NX Wizards will be installed automatically. If not, follow these steps to set up the VB wizard:

Note

Make sure that Microsoft Visual Studio or Microsoft Visual Basic Express is installed.

Note

The installation directory for Visual Basic Express 2010 should be:

...\MVS10\VB\, with “...” representing your base installation location.

Copy any files and/or folders under each subdirectory from the NX directory to the correct Visual Studio installation.

Copy %UGII_BASE_DIR%\UGOPEN\vs_files\VB\VBWizards\ to

...\MVS10\VB\VBWizards\ and

Copy %UGII_BASE_DIR%\UGOPEN\vs_files\VB\vbprojects\ to

...\MVS10\VB\vbprojects\
