Create and initialize object references

In order to create the required objects and use the methods to operate on them, the object references to the Base Part, Base FEMPart and Base FEModel need to be declared. The objects that you will need for this project, like a mesh collector, are created using a reference to these objects. Add the highlighted code below Sub Main(). 

Sub Main()
    ' Create/initialize object references
    Dim basePart As BasePart = theSession.Parts.BaseWork
    Dim baseFEMPart As CAE.BaseFemPart = CType(basePart, CAE.BaseFemPart)
    Dim baseFEModel As CAE.BaseFEModel = CType(baseFEMPart.BaseFEModel, CAE.BaseFEModel)

Note

The code created up to this point is common to many NXOpen.CAE projects, so this does not need to be rewritten for every project. Copy/paste from a existing project or use the Visual Studio Code Snippet capability to save time and effort.
