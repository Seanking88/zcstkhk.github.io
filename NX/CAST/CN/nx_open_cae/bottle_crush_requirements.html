Required software

The activities for this course use the Visual Basic .NET language. To develop applications in this language and to complete the activities, you need certain software:

  • A text editor. While any text editor is sufficient, an editor that supports syntax highlighting for Visual Basic will help you to see the code more clearly.

  • An integrated development environment (IDE), specifically, Microsoft Visual Studio 2010. If you do not have Microsoft Visual Studio 2010, you can use Microsoft Visual Basic Express 2010, a freeware IDE developed by Microsoft. It is a lightweight version of Microsoft Visual Studio, and is sufficient for completing the activities.

    Note

    Microsoft Visual Studio 2010 and Microsoft Visual Basic Express 2010 are the minimum versions or those products that are supported for these activities.

    You can download Microsoft Visual Basic Express 2010 from the following URL: http://www.microsoft.com/express/download/

  • A file comparison tool. To facilitate the activities, you will compare your code with a completed example and resolve any differences. You can use any tool that allows you to compare and merge files, and there are many freeware file comparison tools available. The examples shown in this workshop use WinMerge, a freeware, open-source differencing tool.

    You can download the latest version of WinMerge from the following URL: http://winmerge.org/
