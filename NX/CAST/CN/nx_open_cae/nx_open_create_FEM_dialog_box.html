Define a Create FEM file dialog box

The first dialog box you will define is used to create a FEM file. It contains a selection step that prompts the user to select the solid body used in creating the FEM file. The dialog box also contains the standard navigation buttons OK, Apply, and Cancel.

Note

NX should still be open.
