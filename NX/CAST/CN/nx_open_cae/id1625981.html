Setting up a user directory

You will use the %UGII_USER_DIRECTORY% variable to point to this directory location.

In a Windows command prompt window, use the set command to verify the location of your %UGII_USER_DIR% directory.

If the %UGII_USER_DIR% variable is not set:

  • Check your nx-install-directory\ugii\ugii_env_ug.dat file.

    If the %UGII_USER_DIR% variable is commented out, remove the # to uncomment this line and enable this variable.

  • (Optional) Change the location of your %UGII_USER_DIR% as desired. You can:

    • Modify your ugii_env_ug.dat file.

    • Define the location of the directory using a Windows system variable.

For more information, see “How NX Finds Applications Files” in the NX Open Programmer's Guide, in the NX Help Library.
