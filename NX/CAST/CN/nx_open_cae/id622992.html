Prompt for element size

In order to make this routine more generic, you will create a user prompt to get the desired element size from the user. Later on in the routine you will see two more alternatives for setting the element size that have been commented out.

Add the following lines below the ones from the last step to create the dialog box prompt for the element size. Note that the default value is set to 10. This prompt is placed outside of the Do loop so that the prompt does not occur each time through the loop.

    ' Prompt for element size
    Dim myElementSize As Double
    myElementSize = NXInputBox.GetInputNumber("Element Size ", "Element Size ", "10 ")

    theUfSession.Obj.CycleObjsInPart(basePart.Tag, NXOpen.UF.UFConstants.UF_caegeom_type, objectTag)
