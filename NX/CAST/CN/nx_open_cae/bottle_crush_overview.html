CAE 中的 NX Open 概述

CAE 中的 NX Open 功能允许您自动化进行 CAE 流程，并在 NX 中创建定制的 CAE 应用程序，这个课程将分为三个部分：

  • 在第一个部分，您将学习标准 NX Open 编程与使用 CAE 中的 NX Open 编程的区别。

  • 在第二部分，您将学习创建简单应用程序来使典型的 CAE 工作流程自动进行的过程。

  • 在第三部分，您将通过几个练习来实践所学的 内容，自动化 CAE 工作流程。

先决条件

这个课程假设您已经学习了 NX Open API 编程的引导式课程，或者等价的在非 CAE 环境中的 NX Open 实际应用经验。
