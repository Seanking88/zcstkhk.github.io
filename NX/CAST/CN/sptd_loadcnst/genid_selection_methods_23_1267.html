Filtering entity selection

When you need to select entities in the graphics window, you can filter the entities that you can select using the Type Filter and Selection Scope. As you move your cursor over the model, only the entities that are available for selection are highlighted.

In Design Simulation you can use the Method options on the Selection bar to select related entities easily and more efficiently. To further refine your method filter, use the Smart Selector Options button.

For example, suppose you want to select all cylindrical faces in a body to apply bearing loads. If you set the Method list to Cylinder Faces, and select the polygon body, the software selects all the cylindrical faces in the solid body.
