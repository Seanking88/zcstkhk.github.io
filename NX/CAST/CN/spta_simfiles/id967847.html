使用 Teamcenter Integration 管理 CAE 数据

将 Teamcenter Integration for NX 与 Teamcenter for Simulation 配合使用时，可通过数据库关系跟踪和管理 CAE 组件和相关联的 CAD 文件关系。原生 NX 中的部件版本装配加载选项没有任何限制，您可以轻松地加载 CAE 模型数据。使用 Teamcenter Integration for NX 和 Teamcenter for Simulation 的其他优点包括：

  • 以透明方式支持并行工程，包括更新通知和工作流管理

  • 在 Teamcenter Integration 客户端中使用 CAE 管理器创建并管理大型 CAE 结构的能力。您的 CAE 结构可以包含多个相关的 CAE 模型

  • 能够在 Teamcenter Integration for NX 中使用仿真搜索视图快速找到所有与给定 CAD 部件相关的 CAE 数据

  • 在 Teamcenter Integration for NX 中使用装配加载选项的版本和版本控制。可以选择性加载指定版本的 CAE 模型数据
