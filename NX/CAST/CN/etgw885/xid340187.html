User defined object names in NX

What is it?

The length of names of Features, Feature Groups, and Groups is now increased to 128 bytes.

Example

Model History
Datum Coordinate System (0)
Sketch (1) "FRONT_PROFILE_OF_SECTION_AA_LEFT_SUPPORT_FLANGE"
Extrude (2) "PRIMARY_LEFT_SUPPORT_FLANGE_BODY_FOR_PART_NO_19110"
Shell (3)
Extrude (4) "LEFT_SUPPORT_CUTOUT_FOR_BOLT_CLEARANCE"
Unite (5)
Feature Group (9) "BLENDS_FOR_FRONT_LEFT_SUPPORT_FLANGE_BRACKET"
Edge Blend (6) "LEFT_SUPPORT_CUTOUT_FILLET_RADIUS_PER_SPECS"
Edge Blend (7) "OUTER_CORNER_RADII_FOR_FRONT_LEFT_SUPPORT_FLANGE"

Why should I use it?

You can use descriptive object names for easier identification during different stages of design.

Where do I find it?

Location in dialog box

Any dialog box where you enter the name of the object or rename the object.
