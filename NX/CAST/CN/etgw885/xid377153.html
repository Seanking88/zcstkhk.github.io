Import Points from File

Use this command to import point data stored in ASCII formatted files. The point data is stored as one point per line and the point XYZ coordinates are separated by blanks, tabs, commas, or similar characters.

The point data can originate from different sources. The most typical sources are scanning devices used to digitize physical parts.

You can use the imported point data to reverse engineer sheet bodies. You can also verify the production of physical parts by comparing the points in the ASCII file to the geometry in the NX CAD model.

Where do I find it?

Menu

File→Import→Points from File
