Selection Preferences enhancement

What is it?

You can now use the Lock Dialog Position selection preference to fix the location of the QuickPick dialog box in the graphics window.

If you select this preference and then move the QuickPick dialog box to a new location, that location is fixed. When you reopen the QuickPick dialog box, it appears at the fixed location and not the default location near the selection point.

Where do I find it?

Application

Gateway

Menu

Preferences→Selection

Location in dialog box

Selection Preferences→QuickPick group
