Use the Projected Distance between objects measurement

  1. On the Utility toolbar, click Show , select all three lines and click OK.

  2. Orient the view to Bottom, and in the View toolbar, click Perspective to turn it off.

  3. On the Utility toolbar, click Measure Distance .

  4. From the Type list, select Projected Distance between Object Sets.

  5. In the graphics window, on the OrientXpress tool, select the Y Axis.

  6. From the Type Filter list, select Edge.

  7. For the Start Objects, select the top circular edge.

  8. For the End Objects, select the lower circular edge.
