Save Options enhancement

What is it?

Use the new Share Geometry on Save option to reduce the size of your NX part file when it contains multiple copies of bodies created by the Extract Geometry command.

Note

Set the Enable PMI Support for Geometry Sharing option on the General tab of the PMI Preferences dialog box to ensure PMI attached to the main part geometry treats the shared geometry in the extracted body as if it were the same as the main geometry.

Why should I use it?

This command is useful, for example, when extract bodies are used to represent multiple intermediate manufacturing states within the same part file, and your file contains multiple extracted bodies from the same model.

Where do I find it?

Menu

File→Options→Save Options
