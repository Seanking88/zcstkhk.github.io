Using Simple Distance

  1. Choose Analysis→Measure→Simple Distance.

    Notice the absence of the most of the dialog box groups.

  2. Select outer face of the tube.

  3. Select curved face of the panel. that is the closest to the tube.

  4. Zoom in and note the value and location of the measurement.

  5. Click OK.

  6. Right click and choose Undo.

  7. Choose Analysis→Measure→Simple Distance.

    Again, notice the absence of the most of the dialog box groups.

  8. On the dialog box rail, click Dialog Options and choose Measure Distance (More).

    Notice the dialog box groups are again displayed.

  9. from the Type list, select Diameter.

  10. Select one of the edges at either end of the tube.

  11. Click OK.

  12. Close the part.
