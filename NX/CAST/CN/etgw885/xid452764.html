Set the distance measurement dialog box

  1. Open gwy1_85_measurements_1

  2. Choose Analysis→Measure Distance.

  3. On the dialog box rail, click Reset .

  4. From the Type list, choose Between Object Sets.

  5. In the Associative Measure and Checking group, select the Associative check box.

  6. On the dialog box rail, click Dialog Options and make sure Measure Distance (More) is selected.

  7. In the Results Display group, from the Annotation list, choose Show Dimension.
