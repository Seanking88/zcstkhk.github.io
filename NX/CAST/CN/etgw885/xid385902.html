Use Set Rotation Reference

  1. In the graphic window background, right-click and choose Set Rotation Reference.

  2. From the Type filter list, select Edge.

  3. Select the top edge of the blend.

  4. Using the center mouse button, near the center of the graphic window background, rotate the view.

    Notice it rotates about the selected edge.

  5. Hold down the Alt key and use the center mouse button the rotate the view.

    Notice it now free-rotates about the center of the view.

  6. Release the Alt key and continue to rotate the view. Notice it again rotates about the selected edge.

  7. In the graphic window background, right-click and choose Clear Rotation Reference.

  8. In the graphic window background, right-click and choose Restore.
