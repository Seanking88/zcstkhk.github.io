Use Fit to Work Section

  1. On the View toolbar, click Clip Work Section.

    A box view section has been created showing only one pin.

  2. On the keyboard, hold down the Alt key, and in the graphic window background, use the center mouse button to free-rotate the view.

    Notice it rotates about the view center.

  3. In the graphic window background, right-click and choose Restore.

  4. Release the Alt key and use the center mouse button to rotate the view.

    Notice it now rotates about the center of the displayed geometry.

  5. Choose Preferences→Visualization→View/Screen tab.

  6. In the Session Settings group, select the Fit to Work Section check box and then click OK.

  7. Fit the view.

  8. Close the part without saving.
