Change smooth edge display

  1. Choose Preferences→Visualization.

  2. Click the Visual tab.

  3. Expand Edge Display Settings.

  4. In the Smooth Edges group, on the Color line, clear the Original check box, and click the color swatch .

  5. In the Color dialog box, select color number 189 Strong Brown, and click OK.

  6. On the Width line, from the Width list, select 0.13 mm.

  7. Click OK.
