Create a user defined filter file

Tip

Entity Filters in PCB Exchange control which and how files are translated to the new file when importing or exporting an ECAD model.
Components, restriction areas, and drilled holes can be filtered out or written depending on specified criteria.

  1. In your working folder ..\pcb_activities, create a text file named mapped_components.txt and write in it: 

    xs1a, *, xt41a, $
xs2, *, xt42, $

    Tip

    You are instructing PCB Exchange to read the xt41a components where it finds xs1a instances.

  2. Save and close mapped_components.txt.

  3. You will point to this file from the pcbx_ug_filter.ini file in the next step.
