In the following activities you will get familiar with the capabilities of NX PCB Exchange using a Printed Circuit (PC) Assembly previously created from an existing ECAD file.

Before starting the activities you must download the parts.

  1. On your desktop or the appropriate network drive, create a folder named student_home.

  2. Click the Download link below:

    Download the parts

  3. Extract the part files to your student_home folder.

  4. Start NX.

Note

The first three activities use the same parts located in folder /pcb_activities. Parts for the last activity are located in the folder /pcb_thermal.
