Compare and Update the NX model with an ECAD file

You will use the Compare and Update PCA functionality of PCB Exchange to update the NX PC assembly based on the latest ECAD file, Tut_Motherboard_C.brd.

Compare and Update PCA (PCB Exchange toolbar)

  • Browse (Board File, ECAD Model group)

  • Files of type

    brd File (*.brd)

  • File Name

    Tut_Motherboard_C.brd

  • OK

      Compare to ECAD Model dialog box

    Use Entity Filters on ECAD model

     (Verify)

  • OK

     

  • Update PC Assembly dialog box appears with the Information window.

    Information window

  • Update Complete Assembly

     (Verify)

  • Select all components that begin with an R
    R***
    R1... <shift>... R9

  • No Update

     

    Tip

    Observe that the description of all components beginning with an R changed to “Change: No”

  • OK

     

  • Rotate to see the back of the PCA.

Note

Your file has been updated with new components.
