Modify pcbx_ug_filter.ini

The initialization files contain settings on display, filters, meshing, and other properties.

Note

For more information about .ini and .xml setting files see:
PCB Exchange→PCB Exchange application settings→Modifying settings and PC assembly data

  1. In your working folder, open pcbx_ug_filter.ini with a text editor.

    Tip

    Observe the other predefined filters.

  2. To enable component height filtering, at the end of the document, insert the following:

    # This filter removes components of less than 2mm
[ComponentMinHeight 0.002]
ComponentMinHeight=0.002

  3. To enable user defined component mapping, insert the following:

    # This filter maps components
[mapped components activity]
ComponentMapFile=..full path..\mapped_components.txt

    Caution

    Replace ..full path.. by the full system path of your pcb_activities folder.

  4. Save and close the pcbx_ug_filter.ini file.
