You will temporarily set system environment variables instructing PCB exchange to use the .ini files of your working folder.

  1. Close NX.

  2. Right-click on My Computer and select Properties.

  3. On the Advanced page, click on the Environment Variables button.

  4. Create a new variable called: MAYA_PCB_ENV_DIR and set its value to the folder you are working on ..full path../pcb_activities.

  5. Click OK to all windows.

  6. Start NX.
