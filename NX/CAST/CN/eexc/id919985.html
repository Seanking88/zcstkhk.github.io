移除并重新排列表达式顺序

您将从列表中移除不需要的表达式，并重新排列剩余表达式的顺序。您只需要保留用于修改齿轮的表达式。

  1. 从 B 列中选择所有的 _p 单元格以及在 C 列中相应的值。

    单元格将高亮显示。

  2. 右击高亮显示的单元格并选择清除内容。

  3. 右击包含 D 以及 T 的单元格所在行编号并选择删除。

  4. 删除 hub_offset 以及 length 所在行。
