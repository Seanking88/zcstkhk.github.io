完成柔性连杆的定义

柔性连杆对话框从上一个步骤中保持打开。

  • 柔性连杆预览窗口为您显示划分网格的柔性体，以及它相对于绝对坐标系的方位。

  • 放置

    绝对原点

    因为柔性体与机构中的相应连杆位于相同的方位(相对于绝对坐标系)，因此绝对原点是最适合的选择。

    提示

    为了更形象地观看，可以在柔性连杆预览窗口中按住鼠标中键来旋转预览的模型。

  • 连接

  • 列表中的三个 Marker 对应于定义在这个连杆上的两个衬套和球面副的原点。

    确认三个 Marker 被映射到您在在前面的柔性体练习中的高级仿真中定义的连接节点，如果您遵循练习中指定的网格划分步骤，连接节点应该被 Marker 为 node ID 1、2、3，并且 node ID 应该被映射到下列 Marker 上：

    • ball_jt4.i_marker

    • G001.i_marker

    • G002.i_marker

    如果 Marker 未映射到正确的节点，您可以在 Node ID 单元格中直接输入正确的 Node ID(在单元格中单击并键入 ID 号)。
