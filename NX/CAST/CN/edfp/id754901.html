Activity: Create a drawing

Estimated time to complete: 40–60 minutes

In this project, you will:

  • Create a new drawing sheet.

  • Create a top view, then use it to create an orthographic front view.

  • Add an isometric view at a reduced scale.

  • Change the layer visibility in a view.

  • Remove a view from the drawing sheet.

  • Add a detail view.

  • Change the label on the detail circle on the parent view.

  • Change the boundary of the detail view.

  • Create a simple section view.

  • Create a new drawing sheet.

  • Move the section view from the first drawing sheet to the second.

Launch the Create a drawing activity.
