层片材料

层片材料是层合板特定的对象，它为层片指派材料属性。可以创建以下内容：

  • 组合纤维和基质 NX 材料属性的纤维强化的层片材料。它使用指定的纤维强化类型计算等效的刚度和强度属性。

  • 可对层合板夹层的横向剪切核心失效进行建模的核心层片材料。

    NX 使用核心层片材料中的值计算核心层片的核心剪切失效指数、核心强度比率和核心安全裕度。核心结果将替换层合板结果的高级后处理报告中的平面内层片结果。失效指标是 13 方向和 23 方向的输出。

核心层片材料与标准 NX 材料不同。使用 NX 材料创建层片材料。可在层合板建模器、铺层建模器和层合板层片材料管理器对话框中专门管理层片材料。

层片材料名称和 ID

NX 通过连接层片材料类型和连续 ID 自动创建层片材料名称。通过在层合板层片材料管理器对话框的层片材料列表中单击适当单元格可以更改名称和 ID。
