Using a spreadsheet or a CSV file to define a layup

Use the Import Layup from Spreadsheet or Import Layup from CSV File options to define your layup in an Excel spreadsheet or a CSV file and import it into the NX. The layup must be defined using three columns and as many rows as there are plies. The columns must contain, from left to right:

  • Material name. This is the name of an existing NX material, or ply material. If the material name is not found in the FEM file, the layup is not imported.

  • Thickness, using the FEM units.

  • Orientation angle, in degrees.

Note

When you import a layup into an existing layup, it overrides the layup definition.

When you import a layup from a spreadsheet or an CSV file, the Stacking Recipe list is reset to Regular. The contents of the ply layup is always identical to the contents that is displayed in the Ply Sketcher group.
