Fiber-reinforced ply material reference temperature

The ply material reference temperature is set to (in this order):

  1. The Reference Temperature value of the fiber material, if defined.

  2. The Reference Temperature value of the matrix material, if defined.

  3. 20°C

When the ply material is used in the laminate, the laminate reference temperature overrides the ply material reference temperature.

Fiber-reinforced ply material Tsai-Wu interaction coefficient

The ply material Tsai-Wu interaction coefficient is set to (in this order):

  1. The Tsai Wu Interaction value specified in the Laminate Ply Material dialog box, if defined.

  2. The Tsai-Wu Interaction Coefficient (F12) value of the fiber material, if defined.

  3. The Tsai-Wu Interaction Coefficient (F12) value of the matrix material, if defined.

  4. 0.0

Fiber-reinforced ply material finished thickness

Optionally, you can specify the Finished Thickness for:

  • Unidirectional

  • Random Short Fiber

  • Woven

The ply material Finished Thickness overrides the ply Thickness defined in the Laminate Modeler and Layup Modeler dialog boxes.
