Search Tips

 Using keywords

  • Search for one or more keywords. For example, model or model file.
  • Search for phrases by enclosing the keywords in quotation marks. For example, "model file" returns only pages that contain the words model and file immediately next to each other.
  • Search for forms of keywords. For example, enter extrude for results that contain extrude, extrudes, and extruding.
  • Search using an asterisk (*) as a wildcard character at the end of a word or group of characters. For example, model* returns pages that contain the words models or modeling, but not model. If you use an asterisk, the search engine returns only instances of the keyword when it occurs with additional characters. Note that the asterisk is not required to find forms of a word (see the previous tip).

Using Boolean expressions

  • When you enter more than one keyword, you can include Boolean expressions (and, or, and not).
    • Enter model file to find pages containing both the word model and the word file.
    • Enter model or file to find pages containing either the word model or the word file.
    • Enter model not file to find pages containing the word model, but not the word file.
  • Use parentheses in Boolean expressions. For example, enter (part or assembly) and model to find pages containing the word model and either the word part or the word assembly.
  • If you search for more than one keyword and do not include a Boolean expression, the and expression is used. If you enter model file, it is equivalent to entering model and file.

Using proximity of keywords

  • Search for keywords that occur near each other. For example, "model file"~5 finds pages where the words model and file appear within five words of each other. Note that common words, such as the or and, are not counted as part of the five-word frame.
