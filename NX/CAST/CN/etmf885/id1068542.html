Edit the tool path to extend the walls

  1. In the Path Settings group, click Cutting Parameters .

    The Cutting Parameters dialog box is displayed.

  2. Click the Containment tab.

  3. In the Cut Regions group, select the Extend Walls check box.

  4. Click OK.

    The Floor Wall dialog box is displayed.

  5. In the Actions group, click Generate .

    Notice the tool path is contained by the floor and extended walls of the part.

  6. Click OK to save the operation.

  7. Close the part without saving it.
