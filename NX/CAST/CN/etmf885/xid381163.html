Floor and Wall milling using Exact Positioning

When you create Floor and Wall and Floor and Wall with IPW operations using for example, an end mill with a corner radius, use the Exact Positioning option to control how the cutter is positioned to tapered walls.

Exact Positioning would be most useful in finishing operations, but will require more time for processing the tool path.
