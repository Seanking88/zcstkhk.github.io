Create the operation

  1. In the Operation Navigator, right-click PROGRAM and choose Insert→Operation.

  2. In the Create Operation dialog box, set Type to mill_planar.

  3. In the Operation Subtype group, click Floor and Wall .

  4. In the Location group, set the following:

    • Program = PROGRAM

    • Tool = MILL

    • Geometry = WORKPIECE

    • Method = MILL_FINISH

  5. In the Name box, type LIMIT_FLOOR.

  6. Click OK.

    The Floor Wall dialog box is displayed.
