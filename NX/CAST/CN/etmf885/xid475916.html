Review the existing operation

  1. Open mnf16_85_gaps_start_point.

  2. In the Operation Navigator, double-click the WALL_FINISH_AUTO_START_MID_POINT operation.

    The Contour Profile dialog box is displayed.

  3. In the Geometry group, in the Specify Floor row, click Display .

  4. Refresh the display.

  5. Zoom in on the part.

  6. In the Specify Walls row, click Display .

  7. Refresh the display.

  8. In the Drive Method group, in the Method row, click Edit .

    The Contour Profile Drive Method dialog box is displayed.

  9. In the Start of Cut group, notice the following settings.

    • Start Point Option = Automatic

    • Extend Distance = 0.0 in

    • Tool Axis = Automatic

  10. In the End of Cut group, notice the following settings.

    • End Point Option = Automatic

    • Extend Distance = 0.0 in

    • Tool Axis = Automatic

  11. Click OK.

    The Contour Profile dialog box is displayed.

  12. In the Path Settings group, click Non Cutting Moves .

    The Non Cutting Moves dialog box is displayed.

  13. Click the Start/Drill Points tab.

  14. In the Region Start Points group, notice the following setting.

    • Default Region Start = Mid Point

  15. Click OK.

    The Contour Profile dialog box is displayed.

  16. In the Actions group, click Replay .

    Notice that the Automatic setting starts and ends the cut at the mid point of the longest wall.
