Start at a corner

  1. In the Path Settings group, click Non Cutting Moves .

    The Non Cutting Moves dialog box is displayed.

  2. Click the Start/Drill Points tab.

  3. In the Region Start Points group, set the following:

    • Default Region Start = Corner

  4. Click OK.

    The Contour Profile dialog box is displayed.

  5. In the Actions group, click Generate .

    Notice that the Automatic setting starts and ends the cut at a corner.

  6. In the Drive Method group, in the Method row, click Edit .

    The Contour Profile Drive Method dialog box is displayed.

  7. In the Start of Cut group, set the following:

    • Extend Distance = 100.0 %Tool

  8. In the End of Cut group, set the following:

    • Extend Distance = 100.0 %Tool

  9. Click OK.

    The Contour Profile dialog box is displayed.

  10. In the Actions group, click Generate .

    Notice the start and end of cut extensions.
