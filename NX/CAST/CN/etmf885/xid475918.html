Define a start point

You will start the tool path at the opposite corner.

  1. Open the Non Cutting Moves dialog box

  2. Click the Start/Drill Points tab.

  3. In the Region Start Points group, from the Specify Point list, select End Point .

  4. Select the edge shown to define the end point.

  5. Click OK.

  6. Generate the tool path.

    Notice that the tool path starts and ends where the selected point projects to the current cut level.
