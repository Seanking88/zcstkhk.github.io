Tool export reports

What is it?

When you successfully export a tool to the ASCII tool library or a Teamcenter library, NX:

  • Generates a results report for the export.

  • Displays a simplified version of the report in a message.

  • Writes a detailed version of the report to the log file.

Where do I find it?

Application

Manufacturing

Operation Navigator

Edit a tool

Location in dialog box

Library group, Export Tool to Library
