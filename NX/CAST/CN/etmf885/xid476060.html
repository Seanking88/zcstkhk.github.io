Define a tool library

The steps on this page assume that you do not have write permission to your tool library, or that you do not want to risk adding an example tool that is not available to your company.

  1. Choose Help→Log File.

  2. In the Information window, choose Edit→Find.

  3. In the Find dialog box, in the Find what: box, type UGII_CAM_LIBRARY_TOOL_DIR, and click Find Next.

  4. Copy or make note of the location of the tool library.

    For example, the default location is C:\Program Files\Siemens\NX 8.5\mach\resource\library\tool\.

  5. In your operating system, open a navigation window, for example, Windows Explorer, and navigate to the folder that contains the tool library.

    For example, the default location is C:\Program Files\Siemens\NX 8.5\mach\resource\library\.

  6. Copy the tool folder and paste it into your home folder, or to a convenient location where you have write permission.

  7. Redefine the environment variable UGII_CAM_LIBRARY_TOOL_DIR by doing one of the following:

    • If you are in a classroom location, in your operating system, in your home folder, open the file called ugii_env.dat and add a line to set the variable to your copy of the tool library folder.

      The last few lines might look something like this example:

      # insert any other training-specific variables here
#
UGII_CAM_LIBRARY_TOOL_DIR=${HOME}\tool\

#include ${UGII_ROOT_DIR}\ugii_env_ug.dat

      Be careful not to omit the trailing slash.

    • Set a new environment variable with the name UGII_CAM_LIBRARY_TOOL_DIR and the value <location where you put the tools folder>\tool\

  8. Exit from NX, restart NX, and open mnf16_85_export_tool.

  9. Choose Help→Log File, and verify that UGII_CAM_LIBRARY_TOOL_DIR points to the location of your copy of the tool library.
