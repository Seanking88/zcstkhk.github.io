Gouge checking enhancements

What is it?

Gouge checking now displays message boxes to inform you about the following:

  • Non-uniform stock

  • Non-uniform custom part offsets

  • Negative part offsets that are greater than the tool radius

  • Tools other than milling and drilling tools.

You can choose whether to cancel the gouge check, or to continue to run with some limitations.

Why should I use it?

The new message boxes help to clarify the behavior of gouge check.

Where do I find it?

The new message boxes appear at the start of a gouge check for each operation. For simulation in ISV, the message boxes appear at each tool change.
