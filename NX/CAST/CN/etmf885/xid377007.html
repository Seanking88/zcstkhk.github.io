Contour Profile variable axis profiling enhancements

What is it?

Enhancements to the variable-axis Contour Profile operation give you better control of the operation and improve the effectiveness of the tool path. You also have the ability to use this operation as part of a complete finishing flow. You can now:

  • Extend the tool path along cutouts in wall areas, using the new Extend Distance option.

  • Specify how to handle wall cutouts where the start and end of cut extensions leave a gap, using the new Across Wall Gaps option.

  • Specify a start point when machining contoured parts with a closed cross section.

  • Control axial shift of the tool, using the new Ring Height option.

  • Overlap the wall finish passes and the floor finish passes to generate a cleanup operation that removes uncut blank material left by the previous tool.
