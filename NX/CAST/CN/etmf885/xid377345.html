Allow Selection of 2D IPWs

What is it?

You can now turn on or turn off the selectability of 2D IPW geometry using the new Allow Selection of 2D IPWs option.

Why should I use it?

If you turn this option on, you can select the 2D IPW geometry to measure distance, length, and so on. To ensure that the 2D IPW geometry is not accidentally selected when you select part geometry, turn the option off.

Where do I find it?

Application

Manufacturing

Toolbar

Workpiece→Allow Selection of 2D IPWs
