Add passes to finish the floor

  1. In the Path Settings group, click Cutting Parameters .

    The Cutting Parameters dialog box is displayed.

  2. Click the Multiple Passes tab.

  3. In the Multiple Depths group, review the options.

    • Multiple Depths =

    • Depth Stock Offset = 28

    • Step Method = Passes

    • Number of Passes = 7

  4. In the Multiple Side Passes group, set the following:

    • Multiple Side Passes =

    • Side Stock Offset = 36

    • Step Method = Increment

    • Increment = 6

  5. In the Order group, from the list, select Depth First.

  6. Click OK.

    The Contour Profile dialog box is displayed.

  7. In the Actions group, click Generate .

  8. Click Verify .

    The Tool Path Visualization dialog box is displayed.

  9. On the 2D Dynamic tab, move the Animation Speed slider to 5.

  10. Click Play .

    Notice that the operation finished the floor, but there is wasted motion.

  11. Click OK.

    The Contour Profile dialog box is displayed.
