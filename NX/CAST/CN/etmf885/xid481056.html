View the facet bodies

  1. Choose Tools→Faceting.

    The CAM Faceting dialog box is displayed.

  2. Right-click in the background of the graphics window and choose Zoom.

  3. In the Faceted Representations group, under List, select the first facet body.

  4. Click Display Facets .

  5. Right-click in the background of the graphics window and choose Refresh.

  6. Select the second facet body.

  7. Click Display Facets .

  8. Refresh the view.

  9. Right-click in the background of the graphics window and choose Fit.

  10. Select the first facet body and click Remove .

  11. Select the second facet body and click Remove .

  12. Click OK.

  13. Save the xxx_cap3_setup_1 setup assembly.

  14. Review the size of the xxx_cap3_setup_1.prt file.

    Notice that the file size has decreased because you deleted the IPW facet bodies. However, the file is still larger than when you started the activity.

  15. In the Operation Navigator, right-click the FLOWCUT_REF_TOOL operation and choose Tool Path→Delete.

  16. Right-click the FLOWCUT_REF_TOOL operation and choose Generate.

  17. Save the xxx_cap3_setup_1 setup assembly.

  18. Review the size of the xxx_cap3_setup_1.prt file.

    Notice that the file size has decreased to approximately 500KB.

  19. Close the setup assembly.
