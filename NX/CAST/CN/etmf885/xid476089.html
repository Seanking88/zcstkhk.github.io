Create a coarse resolution IPW

  1. In the Operation Navigator, double-click the FLOWCUT_REF_TOOL operation.

    The Flowcut Ref Tool dialog box is displayed.

  2. In the Actions group, click Verify .

    The Tool Path Visualization dialog box is displayed.

  3. Click the 3D Dynamic tab.

  4. From the IPW Resolution list, select Coarse.

  5. From the IPW list, select Save.

  6. Move the Animation Speed slider to 3.

  7. Click Play .

  8. Click Create.

  9. Click OK.

    The Flowcut Ref Tool dialog box is displayed.

  10. Click OK to save the operation.

  11. Save the xxx_cap3_setup_1 setup assembly.

  12. On the operating system review the size of the xxx_cap3_setup_1.prt file.

    Notice that the file size is approximately 3600 KB. The file size is larger because of the saved IPW facet body.
