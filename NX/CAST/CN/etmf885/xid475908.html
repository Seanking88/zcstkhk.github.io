Vary the amount of tool shift

You can vary the amount of tool shift to equalize tool wear.

  1. Edit the Contour Profile drive method as shown.

    • Ring Height = Variable

    • Top Distance = 9 mm

    • Bottom Distance = 2 mm

  2. Generate the tool path.

  3. Verify the tool path and select the locations shown to see the tool placement.

    Notice the differences in the tool path and where the tool contacts the wall.

    Shift distance

    None

    4 mm

    Top = 9 mm

    Bottom = 2 mm

  4. Click OK to close the Tool Path Visualization dialog box.

  5. Click OK to close the Contour Profile dialog box.

  6. Close the part without saving it.
