Create the operation

  1. Open mnf16_85_floor_wall_multiple_faces_1.

  2. On the Resource bar, click the Operation Navigator tab.

  3. In the Operation Navigator, right-click in the back ground and choose Program Order View.

  4. On the Insert toolbar, click Create Operation .

  5. In the Create Operation dialog box, set Type to mill_planar.

  6. In the Operation Subtype group, click Floor and Wall with IPW .

  7. In the Location group, set the following:

    • Program = PROGRAM

    • Tool = END_MILL_D30_CR0_FL35

    • Geometry = WORKPIECE

    • Method = MILL_ROUGH

  8. In the Name box, type multiple_faces.

  9. Click OK.

    The Floor Wall IPW dialog box is displayed.
