Set the insert shape and the tooling dimensions

  1. Click the Tool tab.

  2. Select your insert shape. In the Insert group, set the following:

    • ISO Insert Shape = R (Round)

    • Insert Position = Topside

  3. In the Dimensions group, set the following parameters:

    • (BD) Button Diameter = .250

    • (OA) Orient Angle = 90

    • (HW) Holder Control Width = .1625

    • (HA) Holder Control Angle = 4.000

    Note

    In the Numbers group, notice the number in the Tool Number box. The tool number 1 for the round tool you are creating is inherited from then FACE 1 pocket ID.

  4. In the Preview group, click Display .

    Tip

    To clearly see the tool and insert, drag any handle to an empty space in the graphics window.
