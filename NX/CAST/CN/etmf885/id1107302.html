Define the tool material

  1. Click the Tool tab.

  2. In the Description group, click Material .

    The Search Result dialog box is displayed. You will now assign the material for your insert.

  3. In the Matching Items group, from the Library Reference column, select TMCO_00006.

    Library Reference

    Material Name

    TMCO_00001

    HSS

    TMCO_00002

    Carbide

    TMCO_00003

    Carbide

    TMCO_00004

    Carbide

    TMCO_00006

    HSS Coated

  4. Click OK.

    The Turning Tool-Standard dialog box is displayed.

  5. Click OK.

  6. In the Machine Tool view of the Operation Navigator, notice that the round insert is placed inside the FACE1 pocket.
