Control the sequence of cuts between faces, diameter, and fillets.

  1. Open mnf16_85_fillets_mfg_1.

  2. In the Program Order view of the Operation Navigator, expand the PROGRAM node, and double-click FINISH_OD to edit the operation.

    The Finish Turn OD dialog box is displayed.

  3. Notice the following settings..

    • Cut Strategy group

    • Strategy = Diameters First, Then Faces

    • Path Settings group

    • Cut Fillets = With Faces

  4. In the Finish Turn OD dialog box, in the Options group, click Edit Display .

    The Edit Display dialog box is displayed.

  5. In the Edit Display dialog box, in the Tool group, from the Tool Display list, select None, and click OK.

    The Finish Turn OD dialog box is displayed.

  6. In the Actions group, click Generate .

  7. Click Verify .

    The Tool Path Visualization dialog box is displayed.

  8. On the Replay tab, select the 2D Material Removal check box.

  9. Move the Animation Speed slider to 1.

  10. Click Play .

    The fillets are cut when the adjacent faces are cut because, the cut strategy is set to Diameters First, Then Faces and the Cut Fillets option is set to With Faces.

  11. Click OK to complete the tool path visualization.

    The Finish Turn OD dialog box is displayed.
