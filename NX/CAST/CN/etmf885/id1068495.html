Edit the existing cut area by adding wall sets

  1. In the Geometry group, click Specify Wall Geometry .

    The Wall Geometry dialog box is displayed.

  2. In the List group, click Remove .

    The existing wall set is deleted from the list.

  3. Select the two walls as shown for Set 1.

  4. Click Add New Set .

  5. Select the single wall as shown for Set 2.

  6. Click OK.

    The Floor Wall dialog box is displayed.

  7. In the Actions group, click Generate

    The tool path is now constrained by the two new wall sets.

  8. Click OK to save the operation.

  9. Close the part without saving it.
