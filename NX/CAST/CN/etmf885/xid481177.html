Orient the OD cutting tool

  1. Open mnf16_85_tmp_common_flip_tool_mfg_3.

  2. Switch to the Geometry View of the Operation Navigator.

  3. In the TURN_WORKPIECE_LT node, verify and replay the operations.

    Notice the direction of the tool holder.

  4. In the TURNING_WORKPIECE_RT node, verify and replay the operations.

    The tool holder is not oriented correctly for optimal machining based on the cut direction needed for the Right Spindle. The current tool orientation is best suited for machining on with the Left Spindle.

    In order to machine the part with in the Right Spindle, you will need to reorient the tool and flip it around to avoid collisions.

  5. Click Cancel to close the Tool Path Visualization dialog box.

  6. Double-click RGH_TURN_OD_RT.

  7. In the Suppressed Geometry message box, click Yes.

    The Rough Turn OD dialog box is displayed.

  8. In the Tool Orientation group, set the following:

    • Flip Tool Around Holder =

    • Tracking Point = Automatic

    If you select the Automatic option, NX chooses tracking points depending on holder orientation. Additional tracking points are automatically created when necessary.

  9. Generate and verify the tool path.

  10. Click OK twice to complete the operation.

  11. Double-click FIN_TURN_OD_RT.

    The Rough Turn OD dialog box is displayed.

  12. In the Tool Orientation group, set the following:

    • Flip Tool Around Holder =

    • Tracking Point = Automatic

  13. Generate and verify the tool path.

    Notice that the tool is now in the correct orientation for the operation.

  14. Click OK to complete the operation.

  15. Close the part without saving it.
