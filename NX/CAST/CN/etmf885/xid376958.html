VNCK machine tool simulation

Machine tool manufacturers who provide the Sinumerik 840D controller can supply you with files to simulate machine code in NX. The machine code may originate from NX or from an external source. The simulation uses the actual logic of the Sinumerik controller and displays an NX kinematic model of your machine tool. When you use the Simulation Control Panel dialog box, if VNCK is installed and configured for the current machine tool, you can change the Visualization option in the Animation group to Machine Code Simulate.

When you use the Machine Code Simulate option, the Machine Control Panel group is added to the dialog box, and the Human Machine Interface dialog box is displayed. You can control the virtual machine and receive the same alarms and messages as you would on a real machine.
