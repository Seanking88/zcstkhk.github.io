Activity: Machine uncut areas using a Flowcut operation

Estimated time to complete: 7–11 minutes

When uncut material remains because a tool is too large to machine certain areas, you can use a Flowcut Reference Tool operation to remove the remaining material. You select the tool from a previous operation to identify the uncut areas, and then machine with a smaller diameter tool.

In this activity, you will:

  • Create an operation to remove material that remains from a previous operation.

  • Create multiple cut patterns in a single operation.

  • Output more uniform cut depths in the tool path.
