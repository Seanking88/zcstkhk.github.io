Create an ID cutting tool

You will create a standard turning ID_55_L tool named diamond_0.250_id with the inherited tool number of 3.

  1. Create the turning tool with the following parameters:

    • Nose Radius = .03125

    • Inscribed Circle Diameter = .250

    • Tool Material = TMC0_00004

    • P Number = P2

  2. View the tool in the Machine Tool View of the Operation Navigator.

  3. Click OK to complete the operation.

  4. Close the part without saving it.
