Export a tool

  1. In the Machine Tool view of the Operation Navigator, double-click the tool named MILL-.25TD-.25R-15DEG.

  2. In the Milling Tool-5 Parameters dialog box, in the Library group, click Export Tool to Library .

  3. In the Select Target Class dialog box, select END_MILL_NON_INDEXABLE and click OK.

  4. In the Template Attributes dialog box, click OK without assigning an attribute.

    By choosing not to assign a holding system, you are making the tool available for any holding system.

  5. In the Export Tool to Library dialog box, make a note of the names of the new tool and new holder that were exported.

    The ugti0201 prefix for the tool name indicates the library type and subtype of the tool. You can verify this by examining your update tool library file.

  6. In your operating system, use a text editor to open your ...\tool\english|tool_database.dat file.

    In the comment section, approximately 190 lines into the file, the subtypes for the milling tool type 02 are identified, as shown by these truncated lines: 

    #    T   Lib Type    ST  ...
#    --------------|-----...
#    02  Milling     01  ...
#    02  Milling     02  ...
#    02  Milling     03  ...
...

    Subtype 01 in the ST column is identified in the Lib Subtype column as End Mills Non-indexable.

    Because you chose a non-indexable end mill, with type 02 and subtype 01, the automatically assigned name starts with ugti0201.

    The rest of the name is defined by the default starting number for user tools, which your administrator can set up in the customer defaults, and by the next available number in the sequence.

  7. Close the text editor, and in NX, in the Export Tool to Library dialog box, click OK.

  8. In the Milling Tool-5 Parameters dialog box, click Cancel.

  9. Choose Help→Log File, and search for the name of your tool in the Information window.

    The information in the log file includes the names and locations of the library files that were updated, the tool file and the holder file. The file path names include folders that designate the units, english or metric. Adjacent lines also show the tool, shank, and holder parameters that appeared in the editing dialog box when you clicked Export Tool to Library

  10. Close the part without saving it.
