Create a tool with a diamond shape insert

  1. On the Insert toolbar, click Create Tool .

    The Create Tool dialog box is displayed.

  2. In the Create Tool dialog box, in the Tool Subtype group, click OD_55_L .

  3. In the Location group, make sure the Tool list is set to FACE2.

  4. Name the insert diamond_0.250_od.

  5. Click Apply.

    The Turning Tool-Standard dialog box is displayed.
