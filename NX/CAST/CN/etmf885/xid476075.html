Open the part and replace the machine tool

  1. Open mnf16_85_cylinder_setup_1.

  2. If the Open message box appears, click OK.

    NX loads the parts automatically when the Manufacturing application starts.

  3. On the Resource bar, click the Machine Tool Navigator tab.

    Notice that the kinematic model is called SIM07_5AX_MILL.

  4. In the Machine Tool view of the Operation Navigator, double-click the 5-AX_MILL_VERTICAL_B-HEAD_C-TABLE node.

  5. In the Example, 5-Ax Mill Vertical B-Head C-Table dialog box, click Retrieve Machine from Library .

  6. In the Library class Selection dialog box, select the Mill node, and click OK.

  7. In the Search Result dialog box, in the Matching Items group, in the libref column, double-click sim07_mill_5ax_sinumerik_mm.

    The machine tool is removed from the assembly and the Part Mounting dialog box is displayed.
