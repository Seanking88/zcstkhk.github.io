Allow selection of 2D IPW’s

  1. In the Program Order view of the Operation Navigator, select the FACING operation.

  2. On the Workpiece toolbar, click Select 2D IPW’s .

  3. Observe the display of the IPW.

    Before

    After

    You can select the 2D IPW geometry to measure distance, length, and so on. To ensure that the 2D IPW geometry is not accidentally selected when you select part geometry, turn the option off

  4. Continue to select each operation and observe the display of the 2D IPW.

  5. Select the ROUGH_BORE_ID operation.

  6. Use the Measure Distance command and measure the remaining stock on the inside bore diameter.

  7. Close the part without saving it.
