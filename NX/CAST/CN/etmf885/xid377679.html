Keep Assembly Constraints

What is it?

A new Keep Assembly Constraints option is available in the Part Mounting dialog box. When you replace a machine tool in a CAM setup, you can use this option to retain assembly constraints between the part, blank, or fixture and the machine tool model. Constraints between components of the machine tool assembly are not affected.

Typically you replace a machine with a modified version of the same machine, for example when you modify the kinematic model by editing an axis, changing travel limits, adding a new machine component, and so on. When you replace the machine tool with the same machine tool, all constraints are retained.

If you need to set up the part on a different machine, the constraints on the part will be missing geometry. You must edit the constraints that have missing geometry and select the corresponding geometry on the replacement machine.

Why should I use it?

You need not recreate assembly constraints that position the part on the machine.
