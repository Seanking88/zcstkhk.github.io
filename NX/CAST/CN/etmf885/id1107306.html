Change the insert size

  1. Click the Tool tab.

  2. In the Insert Size group, from the Measurement list, select Inscribed Circle (IC).

    The Inscribed Circle (I.C.) option allows you to enter the diameter of the circle that determines the overall tool size (1).

  3. In the Diameter box, type 0.250.

  4. Click the Tracking tab.

  5. In the Tracking Points group, from the P Number list, select P3.

  6. In the Preview group, click Display .

  7. Click OK to complete the tool definition.

    The Create Tool dialog box is displayed.

  8. Click Cancel to close the Create Tool dialog box.

  9. . In the Machine Tool view of the Operation Navigator, notice that the diamond insert is placed inside the FACE2 pocket.
