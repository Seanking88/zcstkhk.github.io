Activity: Shift the tool along its axis

Estimated time to complete: 7–12 minutes

When you profile a part with the side of the tool in a Contour Profile operation, you can shift the tool along its axis to reduce the size of scallops.

In this activity, you will:

  • Review an existing Contour Profile operation that does not shift the tool axis.

  • Shift the tool down by a constant value and observe the differences in the tool path.

  • Shift the tool down by different values at the top and bottom of the part and observe the differences in the tool path.
