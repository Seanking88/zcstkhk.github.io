Select the floor cut area

  1. In the Preview group, make sure the Preview check box is selected.

  2. In the Geometry group, click Specify Cut Area Floor .

    The Cut Area dialog box is displayed.

  3. Select the two floor faces as shown.

  4. Click OK.

    The Floor Wall dialog box is displayed.

  5. In the Geometry group, click Specify Wall Geometry .

    The Wall Geometry dialog box is displayed.

  6. Select the three walls as shown for wall Set 1.

  7. Click Add New Set .

  8. Select the three walls as shown for wall Set 2.

  9. Click OK to accept the wall geometry.

    The Floor Wall dialog box is displayed.

  10. In the Path Settings group, from the Cut Pattern list, select Follow Part.

  11. Click Cutting Parameters .

    The Cutting Parameters dialog box is displayed.

  12. Click the Containment tab.

  13. In the Blank group, from the Blank list, select Blank Geometry.

    The Blank Geometry setting lets you preview the blank material.

  14. Click OK.

    The Floor Wall dialog box is displayed.

  15. In the Path Settings group, in the Depth Per Cut box, type .500.

  16. Press Enter.

  17. In the Preview group, clear the Preview check box.

  18. Select the Preview check box to display the blank geometry, cut levels, and cut regions.

  19. In the Actions group, click Generate .

    Notice the tool path is contained by the floor of the part.
