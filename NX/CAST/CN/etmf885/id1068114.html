Create the operation

  1. Open mnf16_85_floor_wall_milling_w_ipw_1.

  2. On the Insert toolbar, click Create Operation .

  3. In the Create Operation dialog box, set Type to mill_planar.

  4. In the Operation Subtype group, click Floor and Wall with IPW .

  5. In the Location group, set the following:

    Program = PROGRAM

    Tool = MILL

    Geometry = WORKPIECE

    Method = MILL_ROUGH

  6. Click OK.

    The Floor Wall IPW dialog box is displayed.
