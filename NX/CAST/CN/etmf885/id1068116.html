Specify the parameters

  1. In the Geometry group, clear the Automatic Walls check box.

  2. Click Specify Wall Geometry .

    The Wall Geometry dialog box is displayed.

  3. Select the three faces as shown.

  4. Click OK.

    The Floor Wall IPW dialog box is displayed.

  5. In the Tool Axis group, from the Axis list, select +ZM Axis.

  6. In the Path Settings group, set the following:

    • Cut Region Containment = Walls

    • Cut Pattern = Profile.

    • Stepover = Constant

    • Maximum Distance = 25 %Tool

    • Depth Per Cut = 12

  7. Press Enter.

  8. In the Preview group, clear the Preview check box.

  9. Click Display .

  10. In the Path Settings group, in the Z-Depth Offset box, type 4.

    The Z-Depth Offset value extends the tool below the bottom of the wall for a full clean up.

  11. Click Display .

    The cut regions are displayed at each cut level.

  12. In the Path Settings group, in the Additional Passes box, type 2.

  13. Click Cutting Parameters .

    The Cutting Parameters dialog box is displayed.

  14. Click the Containment tab.

  15. In the Cut Regions group, set the following:

    • Exact Positioning =

    • Tool Overhang = 100 %Tool

  16. Click OK.

    The Floor Wall IPW dialog box is displayed.
