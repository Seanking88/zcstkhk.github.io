使用 Teamcenter Integration for NX 打开部件

Teamcenter Integration for NX is a data management tool used in conjunction with NX. When NX is used with Teamcenter Integration for NX, Teamcenter is running at the same time as a separate process. Since the two programs communicate, you can create, store, and access your data within a Teamcenter database. It combines the power of NX in modeling and generating data from geometric shapes, with the power of Teamcenter in storing and retrieving data in a controlled fashion.

To open an existing item such as a model, assembly, drawing, or simulation using Teamcenter Integration for NX:

Note

The item must exist in the Teamcenter database.

  1. Choose File→Open.

  2. In the Open Part File dialog box, select the top node from the Look In list.

  3. Double-click the node to display the folders.

  4. Double-click the folder to display the list of items.

  5. Filter the list of items, if necessary.

  6. Search for the item, if necessary.

  7. If you are opening an assembly, select the assembly load options.

  8. Select the item you want to open. The number, revision, and name of the selected item are displayed below the list of items. The item is graphically displayed in the Preview window and information about the item in provided in the Properties group.

  9. Click OK to open the item in NX.
