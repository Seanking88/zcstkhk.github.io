使用 Teamcenter Integration for NX 新建项

To create a new item such as a model, assembly, drawing, or simulation using Teamcenter Integration for NX:

  1. Choose File→New.

    Note

    You can immediately click OK to quickly create a new item. The last template used is automatically selected again.

  2. In the New dialog box, templates are provided to give you a start on creating the item. Depending upon the type of item you want to create click the Model, Drawing, Simulation, or Manufacturing tab.

  3. Select the template that is applicable to the type of item you want to create, such as Model, Shape Studio, Blank, and so on.

  4. Use the assigned values for Number and Revision, click Assign to display the next value in the sequence, or enter specific values. Enter the Item Name and select a Folder location for the new item.

    Note

    If you are creating a drawing, a Dataset Name box is provided instead of an Item Name box. The Number and Revision boxes and their values are grayed out for the item used as the basis for the drawing.

  5. Click Projects to assign the item to a project. The list of projects are those that are available in Teamcenter. You can select one or more projects.

  6. Click Attributes to display the attributes and their values for the item. You can change the value of an attribute by selecting it and entering the value in the Enter Attribute Value box.

  7. If a reference part is used, for example, if you are creating a drawing, the part is displayed in the Reference Part group of the dialog box. You can use Browse to verify it is the correct part or change the reference part.

  8. Click OK to create the item. The applicable application for the item is opened and the template item is displayed.

  9. To save the item to the Teamcenter database, choose File→Save. The first time the item is saved, the Name Parts dialog box is displayed to provide an opportunity for you to review and change the name, location, and attributes of the item before saving it to the Teamcenter database.

  10. Click OK to save the item to the Teamcenter database.
