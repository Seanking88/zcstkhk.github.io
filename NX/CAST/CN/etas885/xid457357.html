Add material behind each hole

You will add material behind each hole in order to recess the alignment tabs.

  1. On the Feature toolbar, click Extrude .

    For the next steps, it will be helpful to turn off the translucency.

  2. Choose Edit→Object Display.

  3. Select the solid body and click OK.

  4. In the Edit Object Display dialog box, in the Shaded Display group, move the Translucency slider to 0.

  5. Click OK.

  6. Select the two edges of the counterbore holes on the inside of the part.

    Zoom and rotate the part as needed to select the edges.

  7. In the Extrude dialog box, in the Limits group, enter the following values:

    • (Start) Distance = 0

    • (End) Distance = 1

  8. From the Boolean list, select Unite.

  9. Expand the Offset group and, from the Offset list, select Two-Sided.

  10. Set the offset Start to 0 and the End to 1.5.

  11. If the extrude vector is pointing into the part, in the Direction group, click Reverse Direction .

  12. Click OK.

    The extrusions should look like this illustration.

  13. Save the part to a directory where you have write access.
