Add reusable component preview window

What is it?

When you add a reusable component from the Reuse Library to an assembly, you can now view the component in a preview window. You can rotate, pan, or zoom the reusable component in the window before you add it to your model.

Where do I find it?

Resource bar

Reuse Library

Prerequisite

You must add a reusable component from the Reuse Library to your model

Location in dialog box

Add Reusable Component dialog box→Display Preview Window check box
