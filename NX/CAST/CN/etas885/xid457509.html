Change the display of the constraint names

  1. In the Assembly Navigator, expand the Constraints node.

  2. Expand the BACKHOE CONSTRAINTS folder.

  3. Rename the Align (ASM2_85_BACKHOE_LINK, ASM2_85_BACKHOE) constraint to backhoe.

    The newly-created user name is placed at the end of the constraint name.

  4. (Optional) Rename additional constraints.

  5. Right-click in the background of the Assembly Navigator and choose Properties.

  6. In the Assembly Navigator Properties dialog box on the General tab, from the Display Constraint Names list, select User and System.

  7. Click Apply.

    Note that the named constraint has the user name in front of the system name.

  8. (Optional) Repeat the last two steps, selecting System Only and User Replaces System from the Display Constraint Names list.

  9. From the Display Constraint Names list, select System and User (the default), and click OK.

  10. Close the part without saving.
