Add multiple components

You will use Repeat after Add with Positioning – Select Origin to add multiple components.

  1. In the Reuse Library Navigator, select Part Families and expand the Member Select group.

  2. Drag the Hex Fit Bolt to one of the holes.

  3. Release the mouse button to place the bolt.

  4. In the Placement group, set the following:

    • Use Inferred Constraints =

    • Positioning = Select Origin

    • Multiple Add = Repeat after Add

  5. Click OK.

  6. In the Point dialog box, from the Type list, select Arc/Ellipse/Sphere Center.

    This will be used to establish the origin point for the first bolt.

  7. Select the circular face as shown.

  8. Select three (3) additional holes in the same way, as shown.

    An additional bolt is created each time you select another hole.

  9. After the last hole is selected, in the Point dialog box, click Cancel.

    You have completed adding parts from Part Families in the Reuse Library to the assembly, using add multiple components.

  10. Close all parts without saving.
