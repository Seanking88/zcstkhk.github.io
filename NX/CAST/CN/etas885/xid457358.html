Create an assembly to display both housings

  1. Ensure that the Assemblies application is active.

  2. On the Assemblies toolbar, from the Components Drop-down list, click Create New Parent .

  3. In the Create New Parent dialog box, in the Name box, type ***_housing_assm, where *** represents your initials.

  4. Click OK.

  5. On the Assemblies toolbar, from the Components Drop-down list, click Add Component .

  6. In the Add Component dialog box, in the Part group, from the Loaded Parts list, select asm2_85_housing_rt.

  7. In the Placement group, from the Positioning list, select Absolute Origin.

  8. Click OK.
