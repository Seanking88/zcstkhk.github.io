Add a new library container to the Reuse Library

  1. Start the modeling application

  2. On the Reuse Library toolbar, click Reuse Library Management , or choose Tools→Reuse Library→Reuse Library Management.

  3. In the Reuse Library Management dialog box, click Add Library to create a new library.

  4. In your parts folder, double-click the Deformable Part folder and then click OK.

  5. Click Add Library .

  6. Double-click the Part Families folder and then click OK.

  7. Click OK.

    The Deformable Part and Part Families folders are added to the Reuse Library list. You have completed adding new library containers to the Reuse Library.
