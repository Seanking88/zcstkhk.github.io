Create a mirror body using Make Position Independent

  1. Open asm2_85_housing_assm_01 again.

  2. Make sure the Modeling and Assemblies applications are active.

  3. In the Assembly Navigator, double-click asm2_85_housing_rgt to make it the work part.

  4. On the Assemblies toolbar, click WAVE Geometry Linker .

  5. In the WAVE Geometry Linker dialog box, ensure that the Type is set to Mirror Body.

  6. In the Settings group, select the Make Position Independent check box .

  7. Select the solid body in asm2_85_housing2_rt.

  8. In the Mirror Plane group, click Mirror Plane and select the Y-Z plane of the Datum CSYS.

  9. If an Alerts message appears, read it and close it.

  10. In the WAVE Geometry Linker dialog box, click OK.

  11. Make asm2_85_housing_assm_01 the work part.

  12. On the Assemblies toolbar, click Move Component .

    The Move Component dialog box is displayed, with Select Components active.

  13. Select the mirror body (asm2_85_housing_rgt).

  14. In the Transform group, ensure that Motion is set to Dynamic.

  15. Click Specify Orientation.

  16. Select the YC handle and enter 15.

  17. Click OK.

    The selected component retains its new position and is independent from its source.

  18. Close the part without saving.
