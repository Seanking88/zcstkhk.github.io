Add multiple components using inferred constraints

  1. In the Reuse Library Navigator, select Part Families and expand the Member Select panel.

  2. Drag the Hex Fit Bolt to one of the holes.

  3. Release the mouse button to place the bolt.

  4. In the Add Reusable Component dialog box, in the Placement group, set the following:

    • Use Inferred Constraints =

    • Multiple Add = Pattern after Add

    • Positioning = Inferred Only

  5. Click OK.

  6. In the Create Component Array dialog box, make sure Array Definition is set to From Pattern Feature.

  7. Click OK.

    The array of bolts is created.

  8. Use Undo to remove the bolts.
