  2. Ensure that the Position column is displayed in the Assembly Navigator.

    Tip

    You can right-click the background of the Assembly Navigator and examine the Columns list.

  3. Right-click the Position column header and choose Report this Column.

  4. In the Report This Column message, click Yes.

    The HD3D Tools manager opens, with the Visual Reporting information for the report requested.

  5. In the Assembly Navigator, observe that the Report Group and Report Property columns are now present.

  6. Close all parts without saving.

  7. Choose File→Options→Assembly Load Options.

  8. From the Load list, select All Components and click OK.
