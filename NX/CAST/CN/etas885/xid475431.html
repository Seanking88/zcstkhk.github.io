Create the Head Tube part

  1. Open asm2_85_expression_bike_assm.

  2. Ensure that the reference set of the head tube component is Entire Part.

  3. Make asm2_85_expression_head_tube the Work Part.

  4. Choose Tools→Expression.

  5. Click Create Multiple Interpart Expressions .

  6. In the Create Multiple Interpart Expressions dialog box, in the Source Part group, select asm2_85_expression_seat_tube.prt.

  7. In the Naming Rule group, set:

    • Naming Rule = Add Prefix

    • Prefix String = IPE_

  8. In the Expressions group, from the Source Expressions list, select tube_id and tube_od.

  9. Click Add to Destination .

  10. Click OK.

    The interpart expressions are created.

  11. In the Expressions dialog box, click OK.
