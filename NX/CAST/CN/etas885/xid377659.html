Date visual reports

What is it?

When the report property is a date, in the Visual Reporting Definition dialog box, the Group By list for automatic or semi-automatic result grouping now contains time interval options. The options include:

  • Day

  • Week

  • Month

  • Year

Where do I find it?

Resource bar

HD3D Tools→Visual Reporting→Report group→Define New Report or Edit Report .

Location in dialog box

Visual Reporting Definition dialog box→Results group→Groups subgroup
