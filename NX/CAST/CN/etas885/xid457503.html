Create a constraint group

  1. Open asm2_85_backhoe_assembly.

  2. Ensure that the Assemblies application is active.

  3. In the Assembly Navigator, right-click the Constraints node and choose New Constraint Group.

  4. In the Constraint Group dialog box, in the Constraints from Selected Components group, click Select Components .

  5. Select the following components:

    • asm2_85_backhoe

    • asm2_85_backhoe_link

    • asm2_85_boom

    • asm2_85_boom_handle

    • asm2_85_knob

    Tip

    If you use the Assembly Navigator to select multiple components, you must press and hold the Ctrl key.

  6. Set Collected Constraints to Between Selected Components.

  7. Select the Remember Components check box.

  8. In the Settings group, in the Name box, enter backhoe constraints.

  9. Click OK.
