Create and locate the counterbore holes

In order to assemble the two halves, you will replace the mounting studs in the linked mirror part with a matching set of counterbore holes and add material to recess the alignment tabs.

  1. Choose Edit→Object Display.

  2. Select the solid body and click OK.

  3. In the Edit Object Display dialog box, in the Shaded Display group, move the Translucency slider to 40.

  4. Click OK.

    If the Translucency Performance Warning dialog box is displayed, click OK.

  5. Choose Insert→Curve from Curves→Project.

  6. Select the two curves shown below.

  7. In the Project Curve dialog box, in the Objects to Project To group, click Select Object .

  8. Select the face shown below.

  9. Click OK.
