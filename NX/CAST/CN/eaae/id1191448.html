Clearance Set overview

All data associated with a clearance analysis is maintained in a clearance set. Use the Clearance Set commands to:

  • Activate an existing set to analyze, modify, copy or delete.

  • Create a new set.

  • Copy an existing set, which you can then modify.

  • Open the Clearance Properties dialog box to modify the active set.

A clearance set contains the following information:

  • Objects to analyze

  • Clearance zones

  • Unit subassemblies — subassemblies that you set to be treated as single objects during analysis

  • Special object or pair inclusions and exclusions

  • Analysis results

You can have more than one clearance set in an assembly, but only one can be analyzed at a time.
