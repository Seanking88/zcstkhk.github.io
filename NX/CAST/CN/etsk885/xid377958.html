Sketch Preferences enhancement

What is it?

Constraint Symbol Size

Sets the initial default size of constraint symbols. After a sketch is created you must use the Edit→Style command to change the size of constraint symbols.

Use Solving Tolerance

By default this check box is cleared and sketches solve constraints with the tightest possible tolerance. You can assume that sketches solve with no tolerance.

Selecting this option lets you set the solving tolerance for the sketch. The tolerance specifies the largest possible gap or overlap of your sketch objects. Situations where you might choose to use this tolerance are:

  • When the model has vastly different scales for different geometries. For example, if some of the dimensions in kilometers while others are in millimeters, solving with tight tolerance most likely leads to conflicting constraints.

  • When sketch offset commands expect the input chain to be end-coincident within the solving tolerance. For example, if you select an external chain of curves to offset and if the chain has gaps or overlaps at the end points, you can set the sketch tolerance to be more than the greatest gap to ensure the offset command is successful.

Dynamic Sketch Display

When this option is selected, constraint and vertex symbols do not display if the associated geometry is very small. To see these sketch objects regardless of associated geometry size, clear this check box.

Display Constraint Symbols

Controls the initial display of geometric constraint symbols.

Name Prefixes

Adds a default name when you create a sketch object. The default name Vertex has been changed to Point.

Part Settings Colors

Sets the default sketch colors.

The Conflicting Constraints preference is now named Conflicting Objects.

The Unsolved Curves preference is new. Use this preference to set the color for geometry that is overconstrained or has conflicting constraints.
