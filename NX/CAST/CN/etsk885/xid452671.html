Add equal radius constraints

  1. In the Constraint group, click Equal Radius .

  2. In the Settings group, clear the Automatic Selection Progression check box.

  3. Select the three circles as shown.

    You can select multiple objects when the Select Object to Constrain selection step is active.

  4. Click the middle mouse button to advance to Select Object to Constrain to selection.

  5. Select the circle as shown.

All four circles have the same radius.
