Vertex display

What is it?

While sketching you can see the vertices of curves. Moving vertices is easier because there is an object that can be dragged.

A drag handle is displayed at the vertex location when you select an endpoint. This helps differentiate between dragging a curve and dragging a vertex of a curve.

If a vertex has an overconstraint status, the vertex and not the entire curve will be displayed in your default Unsolved Curves color. This helps to determine if the problem is with the vertex, the curve, or both.

Where do I find it?

Application

Modeling, Drafting, Shape Studio, Sheet Metal

Prerequisites

In the Sketch task environment, or while a sketch is active
