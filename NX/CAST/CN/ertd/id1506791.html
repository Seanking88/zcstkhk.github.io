设置库部件单位

示例库提供了英制和公制单位，下方列出的环境变量已为两个单位系统都预先定义。即两个版本的环境变量都已定义，但是英制系统对应的环境变量已被注释，要更改单位系统，将位于 Routing Units 下方适合的 UGII_ROUTING_KIT_UNITS 变量注释掉，然后保留另外一个。

  • # Routing Units (管线单位)
    UGII_ROUTING_KIT_UNITS=metric
    # UGII_ROUTING_KIT_UNITS=inch

其它特定于机械管线布置的环境变量

  • # Routing Kit directories (管线套件目录)
    UGII_ROUTE_MECH_KIT_DIR=${UGII_BASE_DIR}\ugroute_mech

  • # Application View File Locations (应用模块视图文件位置)
    UGII_ROUTING_APP_VIEW_DIR=${UGII_ROUTE_MECH_KIT_DIR}\appview\

  • # Application View Files (应用模块视图文件)
    UGII_ROUTE_MECH_APP_VIEW=
    ${UGII_ROUTE_MECH_KIT_DIR}\appview\→
    ugroute_mech_${UGII_ROUTING_KIT_UNITS}.xml

  • # Part Library View Locations (部件库视图位置)
    UGII_EPLIB_PART_LIB_VIEW_PATH=
    ${UGII_ROUTE_MECH_KIT_DIR}\appview\

  • # Part Table Locations (部件表位置)
    UGII_EPLIB_TABLE_PATH=
    ${UGII_ROUTE_MECH_KIT_DIR}\${UGII_ROUTING_KIT_UNITS}\tables\

  • # Library Part File Locations (库部件文件位置)
    UGII_ROUTE_MECH_PART_PATH=
    ${UGII_ROUTE_MECH_KIT_DIR}\${UGII_ROUTING_KIT_UNITS}\parts

要获取更多信息，参见在线帮助文档中的 CAD→Routing Systems→Routing→Configure Routing applications→Routing configuration files.

族成员保存目录

管线部件库中的各个部件都是一个部件族模板，当您从库中选择一个部件添加到管线装配中时，将生成该部件的族成员，并放在由下面的环境变量定义的默认位置中：

  • # Part Family Member Save Location (部件族成员保存位置)
    #UGII_ROUTING_MEMBER_SAVE_DIR

如果您没有定义这个环境变量，部件族成员将会存放在 NX 的启动目录中(如 D:\Program Files\Siemens\NX 9.0\UGII)。
