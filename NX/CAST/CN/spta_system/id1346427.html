Example: Assigning external superelements in the bulk data file

In this example, two instances of the same external superelement are represented in the system model. The following ASSIGN statements appears in the File Management section of the .dat file.

$*  NX Superelement ID: 4
ASSIGN INPUTT2='path\filename_0.op2' UNIT=31
$*  NX Superelement ID: 7
ASSIGN INPUTT2='path\filename_0.op2' UNIT=32

You can click Preview in the Solution dialog box at any time to confirm that all superelements in the model are correctly assigned.
