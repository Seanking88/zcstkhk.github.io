Using superelements in an assembly FEM

In Advanced Simulation, you can replace individual components in your assembly FEM with external superelements to define a system model.

To replace a component in your assembly FEM with an external superelement, you modify the component attributes to reference the binary results file (*_0.op2) from an NX Nastran SOL101 – Superelement or SOL103 – Superelement analysis. You can replace multiple instances of a component in the assembly FEM with the same superelement.

Using external superelements, you can reduce the number of DOFs (and consequently the number of equations) to a manageable number in an arbitrarily large model.
