System modeling with external superelements

Estimated time to complete this course: 37–56 minutes

This course provides an introduction to system modeling in Advanced Simulation using NX Nastran external superelements. In this course, you will learn:

  • The basic workflow common to all system models.

  • How to map external superelements to a system model assembly FEM.

  • How to control the display of superelements in your system model.

  • How system models are solved by NX Nastran.

  • How to post-process a system model.
