Scale deformations and hide element edges

By default, overlayed superelement deformations are displayed with 1:1 absolute scaling. Scale up the deformations to view relative displacements more clearly. To improve graphics performance, hide element edges. You can select multiple post views and apply the same settings to each post view simultaneously.

Press and hold the Shift key to select all overlaid post views. This enables you to apply the same settings to all post views in one operation.
