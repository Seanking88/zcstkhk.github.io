Activity: Modeling with external superelements

Estimated time to complete: 30–45 minutes

You will learn how to:

Obtain performance statistics for a standard finite element analysis.

Replace component FEMs with external superelement representations.

Control the display of superelements in an assembly FEM.

Solve a model that uses external superelements.

Compare the performance and accuracy of the system model to the base FE model.

Post-process results from the system model.

Launch the Modeling with external superelements activity.
