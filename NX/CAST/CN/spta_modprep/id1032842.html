准备修改理想化部件

要对理想化部件进行修改，必须先为要修改的几何体创建关联副本。

根据您的要求，可使用两种不同的技术来实现对理想化部件的修改：

  • 提升

  • WAVE 几何体链接器

尽管所有命令都允许您更改几何体以满足 CAE 分析的需求，但每种方法都分别有它们的优势及局限性。

提升

要把一个片体或实体从理想化部件中关联复制到装配中的更高等级，使用 提升 命令。

在完成提升体之后，您可以使用 CAE 几何体理想化命令以及建模应用模块命令来添加或修改特征。

提升体将与主模型历史保持关联，如果主模型发生更新，NX 也将更新理想化部件中的提升体。

优点：

  • 使用提升比使用 WAVE 几何体链接器更简单，也更直接，尤其是当您没有很多 NX 装配经验的时候。

缺点：

  • 因为您只能在体上使用提升，所以它并不能对您复制的事物进行精细的控制，例如，即使您只想要修改一个面，但是您还是需要提升整个体或片体。

  • 您无法控制提升体后的几何体更新。

  • 除非已提升体的主模型部件得以加载并且可见，否则已提升体也将不可见。

  • 您无法采用无历史记录模式对提升的体建模。

  • 无法重新提升或重新链接提升的体。

您随时可以将提升的体转换为 WAVE 链接以避免这些局限性。

WAVE 几何体链接器

要为所选几何体创建关联性链接副本，使用 WAVE 几何体链接器 命令。理想化部件中的 WAVE 链接几何体和主模型部件中的几何体之间存在直接链接。当主模型部件中的几何体更新时，理想化部件中的任何 WAVE 链接几何体也会更新。

优点：

  • 使用 WAVE 几何链接器命令，可以比使用提升命令更多地控制以关联性方式复制的几何体。虽然提升仅适用于体，但使用 WAVE 几何链接器命令，除了复制体以外，还可以复制诸如单个面或面集合等对象。

  • 使用 WAVE 链接几何体，可以控制何时或者是否接受几何体更新。

  • 使用 WAVE 链接，在以关联性方式复制几何体之后，可以在执行建模操作时引用链接数据，甚至是在未加载含定义几何体的部件时。

缺点：

  • 为了成功使用，WAVE 链接技术一般要求更多地了解 NX 中的装配。

  • 必须执行额外的步骤，以确保未在 FEM 文件中复制 WAVE 链接体。关于更多信息，参见在线帮助中的防止在创建 WAVE 链接后出现重复体。

  • 如果您在受管模式下工作并且大型装配中拥有大量 WAVE 链接，则更新所需的时间可能会较长。
