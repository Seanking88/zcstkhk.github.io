删除部分圆角

  1. 打开 syn3_delete_partial_blend。

  2. 点击同步建模工具条上的删除面 。

  3. 在类型列表中选择面 。

  4. 选择如图所示的面。

  5. 在删除面对话框中，设置如下参数：

    设置组：

    • 修复 =

    • 删除部分圆角 =

    预览将显示该圆角被删除，并且修剪相邻面。

  6. 在删除面对话框中，设置如下参数：

    设置组：

    • 修复 =

    • 删除部分圆角 =

    • 倒角 = 选定的圆角

    预览将显示圆角被删除，并且两个面被沿选定圆角的圆角填充。

  7. 在删除面对话框中，设置如下参数：

    设置组：

    • 修复 =

    • 删除部分圆角 =

    • 倒角 = 相邻圆角

    预览将显示圆角被删除，并且两个面被沿选定圆角的相邻圆角填充。

  8. 点击取消。
