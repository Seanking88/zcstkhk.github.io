移动面选项

  1. 打开 syn3_move_face_options。

  2. 在同步建模工具条上的修改面下拉菜单中，选择移动面 。

  3. 选择如图所示的七个面。

  4. 在移动面对话框中，设置如下参数：

    变换组：

    • 运动 = 距离

    • 指定矢量 = –XC

    • 距离 = 15

    设置组：

    • 移动行为 = 移动和修改

    • 溢出行为 = 自动

    • 阶梯面 = 无

    凸台面将被较小的竖直面修剪。

  5. 在移动面对话框中，设置如下参数：

    设置组：

    • 移动行为 = 移动和修改

    • 修复 =

    • 粘贴 =

  6. 点击确定。

    凸台和圆角将向下延伸到水平面上。

  7. 关闭所有部件。
