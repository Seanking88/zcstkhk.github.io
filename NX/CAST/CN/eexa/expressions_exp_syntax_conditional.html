条件表达式

预计完成这堂课需要：9–13 分钟

表达式可用来根据特定条件定义变量。这类表达式可通过使用 if-else 语句加以创建。

名称

公式

Lgth

12.5

Wdth

if ( Lgth > 10) ( 5)else ( 3 )

Wdth 表达式根据以下语句定义：

  • 如果 Lgth 大于 10，则 Wdth 等于 5。

  • 如果 Lgth 小于等于 10，则 Wdth 等于 3。

表达式还可使用 AND 或 OR 等布尔运算符。

名称

公式

Lgth

12.5

Wdth

if ( Lgth > 0 && Lgth < 10) ( 3)else ( 5 )

此时，Wdth 表达式根据以下语句定义：

  • 如果 Lgth 大于 0 并且小于 10，则 Wdth 的值为 3。

  • 否则，Wdth 的值为 5。

语句的语法和命令部分必须为小写字母。
