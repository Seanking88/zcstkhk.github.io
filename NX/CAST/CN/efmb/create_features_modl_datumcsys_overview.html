基准 CSYS

预计完成这堂课需要：14–19 分钟

使用基准 CSYS 可快速创建包含一组参考对象的坐标系。可以使用这些参考对象来关联地定义其他特征的位置和方向。

绝对坐标系原点处的现有基准 CSYS 包含在许多默认的部件模板中。

基准 CSYS 包含以下参考对象：

  • 坐标系

  • 原点

  • 三个基准平面

  • 三个基准轴

可以创建基准 CSYS：

  • 在相对于工作或绝对坐标系的固定位置

  • 与现有几何体相关联

  • 与现有基准 CSYS 相偏置

供 CSYS 中的参考对象使用

定义：

  • 草图与特征的放置面、约束及位置

  • 特征的矢量方向

  • 模型空间中的关键产品位置，并通过平移及旋转参数来控制它们

  • 约束，以在装配中放置部件
