Select the values for the family attributes

For this wiring harness you want to apply a protective sleeving that is made of green vinyl, has a thickness of 0.3048 mm, and a width of 19.0 mm.

  1. Right-click the Wrapped node and then choose Search Children.

    The Routing Reuse Search dialog box opens, allowing you to filter your search by a specific set of characteristics.

  2. In the Search Database group, under Update Search Criteria, select the COLOR row.

  3. From the COLOR list at the bottom of the dialog box, select Green.

  4. Select the MATERIAL row.

  5. From the MATERIAL list, select Vinyl.

  6. Select the THICKNESS row.

  7. From the THICKNESS list, select 0.3048.

  8. Select the WIDTH row.

  9. From the WIDTH list, select 19.0.

  10. 点击确定。

    Now there is only one matching member listed in the Member View box that meets the defining criteria, Wrap_0126_mm.

  11. Select Wrap_0126_mm, then click the middle mouse button.

    You are returned to the Overstock dialog box, which now displays the name of the overstock you chose:

    • Overstock: Wrap_0126_mm
