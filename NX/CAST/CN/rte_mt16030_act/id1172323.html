Create a subassembly for the wiring harness

The new subassembly will eventually hold all of the objects that comprise the wiring harness: WAVE linked ports from the various devices, connectors, clamps, routing paths, as well as stock or overstock.

To do this you will first create an empty component within the structure of the assembly, then bring the objects you need into it and associate them with the devices in the assembly as you bring them in. You then build the harness around these puts.

  1. Make sure the Assemblies application is started. (Start→Assemblies)

  2. On the Assemblies toolbar, from the Components Drop–down list, select Create New Component .

  3. In the New Component File dialog box, make sure:

    Units

    = Millimeters

    Templates

    = Routing Electrical

  4. For Native NX, in the New File Name group:

    1. In the Name box, type ***_drawer_harness_1, where *** represent your initials.

    2. In the Folder row, click , and then navigate to your student_home folder.

    3. 点击确定。

  5. For Teamcenter Integration, in the New File Name and Attributes group:

    1. In the Number box, type ***_drawer_harness_1, where *** represent your initials.

    2. In the Revision row, click Assign.

    3. 点击确定。

  6. Accept all of the defaults in the Create New Component dialog box.

    (You can create an empty component by not selecting any objects.)

  7. 点击确定。

  8. Open the Assembly Navigator to see the new harness component that has been inserted into your assembly.
