Complete the assignments

  1. Make the rte1_car_assy_3 the displayed part.

    (Window)

  2. Make the harness the work part.

  3. Use Place Part to add your new eyelet (or if you prefer, the rte1_cea_eyelet_2) to the harness subassembly in the same place and orientation as the old one. Use the PORT reference set and lock the engagement and rotation.

  4. In the Electrical Component Navigator, assign the new eyelet.

    (Auto Assign should work).

  5. In the Electrical Connection Navigator, select all of the wires, and then right-click one and choose Unroute.

  6. Heal a new spline path from the port of the wire clip to the new eyelet, using a start and an end Extension of 5.

  7. In the Electrical Connection Navigator, Auto Route all of the wires at the pin level.

  8. If you are working on your saved version of the assembly, save it.

  9. Close the assembly.
