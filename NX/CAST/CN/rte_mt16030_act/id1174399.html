Open the harness assembly

  1. Open rte1_formboard_harness.

  2. On the Formboard Tools toolbar, from the Formboard Create/Update Drop–down list, select Create Formboard Drawing .

  3. In the New Formboard Drawing dialog box, make sure:

    Units

    =

    Millimeters

  4. From the Templates list, select:

    Name

    /

    Relationship

    (Native)

    AO - Size

    /

    Reference Existing Part

    (TCI)

    AO - Size

    /

    specification

  5. For Native NX, in the New File Name group:

    1. In the Name box, type ***_formboard_harness_dwg, where *** represents your initials.

    2. Make sure the path in the Folder box points to student_home.

    3. 点击确定。

  6. For the Teamcenter Integration environment, in the New File Name and Attributes group:

    1. In the Dataset Name box, type ***_formboard_harness-A-dwg, where *** represents your initials.

    2. 点击确定。
