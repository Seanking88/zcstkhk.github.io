Specify the bend radius

You want to apply bends with a uniform radius to individual segments of the harness. You can do this by specifying a bend radius preference.

  1. Continue working with your part.

  2. Start the Routing Electrical application.

  3. Choose Preferences→Routing.

  4. Make sure the Paths tab is open.

  5. Click Set Corner.

    You need to bend the branches with a radius of 40 mm.

  6. In the Corner Preference dialog box, set the following parameters to:

    Type

    =

    Bend

    Method

    =

    Radius

    Radius

    =

    40.00

  7. Click OK two times.
