Annotate harness branches

Now you need to place a Path Length label between two specific points on the harness.

  1. From the Type list, select Points on Curves.

  2. In the Select First End Point group, select End Point from the drop-down menu.

  3. Select the left most point shown in the illustration above.

  4. In the Select Second End Point group, select End Point from the drop-down menu.

  5. Select the right most point.

  6. Drag the label to a suitable location.

  7. Click to place it.

    The combined length of the selected segments is displayed.

  8. Click Close.

  9. Save and close your formboard drawing.
