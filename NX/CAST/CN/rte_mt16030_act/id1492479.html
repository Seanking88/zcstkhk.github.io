Start a new part

  1. On the Standard toolbar, click New .

  2. On the Model tab, make sure:

    Units

    = Millimeters

    Templates

    = Routing Electrical

  3. For Native NX, in the New File Name group, accept the default part name and folder location.

    Note

    In a Teamcenter Integration environment, click Assign in the Number row.

  4. 点击确定。
