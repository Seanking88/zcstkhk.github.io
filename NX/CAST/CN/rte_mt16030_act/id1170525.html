    The Display Color box updates to show red.

  2. Click Finish.

  3. Close the Information window that appears. It simply advises you that there are no physical parts that NX can route to.

    The logical data for the new wire has been added.

  4. Scroll the navigator all the way to the right to see the color for the new conductor.

    Tip

    If you need to make a correction to C5 or any other connection in the list, right-click the row and choose Connection→Edit. In the connection wizard, navigate to the page where you need to make the correction, enter the new data, and then click Finish.
