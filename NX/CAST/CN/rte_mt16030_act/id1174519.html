Review the customer defaults

To complete all of the formboard activities, you may need to make some entries into your customer defaults, and then restart NX.

  1. Choose File→Utilities→Customer Defaults.

  2. In the Customer Defaults indent list, select:

    Routing
     →Formboard

  3. Click the Annotation tab.

    The values in the Attribute List correspond to the data imported into the Electrical Connection Navigator and determine which attribute columns appear in the cut sheet table.

  4. If your Attribute List contains all of the following data, you may proceed to the next page.

    •    Attribute List

      		 
      unique_id
      from_conn
      from_port
      to_conn
      to_port
      cut_length

    If it does not, continue to the next step.

  5. Below unique_id, add the following data:

    • from_conn
      from_port
      to_conn
      to_port
      cut_length

    Tip

    You can copy each line of text from this activity window and then paste it into Attribute List box. Make sure you press Enter at the end of a line to start a new line of text.

    This edit becomes effective when you restart NX, and any cut sheet table you create will automatically contain a column for each of the attributes in this list.

  6. 点击确定。

  7. Terminate your current session of NX, and then start a new session.
