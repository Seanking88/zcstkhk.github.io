Define the ‘To Component’

  1. From the menu lists, select the following:

    Part Name

    =

    rou_6pin

    Component Type

    =

    Connector

    To Device

    =

    DEVICE_2

    To Conn

    =

    P2

  2. In the To Pin box, type 5.

  3. There are no Intermediate Components, so click Next twice to advance to Stock.
