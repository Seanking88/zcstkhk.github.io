Optional: Qualify the 16 pin connector

  1. Open rte1_cea_16pin_conn_optional.

  2. Save it as ***_cea_16pin_conn.

  3. Create a fitting port centered on the bottom face of the connector as shown below; define a rotation vector that points toward the tongue (1) of the connector.

  4. Create a multi port centered on the top of the connector as shown below.

    • Assign an Extension of 10.

    • Assign a Cutback Length of 15.

    • Generate a Sequence using upper case letters, A through S.

    • Optional: Assign all of the terminals at the base of the counter bored holes (1) as shown below. The vectors should point to the top of the connector.

  5. Create a PORT reference set.

    1. Choose Format→Reference Sets.

    2. In the Reference Sets dialog box, click Add New Reference Set .

    3. In the Reference Set Name box, type PORT.

    4. Select the connector, both ports and all of the terminals.

    5. Press Enter then click Close.

  6. Save the part.

  7. Choose Window, and select the car assembly.

  8. Make the harness the Work Part.

  9. Place your connector in the harness subassembly.
