Create the built-in path

  1. On the Routing Electrical toolbar, from the Tools Drop–down list, select Qualify Part .

  2. Make sure that Route Part Type is set to Fitting.

  3. From the list to the right of the Route Part Type options, make sure Fitting is selected.

  4. From the Routing Object list, right-click the Built-in Paths node and choose New.

  5. Select the three curves that comprise the sketch.
