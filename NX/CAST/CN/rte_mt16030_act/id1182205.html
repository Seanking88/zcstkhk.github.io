Review the definition of the port on the eyelet

  1. Remove the rte1_cea_eyelet from the harness subassembly.

  2. Using the Standard option, delete the routing segment that goes to the eyelet.

  3. Make the rte1_cea_eyelet the displayed part.

    (Window →More)

  4. Using Qualify Part, examine the characteristics of the two fitting ports.

    Notice that there is no terminal defined for either one.

    The next action will direct you to delete the fitting port and replace it with a multi port.

  5. In the Qualify Part dialog box, right-click the fitting port on the barrel end of the eyelet and choose Delete.

  6. Right-click the Multi node and choose New.

  7. From the Filter list, select Point.

  8. Use to select the barrel end farthest from the eyelet.

  9. Enter the following data:

    Engagement

    =

    8.60

    Extension

    =

    5.00

    Cutback Length

    =

    0.00

  10. Click the middle mouse button to advance to Align Vector.

  11. Make sure the alignment vector points away from the part. If it does not, then click Cycle Direction.

  12. Click OK until the Assign Terminals dialog box appears.

  13. In the Terminal Name box, type 1, and then 回车。

    The number appears in the list box above Terminal Name.

  14. Select 1 from the list box, and then click Place Terminals.

  15. From the Point Method list, use Arc/Ellipse/Sphere Center to select the end of the barrel nearest the eyelet.

  16. Make sure the terminal points towards the multi port symbol. If it does not, click Cycle Direction.

  17. Click OK two times.

  18. Fully expand the Multi node to make sure the terminal is assigned, and then click OK to save the multi port and close the Qualify Part dialog box.
