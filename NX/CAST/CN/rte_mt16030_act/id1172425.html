Add some slack to the segment

  1. Double-click the segment.

  2. In the Spline Shape group, from the Shape Type list, select Slack.

  3. Under Slack Settings, from the Mode list, select Additional Length.

    Note the Current Length of the segment. You will add slack to this length.

  4. In the Value box, type 10 and 回车。

    Try a little less slack.

  5. Type 3 and then 回车。

  6. 点击确定。
