(Optional) Rename the harness subassembly

If you would like to work on your own saved versions of the harness, do the following steps. Otherwise proceed to the next activity and use the parts supplied.

  1. Make rte1_harness_sub_assy the Work Part.

  2. Choose File→Save As.

  3. For Native NX:

    1. In the Save in: box, navigate to student_home.

    2. In the File name: box, type ***_harness_sub_assy, where *** represent your initials.

    3. 点击确定。

    4. In the next Save As dialog box, type a new name for the top level assembly, ***_car_wiring_assy.

    5. 点击确定。

    6. Click Yes to continue the Save As operation.

    7. Dismiss the Information window.

  4. In a Teamcenter Integration environment:

    1. From the Action list, select Save As New Item.

    2. In the Number box, type ***_harness_sub_assy, where *** represent your initials.

    3. In the Revision row, click Assign.

    4. 点击确定。

    5. In the next Save As dialog box, type a new name for the top level assembly, ***_car_wiring_assy, and in the Revision row, click Assign.

    6. 点击确定。

    7. Click Yes to continue the Save As operation.

    8. Dismiss the Information window.

  5. Click Yes to continue the Save As procedure.

  6. Click OK to acknowledge the Save As Report.

  7. Dismiss the Information window.

  8. Proceed to the next project activity. Otherwise close your parts.
