Create the formboard drawing

  1. Use the Create Formboard Drawing command to generate a new formboard drawing in your home folder called ***_car_formboard_dwg.prt.

  2. From the Templates tab, select A2–Size.

  3. 点击确定。

  4. Define the Main Run by the Thickest cross section.

  5. Define the Branch Angles by Maximum Angles.

  6. 点击确定。

    Notice that there is only one wire clip present in the formboard; the one you brought in to the wiring harness subassembly. (The other wire clip is in the top level assembly.)

    Note

    The illustrations in this activity may not exactly match your formboard layout.
