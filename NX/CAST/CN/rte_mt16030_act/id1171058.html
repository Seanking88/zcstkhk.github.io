Create a fitting port on this connector

This port that will mate with the multi port on the circular connector.

  1. From the Routing Object list, right-click the Fitting node and choose New.

    You will want this port to originate at the upper end point of the sketch line.

  2. Make sure that the Filter is set to Point.

  3. On your Selection bar, first make sure is on, and then turn on .

  4. Turn off all the other Snap Points.

  5. Select the end point of the sketch curve.

  6. Click the middle mouse button to advance to the Align Vector step.

    The port vector arrow must point away from the center of the backshell.

    If it does not, click Cycle Direction.

  7. Click OK twice.
