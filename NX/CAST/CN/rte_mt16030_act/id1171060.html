Create the fixture port on the backshell

  1. Right-click the Fixture node and choose New to open the Fixture Port dialog box.

  2. Make sure the Filter is still set to Point.

  3. Use to select the lower end point of the sketch curve.

  4. Click the middle mouse button to advance to the Align Vector step. Make sure that the port vector arrow points away from the center of the part. If it does not, click Cycle Direction.

    You do not want the spline path to bend as it enters the side of the backshell.

  5. In the Extension box, type 13.

  6. 点击确定。

  7. Click OK in the Qualify Part dialog box.

  8. Click Clip Work Section .

  9. Close the part without saving.
