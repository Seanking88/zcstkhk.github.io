Activity: Create routing segments for the wiring harness

In this activity, you will route paths for the wire harness in the assembly. To do this, you will:

  • Route wiring paths between multi ports on the connectors.

  • Route paths through Stock offset Points.

  • Subdivide a segment in order to branch another segment from it.

  • Re-route an existing path through a clamp.

  • Add slack to a segment.

Launch the Create routing segments for the wiring harness activity.
