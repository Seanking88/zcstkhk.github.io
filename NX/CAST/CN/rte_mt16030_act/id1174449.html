Rotate the 24–pin connector

  1. Zoom in on the 24–pin connector.

  2. On the Formboard Tools toolbar, click Rotate Component .

  3. Select the connector.

  4. In the Axis group, make sure the Axis Type is set to Path Locations.

  5. In the Angle box, type 90.

  6. Click Apply.

    The connector is rotated so that the keying tab is to the right.

  7. Select the connector again.

  8. In the Angle box, type 180, and then 点击确定。

    The connector is revolved 180 degrees about its path so that the keyway lies in the opposite direction.

  9. Save but do not close your part. You will use it in the next activity.
