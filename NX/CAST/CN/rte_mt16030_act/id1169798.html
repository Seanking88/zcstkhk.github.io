Import component and connection data into your part

First verify the format you will be using.

  1. On the Resource bar, click the Component Navigator tab.

  2. Pin the navigator open.

  3. Inside the Electrical Component Navigator, right-click and choose Properties.

  4. In the Component List Properties dialog box, make sure:

    Display Format

    = Simple

    Import Format

    = Example

    Export Format

    = Example

    Note

    If you need to change the display format, click Edit Display Format, and from the Define Format list, select a format name and then 点击确定。

    The Example import format includes the fields:

    • part_name

    • unique_id

    • connector_id

    • signal

    • description

    The corresponding entries in the ugroute_elec.cfg file look like: 

    Component List File Format = Example
Type = comma_delimited
part_name, string
unique_id, string
connector_id, string
signal, string
description, string

  5. In the Component List Properties dialog box, 点击确定。

  6. In the Electrical Component Navigator, right-click and choose Import→Merge.

  7. If necessary, set Files of type: to:

    Bookmark Files and PLM XML (*.plmxml)

  8. Double-click rte1_logical.plmxml.

    The component data is imported into your part and appears in the navigator.

    To see the objects in the harness, expand the HARNESS1 node.
