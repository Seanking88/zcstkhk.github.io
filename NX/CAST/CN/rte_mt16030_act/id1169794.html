Activity: Import electrical data into a part

In this activity, you will:

  • Set up the requirements (properties) for a connection list, then import it into a part file.

  • Edit the display of the Electrical Connection Navigator by adding a column for color information.

  • Edit a connection list file by adding a new wire to it.

  • Export electrical connection data.

  • Update and append connection data.

Launch the Import electrical data into a part activity.
