Create an RCP at the clamp fixture port

  1. Right-click the path going through the clamp and choose Subdivide Segment.

  2. In the Type group, make sure At Point is selected.

  3. In the Subdivision Point group, from the Location list, select Through Point.

  4. Make sure Specify Point is highlighted.

  5. Select the midpoint of the line on which the port lays. You may need to use the QuickPick dialog box to select it.

  6. 点击确定。
