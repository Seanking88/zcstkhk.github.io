Construct a spline path

  1. On the Routing Electrical toolbar, click Spline Path .

  2. In the Path Points group, from the Method list, make sure Specify Points is selected.

  3. Use to select the multi port on the connector, then the port on the clamp.

  4. If the Interpart Copy message window appears, first clear the Don’t display this message again check box, then 点击确定。

    For your next point selection, use the Stock Offset Surface method.

  5. From the Method list, select Stock Offset Surface.

    Notice that now the only Snap Point option available on the Selection bar is .

  6. Somewhere near the hole in the baseplate, select a point.

    The face is WAVE–linked from the baseplate into the Work Part.

  7. From the Method list, select Specify Points.

  8. Use to select the port on the second clamp.

  9. Click OK to complete the segment.
