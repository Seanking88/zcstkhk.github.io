Label the lengths of the harness

Place a length label on a harness segment.

  1. On the Formboard Tools toolbar, click Path Length Annotation .

  2. Make sure the Type is set to Routing Path Length.

  3. Start by selecting the segment shown below.

  4. Drag the label origin to a good location on the drawing sheet, then click to place it.

    The total length of the segment appears in the label.
