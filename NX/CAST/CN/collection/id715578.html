Java support for Web browsers

Navigation and search tools are provided to browse through the textual information of the NX documentation. In order to properly use these tools, it is necessary to have version 1.6.0 or higher of the Java Virtual Machine (JVM) installed locally on your machine. The JVM is available in the Java Runtime Environment (JRE) 6.0. If you need to install the JRE, or an equivalent Java environment, visit the Java download site at www.java.sun.com.
