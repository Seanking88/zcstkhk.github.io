The Advanced visualization tools courses

These courses will familiarize you with the visualization capabilities of NX Render, which is a subset of Shape Studio. The terms NX Render and Visualize Shape are the same and will be used throughout these courses.

The following courses are available. Select a link to go directly to the course.

  • Materials and textures

  • Lighting

  • Visual effects

  • Showroom environments

  • High quality image animation

Audience

These courses are designed for users who are familiar with NX modeling terms and techniques and have a need to learn the industrial design tools within the NX Render software.

Prerequisites

If you are not experienced in NX modeling, you should complete the Feature modeling courses.

If you are unfamiliar with basic NX terms and functionality, you can find this information in the Basic concepts in NX courses.
