运行比较

  1. 右击管道通路导航器背景并选择比较。

  2. 在比较管道通路对话框中，选中报告差异 check box.

    (文件浏览器中应该指向 rtg1_facility_1。)

  3. 点击确定。

  4. 在管道通路导航器中，查看比较状态列。

    将显示 RUN_4中的差异。

  5. 展开 RUN_4。

    注意到 VALVE_4_OUT 显示为丢失，这是因为在3D 模型中，这个组件的引用 ID 为 VALVE_4_OUT，但是在2D 原理图中，它的引用 ID 为 VALVE_A_OUT。VALVE_A_OUT 在3D 模型中并未被引用，所以被认为是多余的组件。
