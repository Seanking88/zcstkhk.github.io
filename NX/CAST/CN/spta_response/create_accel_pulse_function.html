创建加速度脉冲函数

使用响应仿真的函数工具来创建一个1 mm/sec^2 的加速度脉冲函数，以在您的动态激励中使用。

注释

为了方便，在提供的 response_functions.afu 文件中已包含了脉冲函数，如果您无法创建脉冲函数，可以跳过这个步骤。

 响应仿真的函数工具(响应仿真工具条)

注释

这个实用工具需要 Java 运行时环境(Java Runtime Environment，JRE)1.7，您可以从 www.oracle.com 中找到相关下载。

如果您在打开该工具时遇到 Java 内存错误，在您的 eugii_env.dat 文件中设置这个环境变量：UGII_JVM_OPTIONS=-Xmx64M。然后重启 NX。

  • 瞬态

  • 脉冲

     

  • 半正弦

  • 幅值：1.0(默认值)

  • 脉冲宽度：0.02

  • 循环数

    1

  • 间隔数

    128

  • 纵坐标类型

    加速度

  • 单位类型

    mm/sec^2

  • 记录名

    half_sine_pulse_1

  • 浏览

     

  • 浏览至您有写入权限的目录，并键入文件名称以保存该函数。

  • 保存

     

    该函数保存在您指定的 AFU 文件中(AFU 文件是表哥函数的存储文件)。
