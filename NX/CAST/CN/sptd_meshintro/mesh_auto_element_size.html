自动设置单元大小

在网格划分对话框中，可以使用自动单元大小 选项让软件根据所选的几何体计算单元的估计长度。软件根据一系列不同的标准和因子进行自动单元大小计算。

  • 对于 2D 网格，软件进行一系列的计算以确定估计尺寸。

  • 对于 3D 网格，软件先进行 2D 计算，然后进行下一步估算，以确定体单元的估计尺寸。

(左侧) 3D 单元大小选项；(右侧) 2D 单元大小选项

自动单元大小所计算的单元大小是估计值，基于当前选定的几何体的某些特征。您始终都应认真地评估模型，并在确定单元大小时进行正确的工程判断(无论是接受软件的估计值还是指定其他大小)。此估算应该考虑模型几何体的各自特征，比如最小壁厚，以及您的分析需求。
