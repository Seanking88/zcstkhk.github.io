Three Bend Corner enhancement

What is it?

You can now create Three Bend Corner features that have the following corner reliefs:

  • U-shaped

  • V-shaped

Where do I find it?

Application

NX Sheet Metal

Toolbar

NX Sheet Metal→Corner Drop–down list→Three Bend Corner

Menu

Insert→Corner→Three Bend Corner

Location in dialog box

Corner Properties group→Treatment list→U Cutout or V Cutout
