Convert to Sheet Metal Wizard

What is it?

Use this wizard to rip edges, clean up inconsistencies, and convert a non-NX Sheet Metal part to a valid NX Sheet Metal part. The Convert to Sheet Metal Wizard streamlines the workflow for converting a model generated by other CAD systems to a workable NX Sheet Metal model.

Edge Rip step

Use this step to rip edges before converting a solid body to an NX Sheet Metal body.

The options for this step are the same as the options in the Edge Rip dialog box.

Cleanup Utility step

Use this step to create a new body that meets the requirements of the Convert to Sheet Metal command. You must select a base face to anchor the part during cleanup. You can also define the thickness, sliver tolerance, and specify if you want to show or hide the original body.

The options for this step are the same as the options in the Cleanup Utility dialog box.

Convert to Sheet Metal step

Use this step to specify the bend relief parameters and select the base face. If you do not select a base face, NX uses the base face selected during the cleanup task.

The options for this step are the same as the options in the Convert to Sheet Metal dialog box.

Note

  • The steps in the Convert to Sheet Metal Wizard are optional.

  • While the wizard guides you through the workflow, the separate dialog boxes for Edge Rip, Cleanup Utility, and Convert to Sheet Metal commands are also available.
