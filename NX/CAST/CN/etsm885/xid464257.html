Optimize Face in NX Sheet Metal

What is it?

The Optimize Face command is now available in the NX Sheet Metal application. Use this command to optimize faces by simplifying the surface type, merging, improving the edge accuracy, and recognizing blends.

Where do I find it?

Application

NX Sheet Metal

Toolbar

NX Sheet Metal→Convert Drop–down list→Optimize Face

Menu

Insert→Convert→Optimize Face
