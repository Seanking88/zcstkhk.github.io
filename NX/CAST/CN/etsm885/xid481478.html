Create a second array with the same pattern

You will use the Pattern Feature command again to create another array of five louvers 575 mm below the first pattern.

  1. On the Standard toolbar, from the Repeat Command Drop-down list, select Pattern Feature .

  2. Select the louver feature and the linear pattern, as shown.

    You can ignore any alerts that come up.

  3. In the Pattern Definition group, make sure the Layout list is set to Linear, and press the middle mouse button to advance to the Vector step.

  4. Select the same edge you selected before and double-click the direction arrow so that it points down.

  5. In the Pattern Definition group, set the following parameters for Direction 1:

    • Spacing = Count and Pitch

    • Count = 2

    • Pitch Distance = 575

  6. Click OK.

  7. Close the part now, or close all parts when you exit the session.
