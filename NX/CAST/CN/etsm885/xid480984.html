Unbend a bend region, add a bead, and rebend

  1. Open nsm2_85_rebend_bead.

  2. On the NX Sheet Metal toolbar, from the Form Drop-down list, select Unbend .

  3. Select the top face as the stationary face as shown.

  4. Select the bend region as shown.

    Notice the preview.

  5. On the NX Sheet Metal toolbar, from the Punch Drop-down list, select Bead .

    Tip

    If the command dialog box contains an <OK> button, you can select the next command without clicking OK. NX automatically completes the current command.

  6. In the Bead dialog box, in the Section group, click Sketch Section .

  7. Sketch a single curve in the middle of the face as shown.

    You can constrain and dimension the curve, but it is not necessary for this activity.

  8. On the Sketch toolbar, click Finish Sketch .

  9. In the Bead Properties group, set the bead parameters:

    • Cross Section = Circular

    • Depth = 2

    • Radius = 6

    • End Condition = Formed

  10. Click OK.

    The sketch is hidden because it is internal to the bead feature.

  11. On the NX Sheet Metal toolbar, from the Form Drop-down list, click Rebend .

  12. Select the bend region as shown.

  13. Click OK.
