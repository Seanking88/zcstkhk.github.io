Create a linear pattern

You will create a linear pattern feature of five louvers 25 mm apart.

  1. Choose Insert→Associative Copy→Pattern Feature.

  2. Select the louver.

    You can ignore any alerts that appear.

  3. In the Pattern Definition group, from the Layout list, select Linear. Press the middle mouse button to advance to the Vector step.

  4. Select the edge shown.

  5. Double-click the direction arrow so that the direction of the pattern is down.

  6. In the Pattern Definition group, set the following parameters for Direction 1:

    • Spacing = Count and Pitch

    • Count = 5

    • Pitch Distance = 25

  7. Click OK.
