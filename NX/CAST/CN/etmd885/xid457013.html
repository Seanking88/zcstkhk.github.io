Specify Part Module Outputs

  1. In the Part Navigator, right-click Part Module “Trigger Ribs” and choose Specify Part Module Outputs.

  2. On the Selection bar, from the Face Rule list, select Body Faces.

  3. Select the body as shown.

  4. In the References group, click Add New Set .

  5. Select the body as shown.

  6. In the Settings group, make sure the Deactivate after Specifying Outputs check box is selected and click OK.

    Two features are in the Outputs collector of the Part Module.
