Specify the twist

  1. In the Twist Angle Definition group, from the Law Type list, select Cubic Along Spine.

  2. Expand the Values Along Spine and List groups.

  3. With Specify New Location active, select the left point at the left end of the lower edge of the surface. Make sure you select the point.

  4. Make sure the Position on Spine is set to 0, and in the Point 1 box, type 90.

  5. With Specify New Location active, select the lower edge of the surface near the center.

  6. In the Position on Spine box, type 50, and in the Point 2 box, type 0.

  7. With Specify New Location active, select the right end point of the lower edge of the surface. Make sure to select the point.

  8. Make sure the Position on Spine is set to 100, and in the Point 3 box, type 90.

  9. Click Show Result .
