Edit the size and location of the tool body

  1. Double-click Extrude (10).

  2. Select the 0 dimension, type 25 in the entry box, and press Enter.

  3. Select the 100 dimension, type 150 in the entry box, and press Enter.

  4. Click OK.

  5. Examine the effect on the underside of the part.

  6. Close the part.
