Create another timestamped feature group

  1. Open ***_linkage.

  2. Choose Format→Group→Feature Group.

  3. In the Feature Group Name box, type tool_body_left.

  4. Select the Add Dependencies check box.

  5. In the Features in Part list, double-click on SKETCH_002:Sketch(12).

  6. Verify that the Embed Feature Group Members check box is selected.

    Notice that in the Output list you can now choose between Timestamped and Floating feature groups.

  7. Leave the Output list set to Timestamped Feature Group.

  8. Click OK to create the feature group.

  9. In the Part Navigator, expand the tool_body_left feature group.

    The feature group has a timestamp of (48) and the members of the group have retained their original timestamps; notice that the last feature in the group has a timestamp of (47).
