Create the first feature group

  1. Open des18_85_linkage.

  2. Choose Format→Group→Feature Group.

  3. In the Feature Group Name box, type tool_body_right.

  4. Select the Add Dependencies check box.

  5. In the Features in Part list, double-click SKETCH_001:Sketch(9).

  6. Make sure the Embed Feature Group Members check box is selected.

    At the bottom of the dialog box, notice that the Output is set to Timestamped Feature Group.

  7. Click OK.

  8. In the Part Navigator expand the feature group.

    Notice that the feature group has a timestamp of (58) and the members of the group have retained their original timestamps.

  9. Choose File→Save As and save a copy as ***_linkage (where *** is your initials) to a write accessible location.
