Define the first set of regions to remove

  1. In the Region group, select the Define Regions check box.

  2. Verify that Select Region is selected.

  3. Select Remove.

    You will select two sheet body regions to remove.

  4. In the graphics window, first select the sheet body as shown.

  5. Then select the next sheet body as shown.

  6. Click Apply.

    The solids have been united and the selected regions have been removed.
