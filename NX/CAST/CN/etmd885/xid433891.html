Define more regions to remove

The Unite dialog box should still be open; in the Target group, Select Body should be selected.

  1. Select the solid body as shown.

  2. On the Selection bar, from the Body Rule list, select Feature Bodies.

  3. For the tool body, select the extrude feature as shown.

  4. In the Region group, select the Define Regions check box.

  5. Verify that Select Region and Remove are selected.

  6. In the graphics window, select each of the overhanging sheet bodies as shown.

  7. Select the sheet bodies overhanging the hole in the middle of the part.

  8. Click the middle mouse button to complete the unite.

  9. Close the part without saving.
