On your own

Create your own feature in the Linked Part Module using what you just learned and the scenario in the following steps.

  1. Make the Linked Part Module “Trigger Design” the work part.

  2. Use the Linked Datum Coordinate System for a sketch plane to create a sketch and a boss like the one shown.

    Tip

    You must activate the Part Module to create objects in the Work collector.

  3. Add the new faces to the Outputs.

    Tip

    Make sure you select all three faces of the boss.

  4. Check the main part file.
