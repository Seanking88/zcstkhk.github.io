On your own

In your role as the principal designer you need to convert another internal Part Module to an external Linked Part Module for a secondary designer. In your role as a secondary designer you need to create a blend feature in the Linked Part Module. Finish these two tasks using what you just learned and the scenario in the following steps.

  1. Convert Part Module “Trigger Ribs” to a Linked Part Module.

  2. In the Part Navigator notice the changes:

    • The feature name is now Linked Part Module.

    • The Work collector is gone. It moved to the Linked Part Module file.

    • The Outputs collector has two WAVE Linked Faces.

  3. Acting as the secondary designer, open the Linked Part Module file and add blends of an appropriate size to the rib edges shown.

    If you add the blends to the Extracted Faces, the blends will not appear in the main part file.

  4. Deactivate the Linked Part Module.

  5. On the Standard toolbar click, Save .

  6. Make the main part file the work part.
