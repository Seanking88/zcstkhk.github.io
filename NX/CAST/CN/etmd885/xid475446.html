Merge Linked Part Modules

After the secondary designers work is complete, in your role as the principal designer, you need to merge the external Linked Part Module files back into the main part file to complete your design.

  1. In the Part Navigator, right-click Linked Part Module “Heat Sink” and choose Expand All.

    You can see that the Outputs collector has a WAVE link Linked Face (13).

  2. Right-click Linked Part Module “Heat Sink” and choose Merge Part Module.

    All the WAVE linked objects are brought back into the main part file. There is no more use for the Linked Part Module file. In the Part Navigator Notice that:

    • The new feature name is Part Module.

    • The Work collector is returned from Linked Part Module file.

    • The Outputs collector no longer has a WAVE linked face. It is converted back to an Extracted Face.

  3. In the Part Navigator, right-click Linked Part Module “Trigger Ribs” and choose Merge Part Module.

    All the Part Module information exists only in the main part file. The two Linked Part Module files are no longer needed and you can delete them. In your design process, if you need the secondary designers to work on their design elements again, you would follow the same process.

    1. Convert the Part Module into a Linked Part Module for their use.

    2. After the secondary designer is done, merge the Linked Part Module back into the main part file.

    3. Delete the Linked Part Module file.

  4. Close all parts.
