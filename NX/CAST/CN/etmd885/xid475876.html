Update unresolved faces

  1. In the Part Navigator, expand the Part Module Frame Mount node.

  2. Examine the Part Navigator. Notice that in the Inputs collector the Extracted Face is the first suppressed feature.

  3. Double-click the Extracted Face to open it for edit.

    An Alerts message notifies you that the highlighted faces define more than one body. The highlighted faces are no longer contiguous. You need to make the faces that define the input to the part module contiguous again.

  4. Hold Shift and select the face as shown.

  5. Select the three faces as shown.

  6. Click OK to update the Extracted Face feature.

    Once the Extracted Face in the part module updates correctly all the down stream features update as well.

  7. Close the part now or close all parts when you exit the session.
