Activate Part Module

Opens the Part Module so that you can create features in the Work collector. When a Part Module is activated:

  • Only objects in the Inputs and Work collectors are shown in the graphics window. All other features in the file are hidden.

  • The last feature in the Work collector becomes the current feature.

  • Outputs can be specified.

Note

You can change the hide behavior with the Initially Show Only the Part Module when it is Activated customer default.

Deactivate Part Module

Returns NX to the default part modeling mode. When a Part Module is deactivated:

  • Objects in the Outputs collector are shown in the graphics window.

  • Objects in Inputs and Work collector are hidden.

  • The Part Module feature becomes the current feature.

Create Linked Part Module

Creates a WAVE Linked Part Module in another file. The Work collector is moved from the main part file to the WAVE linked file.

Update Input References

When working in a Linked Part Module file, updates the latest version of the features from the Inputs collector in the main part file.

Update Output References

When working in a main part file, updates the latest version of the features from the Outputs collector in the Linked Part Module file.

Merge Part Module

Merges the selected Linked Part Module into a Part Module in the main part file.
