Delete Partial Blend

  1. Open syn4_85_delete_partial_blend.

  2. On the Synchronous Modeling toolbar, click Delete Face .

  3. From the Type list, select Face .

  4. Select the face as shown.

  5. In the Delete Face dialog box, set the following:

    Settings group

    • Heal =

    • Delete Partial Blend =

    The preview shows that the blend is deleted and the neighboring faces are trimmed.

  6. In the Delete Face dialog box, set the following:

    Settings group

    • Heal =

    • Delete Partial Blend =

    • Setback = Selected Blend

    The preview shows that the blend is deleted and the two faces are capped by a setback along the selected blend.

  7. In the Delete Face dialog box, set the following:

    Settings group

    • Heal =

    • Delete Partial Blend =

    • Setback = Neighbor Blend

    The preview shows that the blend is deleted and the two faces are capped by a setback along the neighbor of the selected blend.

  8. Click Cancel.
