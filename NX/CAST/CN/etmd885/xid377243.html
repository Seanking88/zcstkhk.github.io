Point Set enhancement

What is it?

You can use the new Point Set type→Intersection Point to create a set of points that can be used in conjunction with Isolate Object of Feature, or any time you need a set of points as a feature.

Where do I find it?

Application

Modeling and Shape Studio

Toolbar

Feature→Datum/Point drop-down menu→Point Set

Menu

Insert→Datum/Point→Point Set

Location in dialog box

Type list→Intersection Points
