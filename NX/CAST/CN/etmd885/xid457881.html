Move Face options

  1. Open syn4_85_move_face_options.

  2. On the Synchronous Modeling toolbar, from the Modify Face Drop-down list, select Move Face .

  3. Select the seven faces shown.

  4. In the Move Face dialog box, set the following:

    Transform group

    • Motion = Distance

    • Specify Vector = –XC

    • Distance = 15

    Settings group

    • Move Behavior = Move and Adapt

    • Overflow Behavior = Automatic

    • Step Face = None

    The boss faces are trimmed by the small vertical face.

  5. In the Move Face dialog box, set the following:

    Settings group

    • Move Behavior = Cut and Paste

    • Heal =

    • Paste =

  6. Click OK.

    The boss and blends extend down to the horizontal face.

  7. Close the part now or close all parts when you exit the session.
