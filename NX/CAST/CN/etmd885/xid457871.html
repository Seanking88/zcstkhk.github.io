Move faces

  1. On the Synchronous Modeling toolbar, from the Modify Face Drop-down list, select Move Face .

  2. In the Move Face dialog box title bar, click Reset to return the dialog box and the Selection bar to the default settings.

  3. Select the two faces as shown.

  4. In the Move Face dialog box, in the Transform group, set the following:

    • Motion = Distance

    • Distance = .5

  5. Click OK.

    NX recognizes the colored faces as blends. New faces are created at the tangency point of the blends and the new faces.

  6. Close the part now or close all parts when you exit the session.
