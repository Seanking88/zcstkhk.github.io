Automatic reordering of feature groups

What is it?

You can now quickly create both timestamped and floating feature groups in the same part.

Two timestamped feature groups

A timestamped and a floating feature group

You can also control whether features are reordered when a new feature is created. When the Allow Automatic Reordering of Feature Groups check box is selected, newly created features will go into the active feature group, which may cause reordering so that the members of the feature group have a higher timestamp than their parents.

Why should I use it?

Use a mix of timestamped and floating feature groups in the same part when you want to accomplish both of the following:

  • Always position a certain kind of feature, such as a tool body feature, in a feature group at the bottom of the Part Navigator.

  • Position other feature groups at the timestamp at which they were created.

Where do I find it?

Application

Modeling and NX Sheet Metal

Prerequisite

Customer Defaults→Modeling→Feature Group→Prefer Sequential Timestamp Order

This option must be selected in order to see the Allow Automatic Reordering of Feature Groups check box in Modeling Preferences.

Menu

Preferences→Modeling→General tab→Allow Automatic Reordering of Feature Groups check box

Format→Group→Feature Group→Output list
