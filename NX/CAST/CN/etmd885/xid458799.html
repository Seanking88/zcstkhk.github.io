Create positive threaded holes

  1. Open des18_85_positive_holes.

  2. On the Feature toolbar, click Hole .

  3. On the Hole dialog bar rail, click Reset to set the dialog box to the default settings.

  4. From the Type list, select Threaded Hole.

  5. Select one of the four points on the datum plane.

    Make sure to select the Point of Point feature.

  6. In the Direction group:

    • From the Hole Direction list, select Along Vector.

    • From the Specify Vector list, select
      –ZC.

  7. From the Boolean list, select None.

  8. Under Thread Dimensions, from the Size list, select M4 x 0.7.

  9. Click Apply.

    What appears to be external threads will be internal threads when the positive hole is subtracted from a solid body.

  10. Select one of the 3 remaining points.

  11. In the Direction group, from the Hole Direction list, select Along Vector

  12. Click Apply.

  13. Repeat Steps 1 through 3 for the remaining 2 points.

  14. Click Cancel.
