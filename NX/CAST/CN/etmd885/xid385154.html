Create a Boss at the top isolated point

  1. Click Layer Settings and clear the check box for Layer 63.

  2. On the Feature toolbar, click Boss .

  3. For the Placement Face, select the inset face.

  4. Type the following values:

    • Diameter: 0.188

    • Height: 0.25

  5. Click OK.

  6. Click Point onto Point .

  7. Select the top isolate feature. (Make sure to select the Point of Isolate, not the Point of Point Set.)
