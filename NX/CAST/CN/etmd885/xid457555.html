Make the Linked Part Module the work part

  1. Open des18_85_main_part_file.

  2. Make sure that the Tools→Update→Interpart Update→Delay Geometry, Expressions, and PMI Update option is not selected.

  3. In the Part Navigator, right-click Linked Part Module “Trigger Design” and choose Expand All.

    The Linked Part Module contains two collectors: Inputs and Outputs. The Work collector is in the linked part file. In the Outputs collector, notice that the two faces are WAVE links.

    In the linked part file, a rib has been added in the trigger area of this drill housing. To complete the design, you need to add this rib to the main part file.

  4. Right-click Linked Part Module “Trigger Design” and choose Make Linked Work.

    The Linked Part Module file is displayed.
