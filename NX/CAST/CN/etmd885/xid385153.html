Create the Isolate Objects

  1. Click Layer Settings and select the check box for Layer 63.

    This is the plane you will use as the proximity object for isolating a point in each point set.

  2. Choose Insert→Associative Copy→Isolate Object of Feature.

  3. For the Feature, select the first point set you created.

  4. For the Proximity Object, select the plane.

  5. Click Apply.

  6. For the Feature, select the second point set you created.

  7. For the Proximity Object, select the plane.

  8. Click OK.
