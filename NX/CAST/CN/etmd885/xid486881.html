Check your Customer Defaults

Before opening the part you should check your Customer Defaults to ensure they are set properly.

  1. Choose File→Utilities→Customer Defaults.

  2. In the Customer Defaults dialog box, on the left, select Modeling→Feature Group.

  3. Verify the following settings:

    Prefer Sequential Timestamp Order =

    Feature Group Types Allowed = Timestamped Only

  4. Click OK.

  5. (If necessary) Click OK on the Customer Defaults message.

  6. If you made any changes to the Customer Defaults, exit and then restart NX in order for the customer default changes to take effect.
