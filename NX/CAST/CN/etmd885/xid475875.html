Add blend features

  1. Open des18_85_resolve_update.

    Blend features need to be added to the model in the feature history before the part module shown below.

  2. Make sure that the Tools→Update→Interpart Update→Delay Geometry, Expressions, and PMI Update option is not selected.

  3. In the Part Navigator, right-click Extrude (34) and choose Make Current Feature.

  4. Add a 2 mm blend to the inside edge shown.

  5. Add a 2 mm blend to the outside edge shown.

  6. In the Part Navigator, make the last feature in the history tree the current feature.

    The Information window displays an Update Warning and Failure Report. Many features are suppressed because of the two additional blends.
