Resize blends

  1. On the Synchronous Modeling toolbar, from the Detail Feature Drop-down list, select Resize Blend .

  2. Attempt to select either of the colored blend faces.

    Note

    These faces cannot be selected because NX does not recognize them as blends.

  3. On the Synchronous Modeling toolbar, from the Detail Feature Drop-down list, select Label Notch Blend .

  4. Select all six colored blend faces shown.

    Tip

    If you see an active <OK> button in the dialog box, you can choose another command without clicking the middle mouse button to complete the current command.

  5. On the Synchronous Modeling toolbar, from the Detail Feature Drop-down list, select Resize Blend .

  6. Select all six colored blend faces shown.

  7. In the on-screen Radius box, type 0.2 and click the middle mouse button.

    Both blends now have the same radius.
