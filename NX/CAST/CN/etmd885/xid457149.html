Create a Linked Part Module

In your role as the principal designer, you need to convert an internal Part Module to an external Linked Part Module. This will allow a secondary designer to work independently on the Linked Part Module file.

  1. Open des18_85_linked_part_module_main.

  2. Make sure that the Tools→Update→Interpart Update→Delay Geometry, Expressions, and PMI Update option is not selected.

  3. In the Part Navigator, right-click Part Module “Heat Sink” and choose Expand All.

    The part module contains three collectors: Inputs, Work, and Outputs. A sketch and two extrude features are in the Work collector. All three features make up this one part module.

    You need to convert this part module to an external WAVE Linked Part Module.

  4. Right-click Part Module “Heat Sink” and choose Create Linked Part Module Part.

  5. In the Name box, type heat_sink_linked_file.

    In the Folder box, choose the desired folder, and click OK.

  6. Examine the Part Navigator. Notice that:

    • The new feature name is Linked Part Module.

    • The Work collector is gone. It moved to the Linked Part Module file.

    • The Outputs collector has a WAVE link:

      Linked Face (13).

      This child feature is WAVE linked to the parent features in Outputs collector of the Linked Part Module file.

  7. Right-click Linked Part Module “Heat Sink” and choose Make Linked Work, to check the part file you are sending to the secondary designer.

    You can also use the Make Linked Work command to display a secondary designer’s work.

  8. On the Standard toolbar click, Save .

  9. To display the main part again, right-click Linked Part Module “Heat Sink” and choose Make Main Work.

  10. Choose File→Close→Selected Parts.

  11. In the Part group, select heat_sink_linked_file.prt and click, OK.

    Closing the Linked Part Module file unlocks the file so the secondary designer can start working on the file.
