Change the customer default settings and restart NX

  1. Choose File→Utilities→Customer Defaults.

  2. In the Customer Defaults dialog box, on the left, select Modeling→Feature Group.

  3. Select the Prefer Sequential Timestamp Order check box.

  4. From the Feature Group Types Allowed list, select Timestamped and Floating.

  5. Click OK.

  6. (If necessary) Click OK on the Customer Defaults message.

  7. Exit and then restart NX in order for the customer default changes to take effect.
