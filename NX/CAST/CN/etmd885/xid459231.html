Create a Swept feature from the conic curve

  1. From the Surface toolbar, click Swept .

  2. Select the conic curve as shown.

  3. In the Swept dialog box, in the Guides group, click Select Curve.

  4. In the graphics window, select the horizontal edge as shown.

    With all necessary parameters defined, a preview of the Swept feature appears.

  5. Click OK to create the Swept feature.
