Select the surface and the curves

  1. Open des18_85_global_shaping_by_curve .

  2. Choose Edit→Surface→Global Shaping .

  3. In the Global Shaping dialog box, from the Type list, select By Curve.

  4. Select the sheet to deform.

  5. For the Base Curve, select the bottom edge of the surface.

  6. For the Control Curve, use Feature Curves, and select the composite curve on the right end as shown:

    Notice the slight twist.

    It may be easier to see the twist if in the Preview group, you select Show Result. When you are done examining it, click Undo Result.
