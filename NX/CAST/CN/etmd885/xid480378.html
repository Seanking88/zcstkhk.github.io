Clock a pattern instance to a new location

You want to move the two instances, symmetrically, upward 100 mm.

  1. In the Part Navigator, double-click Pattern (Along).

  2. Select the two instance handles as shown, then right-click and select Clock.

  3. Select the axis to translate along the YC-axis.

  4. In the dynamic input box type a distance of 100.

  5. In the Clock dialog box, click OK.

  6. In the Pattern Feature dialog box, click OK.

  7. Close all parts.
