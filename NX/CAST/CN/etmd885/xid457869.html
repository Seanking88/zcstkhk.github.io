Move faces

  1. Open syn4_85_label_notch_blend.

  2. On the Synchronous Modeling toolbar, from the Modify Face Drop-down list, select Move Face .

  3. In the Move Face dialog box title bar, click Reset to return the dialog box and the Selection bar to the default settings.

  4. Select the two faces as shown.

  5. In the Move Face dialog box, set the following:

    Transform group

    • Motion = Distance

    • Distance = .22

  6. Preview the change.

    NX does not recognize the colored faces as blends because their edges change as the top faces move.

  7. Click Cancel.
