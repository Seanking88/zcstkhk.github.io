Specify Inputs

  1. In the Part Navigator, right-click Linked Part Module “Heat Sink Design” and choose Make Linked Work.

  2. In the Linked Part Module Work collector, double-click “SKETCH_001”.

    Note

    In the following graphic, the linked face is hidden for clarity.

  3. Edit the value of the p412 expression from 5.8 to 10.

  4. Press Ctrl + Q to finish the sketch.

    NX updates the sketch but the extrude features are not updated. The face used to trim the extrude is not large enough to accommodate this change. You need to add another face to the Inputs collector in the main part file.

    Note

    In the following graphic, the linked face is displayed.

  5. In the Part Navigator, right-click Linked Part Module “Heat Sink Design” and choose Deactivate Part Module.

    No solid is displayed in the graphics window because in the Outputs collector, the Extracted Face is currently suppressed.

  6. Right-click Linked Part Module “Heat Sink Design” and choose Make Main Work.

  7. Double-click Linked Part Module “Heat Sink Design”.

  8. In the Input References group, from the List box, select Extracted Face (7).

    NX highlights the current input face.

  9. Select the small adjoining face as shown and click OK.

    Notice that the Linked Part Module is updated, and the face highlighted in the following extends graphic extends over the added input face. After you resolve the issue with the Inputs in the main part file, in this Linked Part Module file, NX does the following in the order shown:

    • Updates the WAVE linked input face.

    • Unsuppresses the extrude features in the Work collector.

    • Unsuppresses the suppressed extracted output face.

    • Updates the Linked Part Module in the main part file.

    If you were collaborating with a design team, the process to update Inputs from the main part file into the Linked Part Module file is as follows.

    1. The principal designer changes the inputs in the main part file, and saves the file.

    2. The secondary designer uses the Update Input References command in the Linked Part Module file, and saves the file.

    3. The principal designer uses the Update Output References command in the main part file to see the changes made by the secondary designer.

  10. Close all parts.
