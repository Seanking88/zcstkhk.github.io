Create a pattern of helix features

  1. On the Feature toolbar, click Pattern Feature .

  2. Select the Sweep feature you just created.

  3. In the Pattern Definition group, set Layout to Circular.

  4. Under Rotation Axis, click Specify Vector.

  5. Select the long vertical cylinder as shown.

  6. Under Angular Direction:

    • Spacing = Count and Pitch

    • Count = 4

    • Pitch Angle = 90

  7. Click OK.
