Design a team workflow

Use what you learned in this course and the scenario in the following steps to play the role of both the principal designer and a secondary designer.

  1. In your role as the principal designer, open des18_85_case_on_your_own.

  2. Make sure that the Tools→Update→Interpart Update→Delay Geometry, Expressions, and PMI Update option is not selected.

  3. Convert the Part Module to a Linked Part Module.

  4. In your role as the secondary designer, in the Linked Part Module, extrude the sketch curves as shown and use 12 for the Start Distance Value.

  5. Extrude the sketch curves as shown.

  6. Unite the five solids.

  7. In your role as the principal designer, unite the Linked Part Module to the main body.

    Tip

    Did you define the Outputs in the Linked Part Module?

  8. Prepare the main part file for release. Convert the external Linked Part Module to an internal Part Module.

    These are the basics steps that should be used when a design team needs to collaborate on a single part design. The Principal designer:

    1. Creates Part Modules in unfinished states in the main part file and then breaks these off into separate Linked Part Module files which are distributed to members of the design team.

    2. Updates each Linked Part Module feature in the main part file to view the designer’s progress and ensure that continuing downstream work is based on the latest geometry in the main part file.

    3. Merges the completed Linked Part Module files into the main part file.

  9. Close all parts.
