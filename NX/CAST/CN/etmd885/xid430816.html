Mirror a threaded mounting tab

  1. Open des18_85_box_frame.

  2. Choose Insert→Associative Copy→Mirror Feature

    Note

    Mirror Feature is also available on Associative Copy Drop-down list on the Feature toolbar.

  3. Select the tab extrude feature and the symbolic thread on the right side of the box.

    Note

    If you select the symbolic thread feature first, an Alert message displays telling you that the thread cannot be mirrored without its parents. You can dismiss the Alert.

  4. Click the middle mouse button.

  5. In the Mirror Plane group, from the Plane list, select New Plane.

  6. With Specify Plane highlighted, select one of the inside vertical faces, and then the other to create a mirror plane.

  7. Expand the Settings group and make sure the Maintain Thread Handedness is selected.

  8. Click Apply.

  9. Repeat the above procedure to mirror the two tab features and their threads to the bottom of the box frame.

    • You should have three selected features – the original extrude feature, the symbolic threads, and the first mirrored feature.

    • Click New Plane and select the upper and lower inside faces to specify a bisecting plane.

    • In the Settings group, make sure Maintain Thread Handedness is selected.

    • Click Apply.
