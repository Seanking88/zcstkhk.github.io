Create a conic curve on a solid body

  1. Open des18_85_general_conic.

  2. Choose Insert→Curve→General Conic.

    You will create a conic curve on the planar face of the solid body.

  3. In the General Conic dialog box, from the Type list, select 3 Points, 2 Slopes.

  4. For the Specify Start Point, select the start point of the large arc as shown.

  5. For the Specify Interior Point, select the top quadrant point of the arc as shown.

  6. For the Specify End Point, select the end point of the arc as shown.

  7. From the Specify Start Slope list, select Two Points to define a vector direction.

  8. Select the arc center of the small hole on the right side of the solid body for the start point of the start slope vector.

  9. Select the mid-point of the bottom edge of the solid body for the end point.

  10. From the Specify End Slope list, select Two Points to define the vector direction.

  11. Select the arc center of the small hole on the left side of the solid body for the start point of the end slope vector.

  12. Select the mid-point of the bottom edge of the solid body for the end point.

    A preview of the conic curve appears.

  13. Click OK to create the conic curve.
