Create the intersection point sets

  1. Choose Insert→Datum/Point→Point Set.

  2. From the Type list, choose Intersection Points.

  3. In the Curve, Face or Plane group, with Select Object active, select the three planes shown below:

  4. In the Curve group, with Select Curve active, select the top section line you just created.

  5. Click Apply.

  6. From the Type list, choose Intersection Points.

  7. In the Curve, Face or Plane group, with Select Object active, select the three planes shown below:

  8. In the Curve group, with Select Curve active, select the lower section line.

  9. Click OK.

  10. Hide the planes and the section lines.
