创建部件间关系

  1. 点击标准工具条上的打开 。

  2. 在打开对话框中，点击选项。

  3. 在装配加载选项对话框中，设置如下参数：

    部件版本组：

    • 加载 = 从文件夹

    范围组：

    • 加载 = 所有组件

    • 使用部分加载 =

    • 使用轻量级表示 =

    • 加载部件间数据 =

    • 加载父项 = 所有级别

  4. 点击确定。

  5. 打开 wav3_brake_rotor_machining。

    在装配导航器中，您可以看到部件是一个装配，并且铸件是其中的组件，它当前处于隐藏状态。

    在部件导航器中，您可以看到第一个特征是链接的基准坐标系，这个部件间链接是在加工件与铸件之间，四个草图是基于这个坐标系建立的。

  6. 显示铸件。

    您可以看到，链接的基准坐标系位于铸件的中心，从被所有部件共享的公共坐标系开始创建加工特征是很重要的。

  7. 点击装配工具条上的 WAVE 几何体链接器 。

  8. 在 WAVE 几何体链接器对话框中，设置如下参数：

    • 类型 = 体

      设置组：

    • 关联 =

    • 隐藏原先的 =

    • 固定于当前时间戳记 =

    • 使用父部件的显示属性 =

    • 设为与位置无关 =

    • 复制螺纹 =

  9. 选择铸件并点击确定。

    在部件导航器中，已创建链接体 (6)，部件间链接储存在加工件中。

  10. 在装配导航器中，右击铸件并选择替换引用集→空。

    一旦完成创建部件间链接，您在处理加工部件时就需要再显示铸件。
