创建箱体顶盖

您已经使用 WAVE 几何体链接器来先创建链接的特征，现在您将看到链接的特征可以自动创建。

  1. 将 wav3_box_top 设为工作部件。

  2. 点击特征工具条上的拉伸 。

  3. 在选择条上，设置如下几何体选择过滤器：

    1. 在选择范围列表中选择整个装配

    2. 打开创建部件间链接

    3. 在曲线规则列表中选择相连曲线

    4. 打开路径选择 .

  4. 选择箱体密封垫的外侧边，如图所示。

    阅读部件间复制信息框并点击确定。

  5. 在拉伸对话框中，设置如下参数：

    1. 限制组：

      • 开始 = 值

      • 距离 = 0

      • 结束 = 值

      • 距离 = 5

    2. 拔模组：

      • 拔模 = 从起始限制

      • 角度 = 2

    3. 偏置组：

      • 偏置 = 单侧

      • 结束 = 2

  6. 点击应用。

  7. 使用1 mm 的偏置来拉伸两个孔，以用作螺栓间隙孔。

  8. 在顶盖的底侧拉伸内部边用作密封。

    为这个配合特制设置下列参数：

    1. 限制组：

      • 开始 = 值

      • 距离 = 0

      • 结束 = 值

      • 距离 = 2

    2. 拔模组：

      • 拔模 = 从起始限制

      • 角度 = 2

    3. 偏置组：

      • 偏置 = 单侧

      • 结束 = -1

    在部件导航器中，您可以看到部件中包含三个链接的复合曲线，当您打开创建部件间链接 选项时，将在箱体密封垫与箱体顶盖之间创建 WAVE 链接，部件间链接将储存在箱体顶盖部件中。

  9. 按下 Ctrl+W，以打开显示和隐藏命令。

  10. 在显示和隐藏对话框的全部行中，点击 以显示箱体。
