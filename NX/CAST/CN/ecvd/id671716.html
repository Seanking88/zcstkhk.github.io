在面上偏置曲线的步骤

  1. 在曲线工具条上，点击面中的偏置曲线 。

  2. 通过以下其中一项定义基本偏置曲线：

    • 从原始的面中曲线创建固定偏置的偏置曲线。

      1. 将类型设置为常数。

      2. 选择您想要偏置的一组相连的曲线或边，您选择的线串将成为截面线1:偏置1。

      3. 单击鼠标中键以完成曲线线串选择，曲线将关闭高亮显示，并且选择的线串上将出现一个截面线1标签。

      4. 再次单击鼠标中键以前进到下一个步骤进行偏置面或平面的选择。

      5. 选择截面线1所在的体或面，在您先前选择的线串上将出现一个手柄。

        提示

        如果只有一个体，或者曲线线串是一条面上曲线或边，自动判断一个或多个面，则这一步可能可以跳过。

      6. 在面上拖动截面线1:偏置1手柄。

        图形窗口中将出现一个屏显输入框，持续更新偏置距离值。

        提示

        要定位偏置曲线，可以：

        在屏显输入框中键入距离值并回车
        或
        拖动手柄到想要的距离
        您也可以在树形列表中选择偏置曲线 1的距离输入，然后再那里更改偏置距离值。

      7. 如果您想要从原来的截面线1偏置另一个距离，打开曲线组中的列表，点击新建，在图形窗口中再次选择截面线1的曲线线串，然后拖动它到想要的位置，它将成为截面线2:偏置1。

      8. 当您完成截面线1的偏置曲线创建，可以点击添加新集来选择一条新的曲线或边来创建另外的偏置曲线，只要您喜欢，在面上偏置特征可以包含尽可能多的偏置曲线线串。

    • 以距原始面中曲线上的点不同距离创建可变偏置曲线。

      1. 将类型设置为可变。

      2. 选择一个面或平面。

      3. 选择要偏置的曲线。

      4. 在偏置组中，设置规律类型来定义面中的偏置曲线。

      5. 为选择的规律类型指定参数。

  3. 您可以随时设置以下选项

    • 偏置方法(从以下选项中选择一种)

      • 弦：(默认) 使用线串曲线上各点之间的线段，基于弦距离创建偏置曲线

      • 弧长：沿线串曲线的圆弧创建偏置曲线

      • 测量：沿面上的最短距离创建偏置曲线

      • 相切：沿曲线最初所在面的切线，在一定距离处创建偏置曲线，并将其重新投影到该面上

      • 投影距离： 用于按指定的法向矢量在虚拟平面上指定偏置距离

    • 倒圆尖角(选择以下选项中的一项)

      • 矢量

      • 最适合

      • 投影矢量

    • 修剪和延伸偏置曲线 (选择以下选项中的一项或多项)

      • 在截面内修剪至彼此 (默认)

      • 在截面内延伸至彼此 (默认)

      • 修剪至面的边

      • 延伸至面的边

      • 移除偏置曲线内的自相交 (默认)

    这些选项将持续影响会话中余下的特征创建，直到您更改它们。

  4. 选择确定以完成特征创建。
