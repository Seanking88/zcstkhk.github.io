材料类型

层合板层片可以引用：

  • 各向同性材料

  • 各向异性材料

  • 层片材料

层片材料是特定于层合板的对象，综合了纤维和矩阵材料属性，并计算等效刚度和强度属性。

层合板各向同性材料属性

    必要属性

  • 质量密度(RHO)

  • 三个属性中的两个：

    • 杨氏模量(E)

    • 泊松比(NU)

    • 剪切模量(G)

  • 使用以下关系计算得出的第三个属性：E = 2G (1 + NU)

    可选属性

  • Tsai-Wu 交互作用系数(F12)

  • 热膨胀系数(A)

  • 导热系数(K)

  • 比热(CP)

  • 应力/应变极限

注释

未定义的应力极限值：

  • 如果未定义所有应力极限值，将设为0.0

  • 如果已定义所有应变极限值，将使用应变极限值计算

层合板各向异性材料属性

    必要属性

  • 质量密度(RHO)

  • 杨氏模量(E1 与 E2)

  • 泊松比(NU12)

  • 剪切模量(G12、 G13以及 G23)

    可选属性

  • Tsai-Wu 交互作用系数(F12)

  • 热膨胀系数(A1 与 A2)

  • 导热系数(K1 与 K2)

  • 比热(CP)

  • 应力/应变极限

注释

各向异性应力极限值与各向同性应力极限值的计算方法相同。

各向同性材料与各向异性材料的其它说明

  • 层合板参考温度将替代参考温度材料属性

  • NX 复合层合板不使用第三个方向上的属性：

    • 杨氏模量(E3)

    • 泊松比(NU13)与泊松比(NU23)

    • 热膨胀系数(A3)

    • 应力极限张量(ST3)、应变极限张量(XT3)

    • 压缩应力极限(SC3)、压缩应变极限(XC3)

    • 剪切应力极限(S13)以及剪切应力(S23)、剪切应变极限(X13)以及剪切应变(X23)

    • 导热系数(K3)

    可将这些属性导出至 ANSYS 或 Abaqus。
