Drill the 0.600 diameter centerline hole

  1. On the Insert toolbar, click Create Operation .

  2. In the Create Operation dialog box, click Centerline Breakchip .

  3. In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = DRILLING_TOOL

    Geometry = AVOIDANCE

    Method = LATHE_CENTERLINE

  4. Click OK.

  5. In the Centerline Breakchip dialog box, in the Start Point and Depth group, click Display Start and End .

    You will display the start point (1) and then specify an end point (2) that defines a depth that drills completely through the part.

  6. In the Depth Option list, select End Point

  7. Select the Break Through check box.

  8. Click Inferred Point .

  9. Select the end of the vertical line to define the drilling end point.

  10. In the Centerline Breakchip dialog box, in the Cycle Type group, expand the Chip Removal group, and from the Increment Type list, select Constant.

  11. In the Constant Increment box, type 1.00.

  12. In the Options group, click Edit Display .

  13. In the Display Options dialog box, in the Tool group, from the Tool Display list, select 2D.

  14. Click OK.

  15. In the Centerline Breakchip dialog box, in the Actions group, click Generate .

  16. Click OK to complete the operation.
