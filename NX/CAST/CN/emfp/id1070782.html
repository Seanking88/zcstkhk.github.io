Define collision avoidance geometry

Create an Avoidance operation inside the TURN_WORKPIECE node.

  1. On the Insert toolbar click Create Geometry .

  2. In the Create Geometry dialog box, from the Type list, select turning.

  3. In the Geometry Subtype group, click AVOIDANCE .

  4. In the Location group, from the Geometry list, select TURN_WORKPIECE.

  5. Click OK.

  6. In the Avoidance dialog box, in the Motion to Start Point (ST) group, from the Motion Type list, select Direct.

  7. Click Inferred Point .

  8. Select the point.

    Define a Start Point at the indicated location that uses a direct motion to the point.

  9. In the Motion to Start of Engage group, from the Motion Type list, select Radial→Axial.

  10. In the Motion to Return Point / Clearance (RT) group, from the Motion Type list, select Axial→Radial.

  11. From the Point Option list, select Same as Start.

  12. Click OK.
