Counterbore the 0.750 diameter hole

  1. In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = COUNTERBORING_TOOL_1.100

    Geometry = WORKPIECE

    Method = DRILL

  2. Select the hole as you did for the spot drilling operation.

  3. Specify the tool axis as normal to the part surface and use the arc axis.

  4. Click Generate .

  5. Click OK to complete the operation.
