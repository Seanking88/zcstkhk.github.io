Finish the top face of the part with a manual cut pattern

You will use a Face Milling operation with a manual cut pattern to finish the top face of the part.

  1. On the Insert toolbar, click Create Operation .

  2. In the Create Operation dialog box, in the Type group, from the list, select mill_planar.

  3. In the Operation Subtype group, click Face Milling Manual .

  4. In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = MILL_3

    Geometry = WORKPIECE

    Method = MILL_FINISH

  5. Click OK.

  6. In the Face Milling Manual dialog box, in the Geometry group, click Specify Cut Area .

  7. Select the top face of the part.

  8. Click OK.

  9. Right click in the background of the graphics window, and choose Replace View→Top.

    This view lets you clearly see the manual cut pattern.

    You will specify a 2D tool display and a Silhouette Path Display so you can clearly see the area that the tool cuts.

  10. In the Face Milling Manual dialog box, in the Options group, click Edit Display .

  11. In the Display Options dialog box, in the Tool group, from the Tool Display list, select 2D.

  12. In the Path group, make sure the Path Display is set to Silhouette.

  13. Click OK.
