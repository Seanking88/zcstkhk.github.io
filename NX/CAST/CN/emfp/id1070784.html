Spot drill the 0.600 diameter centerline hole

  1. On the Insert toolbar, click Create Operation .

  2. In the Create Operation dialog box, make sure turning is selected from the Type list.

  3. In the Operation Subtype group, click Centerline Spotdrill .

  4. In the Location group, set the options as shown.

    Program = 1234

    Tool = SPOTDRILLING_TOOL

    Geometry = AVOIDANCE

    Method = LATHE_CENTERLINE

  5. Click OK.

  6. In the Centerline Spotdrill dialog box, in the Options group, click Edit Display .

  7. In the Display Options dialog box, in the Tool group, from the Tool Display list, select 2D.

    The tool path will generate using a 2D tool display.

  8. Click OK.

  9. In the Centerline Spotdrill dialog box, click Generate .

  10. Click OK to complete the operation.
