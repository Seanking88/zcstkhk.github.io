Finish the corners and fillets with multiple passes

  1. On the Insert toolbar, click Create Operation .

  2. In the Create Operation dialog, in the Type group, from the list, choose mill_contour.

  3. In the Operation Subtype group, click Flowcut Reference Tool .

  4. In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = EM_.125_BN

    Geometry = MILL_AREA

    Method = MILL_FINISH

  5. Click OK.

  6. In the Flowcut Ref Tool dialog box, in the Drive Method group, click Edit .

  7. In the Flow Cut Drive Method dialog box, in the Non-steep Cutting group, in the Stepover box, type 30.

    You want the sequence of cuts to start at the center flow cut and move toward the outside passes.

  8. From the Sequencing list, select Inside-Out.

    The cut region needs to cover a width slightly wider than the 0.1500 radius fillets.

  9. In the Reference Tool group, and in the Reference Tool list, select EM-.375”_BN. Verify the Overlap Distance is 0.0300.

  10. Click OK.

  11. In the Flowcut Ref Tool dialog box, in the Actions group, click Generate .

  12. Click OK to complete the operation.
