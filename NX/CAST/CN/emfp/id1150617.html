Spot drill the two 0.300 diameter holes

  1. On the Insert toolbar, click Create Operation.

  2. In the Create Operation dialog box, in the Operation Subtype group, click Spot Drilling .

  3. In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = SPOTDRILLING_TOOL

    Geometry = WORKPIECE

    Method = DRILL

  4. Click OK.

  5. In the Spot Drilling dialog box, in the Geometry group, click Specify Holes .

  6. In the Point dialog box, click Select.

  7. Select the two holes shown to spot drill.

  8. Click OK.

  9. In the Point dialog box, click OK.

  10. In the Spot Drilling dialog box, in the Cycle Type group, click Edit parameters .

  11. In the Specify Number of dialog box, accept the default of 1 set and click OK.

  12. In the Cycle Parameters dialog box, click Depth (Tip).

  13. In the Cycle Depth dialog box, click Tool Tip Depth.

  14. In the Depth box, type 0.05 and click OK.

  15. In the Cycle Parameters dialog box, click OK.

    Because the holes are not parallel to the ZM axis, you must specify the tool axis.

  16. In the Spot Drilling dialog box, in the Tool Axis group, from the Axis list, select Normal to Part Surface.

  17. Select the Use Arc Axis check box.

  18. In the Actions group, click Generate .

  19. Click OK to complete the operation.
