Define an avoid move

  1. In the Path Settings group, click Avoidance .

    To avoid the collision, first you will define a clearance plane 0.250 above the top of the part.

  2. Click Clearance Plane-None.

  3. In the Clearance Plane dialog box, click Specify.

  4. Select the top face of the part.

  5. In the Plane dialog box, in the Offset group, in the Distance box, type 0.25.

  6. Click OK.

  7. In the Specify dialog box, click OK to accept the clearance plane.

  8. Click OK to return to the SPOT_DRILLING dialog box.

    Now you will define an avoid motion that uses the clearance plane to traverse up and over the top of the part between the start and end points shown.

  9. In the Geometry group, click Specify Holes .

  10. In the Point dialog box, click Avoid.

  11. Select the hole shown as the start point.

  12. Select the hole shown as the end point.

  13. Click Clearance Plane.

  14. Click OK.

  15. Click OK to return to the Spot Drilling dialog box.

  16. In the Actions group, click Generate .

  17. Click OK to complete the spot drilling operation.
