Drill the 0.375 diameter holes

  1. On the Insert toolbar, click Create Operation.

  2. In the Create Operation dialog box, in the Operation Subtype group, click DRILLING .

  3. In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = DRILL_.375

    Geometry = WORKPIECE

    Method = DRILL

  4. Click OK.

  5. In the Drilling dialog box, in the Geometry group, click Specify Holes .

  6. In the Point dialog box, click Select.

  7. Select the two holes as you did for the spot drilling operation.

  8. Click OK.

  9. In the Point dialog box, click OK.

  10. In the Drilling dialog box, in the Actions group, click Generate .

  11. Click OK to complete the operation.
