Optimize the tool path and generate it

You need to optimize the tool path to achieve the most efficient motion.

  1. In the Point dialog box, click Optimize.

  2. Click Shortest Path.

  3. Click Start Point-Automatic.

  4. Select the hole shown as the automatic start point.

  5. Click Optimize.

  6. Click Accept.

  7. In the Point dialog box, click Planning Complete.

    You will specify a tool tip depth of .0500.

  8. In the Spot Drilling dialog box, in the Cycle Type group, click Edit parameters .

  9. In the Specify Number of dialog box, accept the default of 1 set and click OK.

  10. In the Cycle Parameters dialog box, click Depth (Tip).

  11. Click Tool Tip Depth.

  12. In the Depth box, type 0.05 and click OK.

    The Cycle Parameters dialog box should reflect the tool tip depth.

  13. Click OK.

  14. In the Spot Drilling dialog box, in the Actions group, click Generate .

    The tool collides with the part when traversing between the two areas.
