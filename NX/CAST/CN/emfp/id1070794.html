Define avoidance geometry for the od grooves

The next three operations will use the indicated Start Point and Return Point to rough and finish the outside diameter grooves.

Create an AVOIDANCE operation object inside TURN_WORKPIECE node.

  1. Click Create Geometry .

  2. In the Geometry Subtype group, click AVOIDANCE .

  3. From the Geometry list, select TURN_WORKPIECE.

  4. Click OK.

  5. In the Avoidance dialog box, in the Motion to Start Point (ST) group, select Direct from the Motion Type list.

    Define a Start Point at the position shown in step 7 that uses a direct motion to the point.

  6. Click Inferred Point .

  7. Select the point.

  8. In the Motion to Start of Engage group, from the Motion Type list, select Axial→Radial.

    Define an Axial→Radial motion to the start of the engage.

  9. In the Motion to Return Point / Clearance (RT) group, select Radial→Axial from the Motion Type list.

    Define a Radial→Axial motion to a Return Point defined at the same position as the Start Point.

  10. From the Point Option list, select Same as Start.

  11. Click OK.
