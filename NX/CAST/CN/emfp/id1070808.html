Finish the inside diameter

You will create an operation that finishes the contoured surface on the inside diameter.

  1. Create a Finish Bore ID operation.

    In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = ID_55_L

    Geometry = AVOIDANCE

    Method = LATHE_FINISH

    You will define trim planes that constrain the cut region so that the contoured surface is cut.

    Specify a radial trim plane at the end of the vertical line (1) and an axial trim plane through the midpoint of the horizontal line (2) to define the cut region (3).

  2. In the Finish Bore ID dialog box, in the Geometry group, in the Cut Regions row, click Edit.

  3. In the Cut Regions dialog box, in the Radial Trim Plane 1 group, from the Limit Option list, select Point.

  4. Click Inferred Point .

  5. Select the end of the vertical line.

  6. In the Axial Trim Plane 1 group, from the Limit Option list, select Point.

  7. Click Inferred Point .

  8. Select the midpoint of the horizontal line.

  9. In the Preview group, click Display .

  10. Click OK to accept the geometry containment.

  11. Generate the tool path using a 2D display.

  12. Click OK to complete the operation.
