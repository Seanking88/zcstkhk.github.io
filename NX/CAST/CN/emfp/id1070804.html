Rough bore the inside diameter

You will create an operation that rough bores the inside diameter.

  1. Create a Rough Bore ID operation.

    In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = ID_80_L

    Geometry = AVOIDANCE

    Method = LATHE_ROUGH

  2. In the Path Settings group, from the Reversal Mode list, select Omit.

    The Reversal Mode should be set to Omit so that the tool does not cut down into the neck down areas.

  3. Generate the tool path using a 2D display.

  4. Click OK to complete the operation.
