Create an operation to semi-finish the non-steep areas

  1. On the Insert toolbar, click Create Operation .

  2. In the Create Operation dialog box, in the Type group, from the list, select mill_contour.

  3. In the Operation Subtype group, click CONTOUR_AREA .

  4. In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = EM_14

    Geometry = WORKPIECE

    Method = MILL_SEMI_FINISH

  5. Click OK.

  6. In the Contour Area dialog box, in the Drive Method group, click Edit .

  7. In the Area Milling Drive Method dialog box, in the Steep Containment group, from the Method list, select Non-steep.

  8. In the Steep Angle box, type 35.

  9. In the Drive Settings group, set the options as shown.

    Stepover = % Tool Flat

    Percent of Flat Diameter = 25.0000

    Cut Angle = Specify

    Angle from XC = 0.0000

  10. Click OK.

  11. In the Contour Area dialog box, in the Actions group, click Generate .

    Notice the edge tracing.

  12. Click OK to complete the operation.

  13. Refresh the graphics window.
