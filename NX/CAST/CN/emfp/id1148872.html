Create an operation to rough out the initial material

You will create a Cavity Mill operation.

  1. On the Insert toolbar, click Create Operation .

  2. In the Create Operation dialog box, in the Type group, from the list, select mill_contour.

  3. In the Operation Subtype group, click Cavity Mill .

  4. In the Location group, set the following:

    Program = PROGRAM

    Tool = MILL_1

    Geometry = WORKPIECE

    Method = MILL_ROUGH

  5. Click OK.
