Finish the end face

  1. On the Insert toolbar, click Create Operation .

  2. In the Create Operation dialog box, in the Operation Subtype group, click Facing .

  3. In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = UGTI0101_011

    Geometry = AVOIDANCE

    Method = LATHE_FINISH

  4. Click OK.

  5. In the Facing dialog box, in the Geometry group, in the Cut Regions row, click Edit .

    You will edit the cut region to define a Trim Plane so that only the end of the part is faced.

  6. In the Cut Regions dialog box, in the Axial Trim Plane 1 group, from the Limit Option list, select Distance.

  7. Click OK to accept the Trim Plane at 0.0000.

  8. In the Facing dialog box, in the Options group, click Edit Display .

  9. In the Display Options dialog box, in the Tool group, from the Tool Display list, select 2D.

  10. Click OK.

  11. In the Facing dialog box, in the Actions group, click Generate .

  12. Click OK.
