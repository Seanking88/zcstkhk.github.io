Edit the operation to drill through the part

  1. In the Operation Navigator, right-click in the background and choose Program Order View.

  2. In the Operation Navigator, double-click the DRILLING_.25 node.

    You will edit the DRILLING_.25 operation so that the blind and thru holes are drilled to model depth.

  3. In the Spot Drilling dialog box, in the Cycle Type group, click Edit Parameters .

  4. In the Specify Number of dialog box, accept the default of 1 set and click OK.

  5. In the Cycle Parameters dialog box, click Depth (Tip).

  6. In the Cycle Depth dialog box, click Model Depth.

    The Cycle Parameters dialog box should reflect the model depth.

  7. Click OK.

  8. In the Spot Drilling dialog box, in the Actions group, click Generate .

  9. Click OK to complete the drilling operation.
