Define an avoid move

You need an avoid move that prevents the tool from colliding with the part.

  1. In the Geometry view of the Operation Navigator, double-click DRILL_GEOM.

  2. In the Drill Geom dialog box, in the Geometry group, click Specify Holes .

  3. In the Point dialog box, click Avoid.

    You will raise the tool 0.500 above the part between the start and end points as shown.

  4. Watch the information in the cue line.

  5. Select the hole shown as the start point.

  6. Select the hole shown as the end point.

  7. Click Distance.

  8. In the Distance box, type 0.5.

  9. Click OK to accept the distance value.

  10. Click OK to return to the Point dialog box.

  11. Click OK until you return to the Drill Geom dialog box.

  12. Click OK to complete editing the Drill Geom object.

  13. Change to a Front view.

  14. In the Operation Navigator, right-click the COUNTERBORING operation and choose Generate.

    Notice that NX raises the tool to avoid the part when traversing between the holes.

  15. Repeat Step 14 and generate the SPOT_DRILLING and DRILLING tool paths.
