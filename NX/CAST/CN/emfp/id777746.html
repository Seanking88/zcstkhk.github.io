The Drilling project

Estimated time to complete this lesson: 31–42 minutes

The English units part that you will use for this project contains blank holes, thru holes and counterbores. You will create operations that spot drill, drill, bore, ream, and counterbore the holes.

To complete this project, you should be familiar with the following operation subtypes.

  • Spot Drilling

  • Drilling

  • Boring

  • Reaming

  • Counterboring
