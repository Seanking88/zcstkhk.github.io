Spot drill the 0.350 diameter holes

  1. On the Insert toolbar, click Create Operation .

  2. In the Create Operation, in the Operation Subtype group, click Spot Drilling .

    You need to spot drill the eleven 0.350 diameter holes defined by the DRILL_GEOM parent group.

  3. In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = SPOTDRILLING_TOOL

    Geometry = DRILL_GEOM

    Method = DRILL

  4. Click OK.

    You will specify a tool tip depth of 0.0500.

  5. In the Spot Drilling dialog box, in the Cycle Type group, click Edit parameters .

  6. In the Specify Number of dialog box, accept the default of 1 set and click OK.

  7. In the Cycle Parameters dialog box, click Depth (Tip).

  8. In the Cycle Depth dialog box, click Tool Tip Depth.

  9. Type 0.050 and click OK.

  10. Click OK to return to the Spot Drilling dialog box.

  11. In the Spot Drilling dialog box, in the Actions group, click Generate to create the tool path.

    The Tool Display is turned off by default. If you want to display the tool as in the following illustration for this or any other operation, expand the Options group and click Edit Display and select 3-D from the Tool Display list. Then click OK to close the Display Options dialog box.

  12. Click OK.
