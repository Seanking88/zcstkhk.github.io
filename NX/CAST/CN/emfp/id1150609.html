Counterbore the holes to 0.500 diameter

  1. On the Insert toolbar, click Create Operation .

  2. In the Create Operation dialog box, in the Operation Subtype group, click Counterboring .

  3. In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = COUNTERBORING_TOOL_.500

    Geometry = WORKPIECE

    Method = DRILL

  4. Click OK.

    You will select the six holes using All Holes on Face.

  5. In the Counterboring dialog box, in the Geometry group, click Specify Holes .

  6. In the Point dialog box, click Select.

  7. Click All Holes on Face.

  8. Click Maximum Diameter.

  9. In the Diameter box, type 0.6.

  10. Click OK.

  11. Select the face containing the holes.

  12. Click OK to complete the selection.

  13. Click End of Selection.

  14. In the Point dialog box, click Planning Complete.

  15. In the Counterboring dialog box, in the Actions group, click Generate .

  16. Click OK to complete the counterbore operation.
