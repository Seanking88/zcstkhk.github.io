Finish the round on the od

You will create an operation that finishes the rounded surface on the outside diameter.

  1. Create a Finish Turn OD operation.

    In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = OD_55_L

    Geometry = AVOIDANCE_1

    Method = LATHE_FINISH

  2. Generate the tool path using a 2D display.

  3. Click OK to complete the operation.
