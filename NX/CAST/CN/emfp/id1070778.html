Define the blank geometry

Define the blank boundary as bar stock with a 4.50 length, 3.20 diameter, and a Mounting Point position at X= –4.35. Y=0.00, Z=0.00.

  1. In the Operation Navigator, double-click the TURN_WORKPIECE node.

  2. In the Turn Workpiece dialog box, in the Geometry group, click Specify Blank Boundaries .

  3. In the Select Blank dialog box, under Mounting Position, click Select.

  4. In the Point dialog box, in the Output Coordinates group, in the XC box, type –4.35.

  5. Click OK.

  6. In the Select Blank dialog box, in the Length box, type 4.50.

  7. In the Diameter box, type 3.20.

  8. Click OK twice.
