Move the operation to the beginning of the program

You want the Cavity Mill operation to be at the beginning of the program.

  1. In the Operation Navigator, in Program Order View, select these operations:

    CONTOUR_AREA_NON_STEEP
    CONTOUR_AREA
    FLOWCUT_SINGLE

    Tip

    To select multiple operations, hold down the Ctrl key as you select the operations.

  2. Right-click the selected operations and choose Cut.

  3. Right-click CAVITY_MILL and choose Paste.

    The operations should appear in the order shown.
