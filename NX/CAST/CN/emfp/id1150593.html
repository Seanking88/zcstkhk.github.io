Counterbore the 0.350 diameter holes

  1. On the Insert toolbar, click Create Operation .

  2. In the Create Operation dialog box, in the Operation Subtype group, click Counterboring .

  3. In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = COUNTERBORING_TOOL.600

    Geometry = DRILL_GEOM

    Method = DRILL

  4. Click OK.

  5. Change to a Top view and display the model as a wireframe.

  6. In the Counterboring dialog box, in the Actions group, click Generate .

    Notice that the side of the tool collides with the part when traversing between the third, fourth, fifth, and sixth holes.

  7. Click OK.
