Create a contour area directional steep finish operation

You will create an operation that finishes only the steep areas.

  1. On the Insert toolbar, click Create Operation .

  2. In the Create Operation dialog box, in the Type group, from the list, select mill_contour.

  3. In the Operation Subtype group, click Contour Area .

  4. In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = EM_14

    Geometry = MILL_AREA

    Method = MILL_FINISH

  5. Click OK.

  6. In the Contour Area dialog box, in the Drive Method group, click Edit .

  7. In the Area Milling Drive Method dialog box, in the Steep Containment group, from the Method list, select Directional Steep.

  8. In the Steep Angle box, type 35.

  9. In the Drive Settings group, in the Percent of Flat Diameter box, type 25.

  10. In the Angle from XC box, type 90.

  11. Click OK.

  12. In the Contour Area dialog box, in the Actions group, click Generate .

  13. Click OK to complete the operation.

  14. Refresh the graphics window.
