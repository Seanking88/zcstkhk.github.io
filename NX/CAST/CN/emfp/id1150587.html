Define the drill geometry

You will create a DRILL_GEOM object using the WORKPIECE as the parent group.

  1. On the Insert toolbar, click Create Geometry .

  2. In the Create Geometry dialog box, in the Type group, from the list, select drill.

  3. In the Geometry Subtype group, click DRILL_GEOM .

  4. In the Location group, from the Geometry list, select WORKPIECE.

  5. Click OK.

  6. In the Drill Geom dialog box, click Specify Holes .

  7. In the Point dialog box, click Select.

  8. Select the eleven holes in the order shown.

    Be sure you select the top of each 0.600 counterbore.

  9. Click OK.

  10. In the Point dialog box, click OK.

  11. In the Drill Geom dialog box, click OK.

    The DRILL_GEOM parent group node is displayed in the Geometry view of the Operation Navigator.
