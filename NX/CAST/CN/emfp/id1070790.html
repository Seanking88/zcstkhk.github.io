Rough the outside diameter

  1. On the Insert toolbar, click Create Operation .

  2. In the Create Operation dialog box, in the Operation Subtype group, click Rough Turn_OD .

    In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = UGTI0101_011

    Geometry = AVOIDANCE

    Method = LATHE_ROUGH

  3. Click OK.

  4. In the Rough Turn OD dialog box, in the Geometry group, in the Cut Regions row, click Edit .

    Define an axial trim plane 0.250 behind the part.

  5. In the Cut Regions dialog box, in the Axial Trim Plane 1 group, from the Limit Option list, select Distance.

    Ignore the warning message. The condition will be solved when the value is entered.

  6. In the Axial ZM/XM box, type –3.75.

  7. Click OK to accept the geometry containment.

  8. In the Rough Turn OD dialog box, in the Cut Regions row, click Display .

  9. In the Path Settings group, from the Reversal Mode list, select Omit.

    The Reversal Mode should be set to Omit so that the tool does not cut down into the neckdown areas.

  10. In the Options group, click Edit Display .

  11. In the Display Options dialog box, in the Tool group, from the Tool Display list, select 2D.

  12. Click OK.

  13. In the Rough Turn OD dialog box, in the Actions group, click Generate .

  14. Click OK to complete the operation.
