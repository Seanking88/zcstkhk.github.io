Thread the outside diameter

You will create an operation that threads the outside diameter.

  1. Create a Thread OD operation.

    In the Location group, set the options as shown.

    Program = PROGRAM

    Tool = UGTI0131_003

    Geometry = AVOIDANCE_1

    Method = LATHE_THREAD

  2. You will define thread geometry to determine the area to be cut.

    Define the thread geometry with the start point (1) and end point (2).

  3. In the Thread OD dialog box, in the Thread Shape group, click Select Crest Line ..

  4. Select the crest line at the right end as illustrated below. This establishes the Start point at the right end and the End point at the left end.

  5. You will define the total cut depth and cut increment to determine the root line.

    Define a total cut depth of 0.035 (1) and a cutting increment of 0.005 (2) using a tolerance of 0.001.

  6. In the Depth Option list, select Depth and Angle.

  7. In the Depth box, type 0.035.

  8. In the Angle from XC box, Type 180.

  9. In the Path Settings group, from the Cut Depth list, select Constant.

  10. In the Depth box, type 0.005.

  11. In the Cut Depth Tolerance box, type 0.001

  12. You will offset the start point slightly to accommodate the chamfer.

    Define a Start Offset of 0.050.

  13. In the Thread Shape→Offset sub-group, in the Start Offset box, type 0.050.

  14. You will specify the engage and retract settings for the cutting passes.

    Specify an engage angle of 45 (1) and a retract angle of 90 (2).

  15. In the Path Settings group, click Non Cutting Moves .

  16. In the Non Cutting Moves dialog box, click the Engage tab.

  17. From the Engage list, select Angle.

  18. In the Angle box, type 45.

  19. From the Move Type list, select Thread.

  20. Click the Retract tab.

  21. From the Retract list, select Angle.

  22. In the Angle box, type 90.

  23. From the Move Type list, select Thread.

  24. Click OK to accept the Engage/Retract settings.

  25. You will specify the number of finish and chase passes.

    Specify 2 finish passes and 2 chase passes.

  26. In the Path Settings group, click Cutting Parameters .

  27. Click the Additional Passes tab.

  28. In the Finish Passes group, in the Number of Passes box, type 2.

  29. In the Chase (Spring) Passes group, in the Chase (Spring) Passes box, type 2.

  30. Click OK to accept the finish and chase passes.

  31. Generate the tool path using a 2D display.

  32. Click OK to complete the operation.
