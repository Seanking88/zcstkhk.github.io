Reduce the depth per cut and specify the range depth

  1. In the Path Settings group, make sure Common Depth per Cut is set to Constant.

  2. In the Maximum Distance box, type 0.15.

  3. Click Cut Levels .

  4. In the Cut Levels dialog box, from the Range Type list, select Single.

  5. Select the bottom face of any one of the four cavities.

    The selected face specifies the range depth.

  6. Click OK.

  7. In the Cavity Mill dialog box, in the Options group, click Edit Display .

  8. In the Display Options dialog box, in the Path Generation group, set the following:

    Pause After Display =

    Refresh Before Display

  9. Click OK.

  10. In the Cavity Mill dialog box, click Generate .

  11. Click OK several times to complete the tool path until you return to the Cavity Mill dialog box..

    Notice that the tool path finishes each cut level by traversing from one region to the next before beginning the next cut level.
