Animate the tool movement for the entire program

Animate the tool movement for the entire program. Display material removal in 2D and display the tool as an assembly.

  1. In the Program Order view of the Operation Navigator, right-click PROGRAM and choose Tool Path→Verify.

  2. Select the 2D Material Removal check box

  3. From the Tool list, select Assembly.

  4. Click Play .

  5. Click OK to complete the tool path visualization.
