检查您的有限元模型

NX 包含许多命令可用于验证模型。可以使用这些命令执行以下操作：

  • 检查模型的 CAE 几何体到基本 CAD 几何体的保真度。

  • 确保网格的质量和一致性。

  • 验证模型已完成并可用于求解。

下表汇总了可用的命令。

命令

描述

单元质量检查

用于根据常规准则和特定于求解器的准则来评估模型中单元的质量。

单元边

可用于显示任何具有自由边的 2D 和 3D 单元。此外，还可用于显示任意非歧义单元边。

2D 单元法向

可用于评估模型中的 2D 单元法向的一致性。

重复节点

可用于检查模型中的重复节点并将其中任意节点合并在一起。

重复单元

可用于检查模型中的重复单元并删除其中任意重复单元。

单元材料方向 可用于显示模型中的 2D 和 3D 单元的材料方向。
调整 CAD 的节点邻近度 可用于调整在多边形几何体上创建的节点与基础 CAD 几何体的邻近度。
CAE 模型一致性 可用于评估模型中多边形几何体的一致性问题。
检测干涉/间隙 可用于检测模型中的干涉区域。
有限元模型汇总 可用于打印 FEM 文件中的实体的信息汇总。
仿真信息汇总 可用于打印仿真文件中的实体的信息汇总。
模型设置检查 可用于评估模型是否已做好解算准备。
