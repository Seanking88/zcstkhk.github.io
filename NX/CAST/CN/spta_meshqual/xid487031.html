显示单元的材料方向

定义网格级或单元级的材料方向之后，可以在图形窗口中查看每个单元的材料坐标系。使用单元材料方向命令可显示模型中 2D 或 3D 单元的材料方向。

要指出材料方向，软件可在每个单元上显示一个箭头，以表示材料坐标系的各个方向。

  • 对于 2D 单元，每个单元上显示的箭头表示材料坐标系的 X 轴。

  • 对于 3D 单元，可以查看材料坐标系的一个、两个或三个方向。您可以单独查看第一、第二及第三方向，也可以同时查看这三个方向。

可以使用单元材料方向对话框中的选项来控制：

  • 软件显示材料方向的单元。

  • 用于显示箭头的颜色。

  • 要显示的材料坐标系方向。
