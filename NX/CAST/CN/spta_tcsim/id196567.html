Importing and exporting data

  • Use the Import Assembly command to add data on your local operating system to the Teamcenter database.

  • Use the Export Assembly command to place an unmanaged copy of Teamcenter data on your local operating system.

When you import a simulation using the Teamcenter for Simulation data model, the software relies on target relationships to locate related files. If you import the Simulation file, all associated data is imported with it. If you import the FEM file, only the FEM file, the idealized part file, and the master part are imported.
