Teamcenter 与 Teamcenter Integration for NX 是什么？

Teamcenter Integration 是一种与 NX 结合使用的数据管理工具。在 Teamcenter Integration 模式下使用 NX 时，Teamcenter 同时作为一个独立进程运行。由于这两个程序可进行通信，因此可在 Teamcenter 数据库中创建、存储和访问数据。

Teamcenter Integration for NX 将 NX 强大的建模与几何形状数据生成功能，与 Teamcenter 强大的储存与数据检索功能组合在一起。

NX 与 Teamcenter 集成可以：

  • 管理 NX 部件以及相关文件

  • 提升对数据的访问控制

  • 轻松校订 NX 部件，并将所有校正版本的部件放在一起

  • 方便地将数据组织进用户定义的文件夹中

  • 手动和自动登录与注销

  • 轻松找到储存的数据

  • 促进团队合作和企业合作
