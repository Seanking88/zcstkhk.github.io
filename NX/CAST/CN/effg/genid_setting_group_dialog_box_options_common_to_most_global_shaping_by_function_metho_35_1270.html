大多整体变形方法公用的设置组选项

过渡将定义要对片体进行的修改。不适用于“按曲面”和“按曲线”类型，选项包括：

  • 匹配相切 — 在区域边界处匹配已变形片体的相切性。

  • 受控制的形状 — 只匹配区域边界处的位置，从而在区域边界有一个角度。

    • 形状控制滑块可以更改曲线的斜率。范围是 0.0 到 1.0。“按函数变形”产生在中心具有最高值和零斜率的 S 外形，并且在区域边界处下降到零。使用此滑块可控制下降的速度。

  • 按规律 — 使用规律子函数选项对片体进行变形。在此情况下，“区域中的点”(t=0)的值必须是 1，并且区域边界(t=1)上的值必须为 0。

  • 曲率连续 — 在区域边界处匹配已变形片体的曲率连续性。

保留输入片体将创建新的片体而不修改输入片体。

体类型 — 默认情况下，整体变形将生成片体。选择实体后可允许整体变形将生成的片体缝合到实体中。片体必须为封闭片体。

距离公差将指定用于确定两个点是否相等的公差。

角度公差将指定用于确定两个角度是否相等的公差。
